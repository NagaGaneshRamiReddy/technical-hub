Rust While Loop

Understanding While Loops in Rust

The while loop in Rust is a control flow construct that allows you to repeatedly execute a block of code as long as a specified condition remains true. This type of loop is particularly useful when the number of iterations is not known in advance.

Basic Syntax

The syntax for a while loop is straightforward:

while condition {
    // Code to execute
}

Basic Example

Hereâ€™s a simple example of a while loop that counts from 0 to 4:

let mut count = 0;

while count < 5 {
    println!("Count is: {}", count);
    count += 1;
}

In this example, the loop will print the value of count starting from 0 and incrementing until it reaches 5. Once count is no longer less than 5, the loop terminates.

Using While Loops with Conditions

You can use while loops to handle more complex conditions. For instance, consider a scenario where you want to read user input until a specific value is provided:

use std::io;

let mut input = String::new();

while input.trim() != "exit" {
    println!("Type 'exit' to quit:");
    io::stdin().read_line(&mut input).expect("Failed to read line");
}

This loop will continue prompting the user for input until they type "exit". The trim() method is used to remove any whitespace from the input.

Infinite While Loops

You can create an infinite while loop by using a condition that always evaluates to true. Hereâ€™s an example:

let mut count = 0;

while true {
    println!("Count is: {}", count);
    count += 1;

    if count >= 5 {
        break; // Exit the loop
    }
}

In this example, the loop continues indefinitely until the break statement is encountered, which exits the loop when count reaches 5.

Summary

The while loop is a powerful construct in Rust that allows for conditional iteration. It is particularly useful when the number of iterations is not known ahead of time. Understanding how to use while loops effectively will enhance your ability to manage flow control in your programs.
