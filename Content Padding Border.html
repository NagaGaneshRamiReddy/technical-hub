Content, Padding, and Border

Understanding the Box Model

The CSS box model consists of four main parts: content, padding, border, and margin. This example will show how these parts affect the size and layout of an element.

1. Content

The content is the area where text or other media is displayed. Its size is determined by the width and height properties of the element.

2. Padding

The padding is the space between the content and the element's border. It adds space inside the element, pushing the content away from the border.

3. Border

The border surrounds the content and padding. You can style it using the border property (e.g., width, style, and color).

Visual Example

In the box below, the content, padding, and border are visually highlighted:

This is the content inside the box.

Code Example


/* CSS */
.box {
    width: 250px;
    height: 150px;
    background-color: #e0f7fa; /* Light background for content */
    border: 5px solid #00695c; /* 5px solid border */
    padding: 20px; /* 20px padding inside the box */
    margin: 30px; /* Space outside the box */
    color: #004d40; /* Darker text color */
    box-sizing: border-box; /* Includes padding and border in box size */
}

Explanation of Properties

  • Width and Height: Defines the size of the content area, in this case 250px wide and 150px tall.
  • Padding: Adds 20px of space inside the box, between the content and the border.
  • Border: A 5px solid green border surrounds the content and padding.
  • Margin: There is a 30px margin outside the box, separating it from other elements on the page.
