Your First Rust Program

Creating Your First Rust Program

Now that you have Rust installed and your development environment set up, it's time to write your first Rust program. In this guide, we’ll walk you through creating a simple "Hello, World!" application.

Step 1: Create a New Project

Rust comes with a powerful tool called Cargo, which helps you manage your projects, dependencies, and builds. To create a new Rust project, open your terminal or command prompt and run the following command:

cargo new hello_world

This command will create a new directory called hello_world with all the necessary files for a basic Rust project. The structure will look like this:

hello_world/
├── Cargo.toml
└── src
    └── main.rs
  • Cargo.toml: This file contains the project’s metadata, such as the project name, version, and dependencies.
  • main.rs: This is where your program code lives. By default, it contains a simple "Hello, World!" program.

Step 2: Write Your First Rust Program

Open the src/main.rs file in your favorite code editor. You should see the following code:

fn main() {
    println!("Hello, world!");
}

This is the simplest Rust program. The main function is the entry point of every Rust program. The println! macro prints text to the console.

Step 3: Run Your Program

To run your program, use the following command in your terminal from the project directory:

cargo run

Cargo will compile your code and execute it. You should see the output:

Hello, world!

This confirms that your Rust installation is working correctly and that you’ve successfully written and executed your first Rust program.

Step 4: Understanding Cargo

Cargo not only helps you run your programs but also manages dependencies and builds. Here are some common commands you will use frequently:

  • cargo build: Compiles your project without running it.
  • cargo run: Compiles and runs your project.
  • cargo check: Quickly checks if your code compiles without producing an executable.
  • cargo test: Runs tests if you have any defined in your project.

Step 5: Modify Your Program

Let’s modify the program to print a custom message. Open main.rs again and change the code to this:

fn main() {
    println!("Welcome to Rust programming!");
}

Save the file and run the program again using cargo run. You should now see:

Welcome to Rust programming!

Summary

In this guide, you learned how to create a new Rust project using Cargo, write a simple "Hello, World!" program, and run it. You also got an introduction to how Cargo helps manage your Rust projects. Rust’s simplicity, combined with Cargo’s powerful features, makes it easy to get started with writing reliable and efficient programs.
