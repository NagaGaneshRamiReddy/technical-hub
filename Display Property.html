CSS Display Property

1. Block

The block value makes the element take up the full width available and starts on a new line. Block elements respect width and height properties.

Block Element 1
Block Element 2

2. Inline

The inline value makes the element take up only as much width as necessary and does not start on a new line. Inline elements do not respect width and height properties.

Inline Element 1
Inline Element 2

3. Inline-Block

The inline-block value combines features of both block and inline elements. It allows the element to sit next to other inline elements but respects width and height properties.

Inline-Block Element 1
Inline-Block Element 2

4. None

The none value hides the element, and it does not take up any space in the layout.

This element is hidden
Visible Element

Code Example


/* Block */
.block {
    display: block;
}

/* Inline */
.inline {
    display: inline;
}

/* Inline-Block */
.inline-block {
    display: inline-block;
}

/* None */
.none {
    display: none;
}

Summary

The display property is fundamental for controlling the layout of elements on a web page. Understanding the differences between block, inline, inline-block, and none is crucial for effective CSS styling.
