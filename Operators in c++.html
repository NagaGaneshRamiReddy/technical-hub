Operators in C++

Introduction

Operators are special symbols in C++ that perform operations on variables and values. They are essential for creating expressions that perform calculations, comparisons, and more. Understanding the different types of operators will help you write effective and efficient C++ code.

Types of Operators

C++ provides various types of operators, each serving a different purpose. Here are the main categories:

1. Arithmetic Operators

Used for basic mathematical operations:

+  -  *  /  %
  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder)

Example:

int sum = 10 + 5; // sum is 15

2. Relational Operators

Used to compare two values:

==  !=  >  <  >=  <=
  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to

Example:

bool isEqual = (10 == 5); // isEqual is false

3. Logical Operators

Used for logical operations:

&&  ||  !
  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT

Example:

bool result = (10 > 5) && (5 < 10); // result is true

4. Assignment Operators

Used to assign values to variables:

=  +=  -=  *=  /=  %=
  • = : Assign
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
  • %= : Modulus and assign

Example:

int x = 10; 
x += 5; // x is now 15

5. Increment and Decrement Operators

Used to increase or decrease a variable's value by 1:

++  --
  • ++ : Increment
  • -- : Decrement

Example:

int count = 10; 
count++; // count is now 11

6. Conditional (Ternary) Operator

Used to return a value based on a condition:

? :

Syntax:

condition ? expression1 : expression2;

Example:

int max = (a > b) ? a : b; // max gets the value of a if a > b, otherwise b

7. Bitwise Operators

Used for bit-level operations:

&  |  ^  ~  <<  >>
  • & : Bitwise AND
  • | : Bitwise OR
  • ^ : Bitwise XOR
  • ~ : Bitwise NOT
  • << : Left shift
  • >> : Right shift

Example:

int bitwiseAnd = 5 & 3; // bitwiseAnd is 1

Common Issues and Troubleshooting

If you encounter problems:

  • Operator Precedence: Be aware of operator precedence and use parentheses to clarify order of operations.
  • Type Errors: Ensure that operators are used with compatible data types.
  • Unexpected Results: Check for logical errors or incorrect assumptions in your expressions.
