Basic CSS Selectors

What are CSS Selectors?

CSS selectors are used to target HTML elements for styling. They allow you to apply styles to elements based on their tag, class, ID, or other attributes.

1. Element Selector

The element selector selects elements based on their HTML tag.


/* Select all <p> elements */
p {
    color: blue;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Element Selector Example</title>
    <style>
        p {
            color: blue;
        }
    </style>
</head>
<body>
    <p>This paragraph text will be blue.</p>
    <p>All paragraphs will have blue text.</p>
</body>
</html>

2. Class Selector

The class selector selects elements based on the class attribute. It is denoted by a period (.) followed by the class name.


/* Select elements with the class "highlight" */
.highlight {
    background-color: yellow;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Class Selector Example</title>
    <style>
        .highlight {
            background-color: yellow;
        }
    </style>
</head>
<body>
    <p class="highlight">This paragraph has a yellow background.</p>
    <p>This paragraph does not.</p>
</body>
</html>

3. ID Selector

The ID selector targets an element based on its unique ID attribute. It is denoted by a hash (#) followed by the ID name. Each ID must be unique within a document.


/* Select the element with the ID "unique" */
#unique {
    color: red;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>ID Selector Example</title>
    <style>
        #unique {
            color: red;
        }
    </style>
</head>
<body>
    <p id="unique">This paragraph has red text.</p>
    <p>This one does not.</p>
</body>
</html>

Summary

CSS selectors are the foundation for applying styles to your HTML elements. The basic selectors—element, class, and ID—allow you to select elements in a flexible and powerful way.
