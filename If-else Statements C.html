If Statement in C

Understanding the If Statement

The if statement is a fundamental building block of conditional logic in programming. It allows your program to make decisions based on certain conditions.

Basic If Statement

The if statement evaluates a condition enclosed in parentheses. If the condition evaluates to true, the code block inside the curly braces is executed. If the condition is false, the code block is skipped.


if (condition) {
    // code to be executed if condition is true
}

Example:


int age = 16;
if (age >= 18) {
    printf("You are an adult.\n");
} else {
    printf("You are a minor.\n");
}

Here, "You are a minor." will be printed because the condition age >= 18 is false.

3. If-Else If-Else Statement

The if-else if-else statement allows you to check multiple conditions sequentially.


if (condition1) {
    // code to be executed if condition1 is true
} else if (condition2) {
    // code to be executed if condition2 is true
} else {
    // code to be executed if none of the above conditions are true
}

Example:


int score = 75;
if (score >= 90) {
    printf("Grade: A\n");
} else if (score >= 80) {
    printf("Grade: B\n");
} else if (score >= 70) {
    printf("Grade: C\n");
} else {
    printf("Grade: D\n");
}

In this case, "Grade: C" will be printed because the score falls within the range for grade C.

4. Nested If-Else Statements

You can nest if-else statements inside other if-else statements to check multiple conditions.

Example:


int age = 20;
int hasID = 1;

if (age >= 18) {
    if (hasID) {
        printf("Access granted.\n");
    } else {
        printf("ID required.\n");
    }
} else {
    printf("You are too young.\n");
}

In this example, if age >= 18 is true and hasID is also true, "Access granted." will be printed. If hasID is false, "ID required." will be printed. If age >= 18 is false, "You are too young." will be printed.

5. Using Logical Operators with If-Else

Logical operators can be used within if statements to combine multiple conditions.


int age = 20;
int hasTicket = 1;

if (age >= 18 && hasTicket) {
    printf("You can enter.\n");
} else {
    printf("You cannot enter.\n");
}

Here, "You can enter." will be printed if both conditions age >= 18 and hasTicket are true. Otherwise, "You cannot enter." will be printed.
