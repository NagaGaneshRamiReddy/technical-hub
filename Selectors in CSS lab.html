Web Development Lab: Task - CSS Selectors

Task: Apply Different Types of CSS Selectors (50 Marks)

In this task, you will learn how to apply CSS using different types of selectors:

  • **Element selectors** (e.g., h1, p)
  • **Class selectors** (e.g., .classname)
  • **ID selectors** (e.g., #idname)
  • **Group selectors** (e.g., h1, h2, h3)
  • **Pseudo-class selectors** (e.g., :hover, :first-child)
  • **Attribute selectors** (e.g., [type="text"])

Instructions

Use a single HTML document and apply various CSS selectors to style it. Your document should include:

  • At least two **element selectors**
  • At least two **class selectors**
  • At least two **ID selectors**
  • At least one **group selector**
  • At least two **pseudo-class selectors** (e.g., hover)
  • At least one **attribute selector** (e.g., [type="text"])

Example Output

Your webpage should look like this:

This is an Element Selector (h1)

This paragraph uses a Class Selector (class="info").

This paragraph uses an ID Selector (id="special").
