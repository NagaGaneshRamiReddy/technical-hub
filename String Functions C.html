String Functions

Introduction to String Functions

In C programming, string functions are used to perform various operations on strings. These functions are provided in the C standard library, specifically in the string.h header file. Here, we will cover some of the most commonly used string functions.

Common String Functions

strlen

The strlen function calculates the length of a string, excluding the null terminator.

#include <string.h>

size_t strlen(const char *str);

Example:

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World!";
    printf("Length of str: %lu\n", strlen(str)); // Output: Length of str: 13
    return 0;
}

strcmp

The strcmp function compares two strings and returns an integer indicating their lexicographical order.

#include <string.h>

int strcmp(const char *str1, const char *str2);

Example:

#include <stdio.h>
#include <string.h>

int main() {
    char str1[] = "apple";
    char str2[] = "banana";
    int result = strcmp(str1, str2);
    if (result < 0) {
        printf("%s is less than %s\n", str1, str2);
    } else if (result > 0) {
        printf("%s is greater than %s\n", str1, str2);
    } else {
        printf("%s is equal to %s\n", str1, str2);
    }
    return 0;
}

strcpy

The strcpy function copies the content of one string to another.

#include <string.h>

char *strcpy(char *dest, const char *src);

Example:

#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "Hello";
    char dest[20];
    strcpy(dest, src);
    printf("Copied string: %s\n", dest); // Output: Copied string: Hello
    return 0;
}

strcat

The strcat function concatenates (appends) one string to the end of another.

#include <string.h>

char *strcat(char *dest, const char *src);

Example:

#include <stdio.h>
#include <string.h>

int main() {
    char dest[50] = "Hello";
    char src[] = " World!";
    strcat(dest, src);
    printf("Concatenated string: %s\n", dest); // Output: Concatenated string: Hello World!
    return 0;
}

strchr

The strchr function searches for the first occurrence of a character in a string and returns a pointer to it.

#include <string.h>

char *strchr(const char *str, int c);

Example:

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World!";
    char *ptr = strchr(str, 'W');
    if (ptr) {
        printf("Found character: %c\n", *ptr); // Output: Found character: W
    }
    return 0;
}

Summary

String functions are essential for manipulating and handling strings in C programming. Functions such as strlen, strcmp, strcpy, strcat, and strchr provide a variety of tools for working with text data. Mastering these functions will help you efficiently manage strings in your C programs.
