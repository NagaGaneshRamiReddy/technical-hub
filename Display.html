Bootstrap Display Utilities


Introduction to Bootstrap Display Utilities

Bootstrap provides various utility classes for controlling the display and visibility of elements. These classes allow you to show or hide elements based on different screen sizes.

Basic Display Utilities

You can use the following classes to control the display of an element:

  • .d-none: Hides an element.
  • .d-block: Displays an element as a block.
  • .d-inline: Displays an element as an inline element.
  • .d-inline-block: Displays an element as an inline-block.
<div class="d-none">This will not be displayed.</div>
<div class="d-block">This is a block element.</div>

Expected Output:

This will not be displayed.
This is a block element.

Responsive Display Utilities

You can also control the display of elements responsively using breakpoint-specific classes:

<div class="d-none d-md-block">Visible on medium devices and larger.</div>

Expected Output:

Visible on medium devices and larger.

Display Flex Utilities

Bootstrap allows you to set an element's display to flex:

<div class="d-flex">
    <div>Flex Item 1</div>
    <div>Flex Item 2</div>
</div>

Expected Output:

Flex Item 1
Flex Item 2
