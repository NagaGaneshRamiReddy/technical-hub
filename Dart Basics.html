Dart Basics

Welcome to Dart

Dart is a client-optimized programming language developed by Google, designed primarily for building mobile, desktop, and web applications. It is the language that powers Flutter, and understanding the basics of Dart is essential for Flutter development.

Why Learn Dart?

1. Simple Syntax

Dart is easy to learn, especially if you have experience with languages like Java, JavaScript, or C#. It features clear and expressive syntax, making it ideal for both beginners and experienced developers.

2. Object-Oriented

Dart is a fully object-oriented language, which means everything in Dart is an object. This makes it easy to structure applications and use principles like encapsulation, inheritance, and polymorphism.

3. Cross-Platform

Dart can compile to native code or JavaScript, making it versatile for building cross-platform applications that run smoothly on various devices and environments.

Getting Started with Dart

Here are some basic concepts and features of Dart:

  • Variables: In Dart, variables can be declared using `var`, `final`, or `const`. For example: 
    var name = 'John';
  • Functions: Functions in Dart are declared using the `returnType functionName()` syntax. For example: 
    void sayHello() {
    print('Hello!');
}
  • Control Structures: Dart supports common control structures like `if`, `else`, `for`, and `while` loops. For example: 
    if (age > 18) {
    print('Adult');
} else {
    print('Minor');
}
  • Classes: Dart is an object-oriented language, so you can create classes and objects. For example: 
    class Person {
    String name;
    int age;
    
    Person(this.name, this.age);
    
    void displayInfo() {
        print('Name: \$name, Age: \$age');
    }
}
  • Null Safety: Dart features sound null safety, meaning variables can’t be null unless explicitly declared. This helps prevent null reference errors.

Dart Basics

1. Variables and Data Types

Dart supports various data types, including integers (`int`), floating-point numbers (`double`), strings (`String`), booleans (`bool`), and lists (`List`). Here's an example of different data types:

var age = 25; // int
double height = 5.9; // double
String name = 'Alice'; // String
bool isStudent = true; // bool
List cities = ['New York', 'London', 'Tokyo']; // List

2. Functions

Functions in Dart can return values or be void (no return). You can also use lambda or arrow functions for short expressions. For example:

int add(int a, int b) {
    return a + b;
}

// Arrow function
int multiply(int a, int b) => a * b;

3. Conditional Statements

Use `if`, `else if`, and `else` statements to control the flow of your application. Dart also supports `switch` statements for multiple conditions. For example:

if (score >= 90) {
    print('A');
} else if (score >= 80) {
    print('B');
} else {
    print('C');
}

// Switch statement
switch (day) {
    case 'Monday':
        print('Start of the week');
        break;
    case 'Friday':
        print('End of the week');
        break;
    default:
        print('Middle of the week');
}

4. Loops

Dart supports `for`, `while`, and `do-while` loops to iterate through data or repeat tasks. For example:

for (var i = 0; i < 5; i++) {
    print(i);
}

while (n > 0) {
    print(n);
    n--;
}

Explore More Dart Concepts

Once you're comfortable with these basics, you can dive deeper into Dart features like asynchronous programming, classes and objects, collections, and mixins.
