Classes and Objects in C++

Introduction

In C++, classes and objects are fundamental concepts in Object-Oriented Programming (OOP). A class is a blueprint for creating objects, and an object is an instance of a class. Classes encapsulate data and functions that operate on that data, allowing for modular and reusable code.

Classes

A class in C++ defines a new data type by grouping data members (attributes) and member functions (methods) together. This encapsulation allows you to model real-world entities more effectively.

Defining a Class

To define a class in C++, use the class keyword followed by the class name and a pair of curly braces:

class ClassName {
    // Data members
    // Member functions
};

Hereâ€™s an example of a simple class definition:

#include <iostream>

class Car {
public:
    // Data members
    std::string brand;
    int year;

    // Member function
    void displayInfo() {
        std::cout << "Brand: " << brand << ", Year: " << year << std::endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2022;
    myCar.displayInfo(); // Outputs: Brand: Toyota, Year: 2022

    return 0;
}

Objects

An object is an instance of a class. When you create an object, you can access the data members and member functions of the class through that object.

Creating Objects

To create an object of a class, simply declare a variable of that class type:

ClassName objectName;

For example, in the code snippet above, myCar is an object of the Car class.

Accessing Members

You can access the data members and member functions of an object using the dot operator (.):

objectName.dataMember;
objectName.memberFunction();

In the Car example, myCar.brand accesses the brand data member, and myCar.displayInfo() calls the displayInfo method.

Constructors and Destructors

Constructors and destructors are special member functions used to initialize and clean up objects, respectively.

Constructors

A constructor is a special function that is automatically called when an object is created. It is used to initialize the object's data members.

class ClassName {
public:
    ClassName() {
        // Initialization code
    }
};

Example:

#include <iostream>

class Car {
public:
    std::string brand;
    int year;

    // Constructor
    Car(std::string b, int y) : brand(b), year(y) {}

    void displayInfo() {
        std::cout << "Brand: " << brand << ", Year: " << year << std::endl;
    }
};

int main() {
    Car myCar("Toyota", 2022);
    myCar.displayInfo(); // Outputs: Brand: Toyota, Year: 2022

    return 0;
}

Destructors

A destructor is a special function that is automatically called when an object goes out of scope. It is used to release resources or perform clean-up tasks.

class ClassName {
public:
    ~ClassName() {
        // Clean-up code
    }
};

Example:

#include <iostream>

class Car {
public:
    std::string brand;
    int year;

    // Constructor
    Car(std::string b, int y) : brand(b), year(y) {}

    // Destructor
    ~Car() {
        std::cout << "Car object destroyed" << std::endl;
    }

    void displayInfo() {
        std::cout << "Brand: " << brand << ", Year: " << year << std::endl;
    }
};

int main() {
    Car myCar("Toyota", 2022);
    myCar.displayInfo(); // Outputs: Brand: Toyota, Year: 2022

    return 0;
}

Access Specifiers

Access specifiers determine the visibility of class members. There are three main access specifiers:

  • public: Members are accessible from outside the class.
  • private: Members are only accessible from within the class itself.
  • protected: Members are accessible from within the class and derived classes.

Example:

#include <iostream>

class Person {
private:
    std::string name;

public:
    void setName(std::string n) {
        name = n;
    }

    std::string getName() {
        return name;
    }
};

int main() {
    Person person;
    person.setName("John Doe");
    std::cout << "Name: " << person.getName() << std::endl; // Outputs: Name: John Doe

    return 0;
}

Summary

Classes and objects are the core components of Object-Oriented Programming in C++. By defining classes, you can create structured and reusable code. Objects are instances of these classes and interact with the class's data and functions. Understanding these concepts is crucial for effective C++ programming and building complex applications.
