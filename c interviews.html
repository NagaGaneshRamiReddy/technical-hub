C Programming Interview Questions

Basic Level

1. What is a pointer in C? ›

A pointer in C is a variable that stores the memory address of another variable. Pointers are used for dynamic memory allocation, arrays, and functions.

Example:

int x = 10;
int *p = &x; // p points to x
printf("%d", *p); // Output: 10
2. What is the difference between "==" and "="? ›

"==" is the equality operator that checks if two values are equal, while "=" is the assignment operator used to assign a value to a variable.

Example:

int a = 5; // Assignment
if (a == 5) { // Equality check
    printf("a is equal to 5");
}
3. What is a structure in C? ›

A structure is a user-defined data type in C that allows grouping of different data types together. Structures are used to represent a record.

Example:

struct Student {
    char name[50];
    int age;
};

struct Student s1; // Creating a structure variable
strcpy(s1.name, "John");
s1.age = 20;
printf("%s is %d years old.", s1.name, s1.age);

Intermediate Level

4. What are the different storage classes in C? ›

There are four storage classes in C:

  • auto: The default storage class for local variables.
  • register: Stores variables in CPU registers for faster access.
  • static: Preserves variable value across function calls.
  • extern: Declares a variable defined in another file.
5. What is the difference between malloc() and calloc()? ›

malloc(): Allocates a specified number of bytes but does not initialize the memory.

calloc(): Allocates memory for an array of elements and initializes all bytes to zero.

Example:

// Using malloc
int *arr1 = (int *)malloc(5 * sizeof(int));

// Using calloc
int *arr2 = (int *)calloc(5, sizeof(int));
6. What is a function pointer? ›

A function pointer is a pointer that points to a function instead of a variable. It allows the function to be called dynamically.

Example:

#include <stdio.h>

void greet() {
    printf("Hello, World!");
}

int main() {
    void (*funcPtr)() = &greet; // Pointer to the function greet
    funcPtr(); // Call the function
    return 0;
}

Advanced Level

7. What is a segmentation fault? ›

A segmentation fault occurs when a program tries to access a memory location that it's not allowed to. This usually happens due to dereferencing a null or uninitialized pointer.

Example:

int *p = NULL;
printf("%d", *p); // This will cause a segmentation fault
8. What is the use of the 'static' keyword? ›

The 'static' keyword is used to declare static variables. Static variables maintain their value between function calls and are only initialized once.

Example:

#include <stdio.h>

void count() {
    static int count = 0; // Initialized only once
    count++;
    printf("%d\n", count);
}

int main() {
    count(); // Output: 1
    count(); // Output: 2
    return 0;
}
9. What is the difference between a stack and a heap? ›

The stack is a region of memory that stores local variables and function call information, managed automatically. The heap is a region of memory used for dynamic memory allocation, managed manually using functions like malloc and free.

Example:

// Stack allocation
void function() {
    int a = 10; // Allocated on the stack
}

// Heap allocation
int *ptr = (int *)malloc(sizeof(int)); // Allocated on the heap
free(ptr); // Free the allocated memory
