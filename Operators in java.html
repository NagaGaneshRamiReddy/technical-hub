Operators in Java

Introduction

In Java, operators are special symbols used to perform operations on variables and values. Operators play an essential role in Java programming, allowing you to manipulate data, control flow, and perform calculations. Below are the different types of operators in Java:

1. Arithmetic Operators in Java

Arithmetic operators are used to perform basic mathematical operations such as addition, subtraction, multiplication, and division. Below are the commonly used arithmetic operators:

Operator Description Example
+ Addition int sum = 5 + 3;
- Subtraction int difference = 5 - 3;
* Multiplication int product = 5 * 3;
/ Division int quotient = 5 / 3;
% Modulus (remainder) int remainder = 5 % 3; 

// Simple Java Program to demonstrate Arithmetic Operators
 public class SimpleArithmeticExample {
 public static void main(String[] args) {
 int a = 5;
 int b = 3;

// Performing arithmetic operations
int sum = a + b;             // Addition
int difference = a - b;      // Subtraction
int product = a * b;         // Multiplication
int quotient = a / b;        // Division
int remainder = a % b;       // Modulus

    // Displaying the results
System.out.println("Sum: " + sum);
System.out.println("Difference: " + difference);
System.out.println("Product: " + product);
System.out.println("Quotient: " + quotient);
System.out.println("Remainder: " + remainder);
    }
}

Arithmetic Operators Output

This is the output from the simple arithmetic program:

Sum: 8
Difference: 2
Product: 15
Quotient: 1
Remainder: 2

Explanation of the Output

Sum: 8 — 5 + 3 is 8, so Sum is 8.

Difference: 2 — 5 - 3 is 2, so Difference is 2.

Product: 15 — 5 * 3 is 15, so Product is 15.

Quotient: 1 — 5 / 3 is 1 (integer division truncates the decimal part), so Quotient is 1.

Remainder: 2 — 5 % 3 gives a remainder of 2, so Remainder is 2.

2. Relational Operators in Java

Relational operators are used to compare two values and return a boolean result (true or false).

Operator Description Example
== Equals boolean isEqual = (5 == 3);
!= Not Equals boolean isNotEqual = (5 != 3);
> Greater Than boolean isGreater = (5 > 3);
< Less Than boolean isLess = (5 < 3);
>= Greater Than or Equal To boolean isGreaterOrEqual = (5 >= 3);
<= Less Than or Equal To boolean isLessOrEqual = (5 <= 3); 

// Java Program using Relational Operators
public class RelationalExample {
public static void main(String[] args) {
boolean isEqual = (5 == 3);
boolean isNotEqual = (5 != 3);
boolean isGreater = (5 > 3);
boolean isLess = (5 < 3);
boolean isGreaterOrEqual = (5 >= 3);
boolean isLessOrEqual = (5 <= 3);

System.out.println("isEqual: " + isEqual);
System.out.println("isNotEqual: " + isNotEqual);
System.out.println("isGreater: " + isGreater);
System.out.println("isLess: " + isLess);
System.out.println("isGreaterOrEqual: " + isGreaterOrEqual);
System.out.println("isLessOrEqual: " + isLessOrEqual);
            }
        }

Relational Operators Output

This is the output from the simple Relational Operators:

isEqual: false
isNotEqual: true
isGreater: true
isLess: false
isGreaterOrEqual: true
isLessOrEqual: false

Explanation of the Output

isEqual: false — 5 == 3 is false, so isEqual is false.

isNotEqual: true — 5 != 3 is true, so isNotEqual is true.

isGreater: true — 5 > 3 is true, so isGreater is true.

isLess: false — 5 < 3 is false, so isLess is false.

isGreaterOrEqual: true — 5 >= 3 is true, so isGreaterOrEqual is true.

isLessOrEqual: false — 5 <= 3 is false, so isLessOrEqual is false.

3. Logical Operators in Java

Logical operators are used to perform logical operations on boolean values. They help in combining multiple boolean expressions.

Operator Description Example
&&& Logical AND boolean and = (true && false);
|| Logical OR boolean or = (true || false);
! Logical NOT boolean not = !true; 

// Java Program using Logical Operators
public class LogicalExample {
 public static void main(String[] args) {
    boolean and = (true && false);
    boolean or = (true || false);
    boolean not = !true;

        System.out.println("AND: " + and);
        System.out.println("OR: " + or);
        System.out.println("NOT: " + not);
            }
        }

Logical Operators Output

This is the output from the simple Logical Operators:

AND: false
OR: true
NOT: false

Explanation of the Output

AND: false — true && false evaluates to false because both operands must be true for the AND operator to return true, but one of them is false.

OR: true — true || false evaluates to true because only one operand needs to be true for the OR operator to return true, and here the first operand is true.

NOT: false — !true is the logical negation of true, which results in false.

4. Assignment Operators in Java

Assignment operators are used to assign values to variables, and can also combine operations with assignment.

Operator Description Example
= Assignment int x = 10;
+= Addition Assignment x += 5; // x = x + 5
-= Subtraction Assignment x -= 5; // x = x - 5
*= Multiplication Assignment x *= 5; // x = x * 5
/= Division Assignment x /= 5; // x = x / 5 

// Java Program using Assignment Operators
public class AssignmentExample {
public static void main(String[] args) {
    int x = 10;
    x += 5;
    x -= 5;
    x *= 5;
    x /= 5;
    System.out.println("Value of x: " + x);
            }
        }

Assignment Operators Output

This is the output from the simple Assignment Operators:

    Value of x: 10

Explanation of the Output

x = 10 — Initially, the variable x is assigned the value 10.

x += 5 — The += assignment operator adds 5 to the current value of x, so x becomes 15.

x -= 5 — The -= assignment operator subtracts 5 from the current value of x, so x becomes 10 again.

x *= 5 — The *= assignment operator multiplies the current value of x by 5, so x becomes 50.

x /= 5 — The /= assignment operator divides the current value of x by 5, so x becomes 10 again (since 50 divided by 5 is 10).

5. Unary Operators in Java

Unary operators operate on a single operand, modifying its value or performing actions such as increment and decrement.

Operator Description Example
+ Unary Plus int positive = +5;
- Unary Minus int negative = -5;
++ Increment count++;
-- Decrement count--; 

// Java Program using Unary Operators
public class UnaryExample {
public static void main(String[] args) {
   int count = 5;
   count++;  // Increment
   count--;  // Decrement
      System.out.println("Incremented Count: " + count);
      System.out.println("Decremented Count: " + (count-1));
            }
        }

Unary Operators Output

This is the output from the simple Unary Operators:

Incremented Count: 6
Decremented Count: 5

Explanation of the Output

count = 5 — Initially, the variable count is assigned the value 5.

count++ — The ++ operator is a unary increment operator that increases the value of count by 1. After this operation, count becomes 6.

count-- — The -- operator is a unary decrement operator that decreases the value of count by 1. After this operation, count becomes 5 again.

6. Bitwise Operators in Java

Bitwise operators perform bit-level operations, which can be used for low-level manipulations and optimizations.

Operator Description Example
& Bitwise AND int result = 5 & 3;
| Bitwise OR int result = 5 | 3;
^ Bitwise XOR int result = 5 ^ 3;
~ Bitwise Complement int result = ~5;
<< Left Shift int result = 5 << 1;
>> Right Shift int result = 5 >> 1;
>>> Unsigned Right Shift int result = 5 >>> 1; 

// Java Program using Bitwise Operators
public class BitwiseExample {
public static void main(String[] args) {
    int bitwiseAnd = 5 & 3;
    int bitwiseOr = 5 | 3;
    int bitwiseXor = 5 ^ 3;
    int bitwiseComplement = ~5;
    int leftShift = 5 << 1;
    int rightShift = 5 >> 1;
    int unsignedRightShift = 5 >>> 1;

    System.out.println("Bitwise AND: " + bitwiseAnd);
    System.out.println("Bitwise OR: " + bitwiseOr);
    System.out.println("Bitwise XOR: " + bitwiseXor);
    System.out.println("Bitwise Complement: " + bitwiseComplement);
    System.out.println("Left Shift: " + leftShift);
    System.out.println("Right Shift: " + rightShift);
    System.out.println("Unsigned Right Shift: " + unsignedRightShift);
            }
        }

Bitwise Operators Output

This is the output from the Bitwise Operators program:

Bitwise AND: 1
Bitwise OR: 7
Bitwise XOR: 6
Bitwise Complement: -6
Left Shift: 10
Right Shift: 2
Unsigned Right Shift: 2

Explanation of the Output

bitwiseAnd = 5 & 3 — The & operator performs a bitwise AND. It compares the binary representation of 5 (0101) and 3 (0011) and returns 1 where both bits are 1, resulting in 1 (binary 0001).

bitwiseOr = 5 | 3 — The | operator performs a bitwise OR. It compares the binary representation of 5 (0101) and 3 (0011) and returns 1 where either of the bits is 1, resulting in 7 (binary 0111).

bitwiseXor = 5 ^ 3 — The ^ operator performs a bitwise XOR (exclusive OR). It compares the binary representation of 5 (0101) and 3 (0011) and returns 1 where only one of the bits is 1, resulting in 6 (binary 0110).

bitwiseComplement = ~5 — The ~ operator performs a bitwise complement (NOT). It flips all the bits of 5 (binary 0101), resulting in -6 because of the way negative numbers are represented in two's complement.

leftShift = 5 << 1 — The << operator performs a left shift. It shifts the bits of 5 (binary 0101) one position to the left, resulting in 10 (binary 1010).

rightShift = 5 >> 1 — The >> operator performs a right shift. It shifts the bits of 5 (binary 0101) one position to the right, resulting in 2 (binary 0010).

unsignedRightShift = 5 >>> 1 — The >>> operator performs an unsigned right shift. It shifts the bits of 5 (binary 0101) one position to the right, resulting in 2 (binary 0010), and does not preserve the sign bit.
