HTML Lists

Understanding Lists in HTML

Lists are a fundamental part of HTML, allowing you to create organized collections of items. There are three main types of lists: unordered lists, ordered lists, and description lists.

Unordered Lists

An unordered list is used to group items that do not have a specific order. It is created using the <ul> tag, with each item defined by the <li> tag.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Expected Output:

  • Item 1
  • Item 2
  • Item 3

Ordered Lists

An ordered list is used for items that should be listed in a specific sequence. It is created using the <ol> tag, with each item also defined by the <li> tag.

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Expected Output:

  1. First item
  2. Second item
  3. Third item

Description Lists

A description list is used to group terms and their descriptions. It is created using the <dl> tag, with each term defined by the <dt> tag and each description by the <dd> tag.

<dl>
    <dt>Term 1</dt>
    <dd>Description for Term 1</dd>
    <dt>Term 2</dt>
    <dd>Description for Term 2</dd>
</dl>

Expected Output:

Term 1
Description for Term 1
Term 2
Description for Term 2

Styling Lists with CSS

Lists can be styled using CSS to enhance their appearance. You can change the list style type, padding, and margins.

ul {
    list-style-type: square;
    padding: 10px;
}
ol {
    list-style-type: upper-alpha;
    padding: 10px;
}

Expected Output: The list styles will change according to the CSS applied.
