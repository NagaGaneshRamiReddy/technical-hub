HTML Headings

What are Headings?

Headings in HTML are used to define the structure and hierarchy of the content. They range from <h1> to <h6>, where <h1> is the highest level (most important) and <h6> is the lowest level (least important).

Importance of Headings

  • SEO Benefits: Search engines use headings to understand the structure of the content, improving search rankings.
  • Accessibility: Screen readers rely on headings to help visually impaired users navigate through content.
  • Readability: Proper use of headings breaks up text, making it easier for readers to scan and comprehend the content.

HTML Heading Tags

Tag Description Example
<h1> Main heading of the page. 
<h1>This is a Heading 1</h1>
<h2> Subheading of the main heading. 
<h2>This is a Heading 2</h2>
<h3> Subheading of a heading 2. 
<h3>This is a Heading 3</h3>
<h4> Subheading of a heading 3. 
<h4>This is a Heading 4</h4>
<h5> Subheading of a heading 4. 
<h5>This is a Heading 5</h5>
<h6> Subheading of a heading 5. 
<h6>This is a Heading 6</h6>

Example of Headings in Use

<h1>Welcome to My Website</h1>
<h2>About Us</h2>
<p>We are a company that values excellence.</p>
<h2>Our Services</h2>
<h3>Web Development</h3>
<p>We create amazing websites.</p>
<h3>App Development</h3>
<p>We develop mobile applications.</p>
<h2>Contact Us</h2>

Expected Output:

Welcome to My Website

About Us

We are a company that values excellence.

Our Services

Web Development

We create amazing websites.

App Development

We develop mobile applications.

Contact Us
