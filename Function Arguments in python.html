Function Arguments in Python

What are Function Arguments?

Function arguments are values passed into a function when it is called. They allow you to provide input to the function, which can be used within the function to perform operations or calculations. Python supports several types of function arguments, each serving different purposes.

Types of Function Arguments

  • Positional Arguments: Arguments that must be provided in the correct order. The values are assigned to parameters based on their position in the function call.
  • Keyword Arguments: Arguments passed by explicitly naming each parameter. This allows you to specify arguments in any order and improves code readability.
  • Default Arguments: Arguments that assume a default value if no value is provided during the function call. This makes some arguments optional.
  • Variable-Length Arguments: Allows a function to accept an arbitrary number of arguments. There are two types: *args (for non-keyword arguments) and **kwargs (for keyword arguments).

Syntax and Examples

Positional Arguments

Positional arguments are the most common type of arguments. The function parameters receive values based on their position in the argument list:

# Function with positional arguments
def greet(name, age):
    print(f"Hello, {name}! You are {age} years old.")

# Calling the function
greet("Alice", 30)

Keyword Arguments

Keyword arguments are specified by name, allowing you to pass arguments in any order:

# Function with keyword arguments
def greet(name, age):
    print(f"Hello, {name}! You are {age} years old.")

# Calling the function with keyword arguments
greet(age=30, name="Bob")

Default Arguments

Default arguments are used when no value is provided for a parameter. The default value is used instead:

# Function with default arguments
def greet(name, age=25):
    print(f"Hello, {name}! You are {age} years old.")

# Calling the function with and without default argument
greet("Charlie")       # Uses default age
greet("Dana", 40)      # Provides custom age

Variable-Length Arguments

Variable-length arguments allow you to pass an arbitrary number of arguments to a function. *args captures non-keyword arguments, while **kwargs captures keyword arguments:

# Function with variable-length arguments
def print_info(*args, **kwargs):
    print("Arguments:", args)
    print("Keyword Arguments:", kwargs)

# Calling the function
print_info(1, 2, 3, name="Eve", age=22)

Key Points

  • Positional Arguments: Must match the order of parameters in the function definition.
  • Keyword Arguments: Improve readability and allow arguments to be passed in any order.
  • Default Arguments: Provide default values for parameters, making them optional.
  • Variable-Length Arguments: Useful for functions that need to handle a varying number of arguments.

Summary

Understanding function arguments is crucial for writing flexible and reusable code in Python. By mastering positional, keyword, default, and variable-length arguments, you can create functions that are more adaptable and easier to work with.
