If-Else Statements in Python

1. What Are If-Else Statements?

If-else statements are used to make decisions in your program. They help your code choose different actions based on certain conditions. Think of them like asking a question and deciding what to do based on the answer.

2. Basic if Statement

The basic if statement checks if a condition is true. If it is, the code inside the if block runs:

temperature = 30

if temperature > 25:
    print("It's a hot day!")

In this example, if the temperature is more than 25 degrees, it will print "It's a hot day!"

3. if-else Statement

The if-else statement lets you handle two possibilities. If the condition is true, the code in the if block runs. If not, the code in the else block runs:

temperature = 20

if temperature > 25:
    print("It's a hot day!")
else:
    print("It's not a hot day.")

Here, if the temperature is not greater than 25 degrees, it will print "It's not a hot day."

4. if-elif-else Statement

With if-elif-else, you can check multiple conditions. The program tests each condition in order, and runs the block of code for the first condition that is true:

temperature = 15

if temperature > 25:
    print("It's a hot day!")
elif temperature > 15:
    print("It's a pleasant day.")
else:
    print("It's a cold day.")

In this example, if the temperature is not hot (greater than 25), but still pleasant (greater than 15), it prints "It's a pleasant day." If none of these conditions are true, it prints "It's a cold day."

5. Nested if Statements

Sometimes, you need to make decisions inside other decisions. This is called nesting. Here's how you can do it:

temperature = 18
humidity = 70

if temperature > 15:
    if humidity > 60:
        print("It's warm and humid today.")
    else:
        print("It's warm and dry today.")
else:
    print("It's cool today.")

This checks if it's warm first. If it is, it then checks if it's humid. Depending on these conditions, it prints a message about the weather.

6. Summary

If-else statements help your program make choices. They let you run different pieces of code based on whether conditions are true or false. Understanding how to use them is key to making your programs respond to different situations.
