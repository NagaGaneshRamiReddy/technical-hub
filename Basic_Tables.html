Basic Tables in HTML

Understanding Basic Tables

HTML tables are used to present data in a structured format. They are composed of rows and columns, making it easier to read and compare data.

Creating a Basic Table

The basic structure of a table in HTML consists of the following elements:

  • <table>: The container for the entire table.
  • <tr>: Represents a table row.
  • <th>: Represents a table header cell.
  • <td>: Represents a standard table cell.

Example: A Simple Table

Here’s how to create a simple table:

<table>
    <tr>
        <th>Product</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>Apple</td>
        <td>$1.00</td>
    </tr>
    <tr>
        <td>Banana</td>
        <td>$0.50</td>
    </tr>
</table>

Expected Output:

Product Price
Apple $1.00
Banana $0.50

Example: A Table with Multiple Rows

Here’s another example with more products:

<table>
    <tr>
        <th>Product</th>
        <th>Price</th>
        <th>Quantity</th>
    </tr>
    <tr>
        <td>Apple</td>
        <td>$1.00</td>
        <td>10</td>
    </tr>
    <tr>
        <td>Banana</td>
        <td>$0.50</td>
        <td>20</td>
    </tr>
    <tr>
        <td>Orange</td>
        <td>$0.80</td>
        <td>15</td>
    </tr>
</table>

Expected Output:

Product Price Quantity
Apple $1.00 10
Banana $0.50 20
Orange $0.80 15

Table with Captions

You can add captions to tables for clarity:

<table>
    <caption>Fruit Price List</caption>
    <tr>
        <th>Product</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>Apple</td>
        <td>$1.00</td>
    </tr>
</table>

Expected Output:

Fruit Price List
Product Price
Apple $1.00

Summary

Basic tables in HTML are straightforward to create and useful for organizing data in a clear and readable format. You can enhance tables by adding headers, captions, and more rows and columns.
