Permutations & Combinations

Combinations

Combinations refer to the different ways of selecting items from a larger set where the order does not matter. Unlike permutations, which focus on arrangement, combinations are concerned with the selection of items.

Key Concepts

1. Definition of Combinations

A combination of a set is a selection of items where the order of the items does not matter. For example, the combinations of choosing 2 items from the set {A, B, C} are AB, AC, and BC. Note that AB is considered the same as BA in combinations.

2. Formula for Combinations

The formula for finding the number of combinations of selecting r items from a total of n distinct items is:

Formula:
C(n, r) = n! / [r! × (n - r)!]

Where:
n: Total number of items.
r: Number of items to select.
n!: Factorial of n.
r!: Factorial of r.
(n - r)!: Factorial of (n - r).

Example:
To find the number of ways to choose 2 items from a set of 4 items (say A, B, C, D):
C(4, 2) = 4! / [2! × (4 - 2)!] = 24 / [2 × 2] = 6
Thus, there are 6 different combinations of 2 items from 4.

3. Combinations with Repeated Elements

When you have a situation where items can be chosen more than once, you use the formula for combinations with repetition:

Formula:
C(n + r - 1, r) = (n + r - 1)! / [r! × (n - 1)!]

Where:
n: Number of types of items.
r: Number of items to choose.

Example:
To find the number of ways to choose 3 items from 5 types of items where repetition is allowed:
C(5 + 3 - 1, 3) = (7)! / [3! × (5 - 1)!] = 5040 / [6 × 24] = 35
Thus, there are 35 different combinations of 3 items from 5 types with repetition allowed.

4. Special Cases

Here are some special cases of combinations:

  • Choosing All Items: The number of ways to choose all items from a set is always 1 (C(n, n) = 1).
  • Choosing Zero Items: The number of ways to choose zero items from a set is also 1 (C(n, 0) = 1).
Example:
The number of ways to choose all 4 items from a set of 4 (C(4, 4)) is:
C(4, 4) = 1
The number of ways to choose 0 items from a set of 4 (C(4, 0)) is also:
C(4, 0) = 1

Applications of Combinations

  • Lottery Games: Calculating the number of possible combinations for lottery numbers.
  • Team Selection: Choosing team members from a larger group without regard to order.
  • Data Sampling: Selecting samples from a population in statistical studies.
