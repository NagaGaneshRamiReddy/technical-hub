Max-Heap

Introduction

A Max-Heap is a specialized binary tree-based data structure that maintains the heap property where the key value of each node is greater than or equal to the key values of its children. The largest key is always at the root of the tree. Max-Heaps are commonly used for implementing priority queues where the maximum element needs to be accessed quickly.

Properties of Max-Heaps

  • Heap Property: In a Max-Heap, for any given node `i`, the key of `i` is greater than or equal to the keys of its children.
  • Complete Binary Tree: A Max-Heap is a complete binary tree, meaning all levels are fully filled except possibly the last level, which is filled from left to right.
  • Root Element: The root node contains the maximum key value in the heap.

Basic Operations

  • Insertion: Adding a new element to the heap while maintaining the Max-Heap property. The element is initially added at the end of the heap and then "bubbled up" to restore the heap property.
  • Deletion: Removing the root element (which is the maximum element) and then restoring the heap property by "bubbling down" the new root element.
  • Peek: Retrieving the root element without removing it. In a Max-Heap, this is the maximum element.
  • Heapify: Converting an arbitrary array into a Max-Heap. This can be achieved in linear time using the heapify process.

Implementation Examples

1. Max-Heap in C++


#include <iostream>>
#include <

class MaxHeap {
    std::vector heap;

    void heapify(int i) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;

        if (left < heap.size() && heap[left] > heap[largest]) {
            largest = left;
        }

        if (right < heap.size() && heap[right] > heap[largest]) {
            largest = right;
        }

        if (largest != i) {
            std::swap(heap[i], heap[largest]);
            heapify(largest);
        }
    }

public:
    void insert(int key) {
        heap.push_back(key);
        int i = heap.size() - 1;

        while (i != 0 && heap[(i - 1) / 2] < heap[i]) {
            std::swap(heap[i], heap[(i - 1) / 2]);
            i = (i - 1) / 2;
        }
    }

    int extractMax() {
        if (heap.size() == 0) return -1;
        if (heap.size() == 1) {
            int root = heap[0];
            heap.pop_back();
            return root;
        }

        int root = heap[0];
        heap[0] = heap.back();
        heap.pop_back();
        heapify(0);

        return root;
    }

    int getMax() {
        return heap.size() > 0 ? heap[0] : -1;
    }

    void printHeap() {
        for (int val : heap) {
            std::cout << val << " ";
        }
        std::cout << std::endl;
    }
};

int main() {
    MaxHeap mh;
    mh.insert(10);
    mh.insert(20);
    mh.insert(15);
    mh.printHeap();
    std::cout << "Extracted Max: " << mh.extractMax() << std::endl;
    mh.printHeap();
    return 0;
}

2. Max-Heap in Python


import heapq

class MaxHeap:
    def __init__(self):
        self.heap = []

    def insert(self, key):
        heapq.heappush(self.heap, -key)

    def extract_max(self):
        if not self.heap:
            return None
        return -heapq.heappop(self.heap)

    def peek(self):
        if not self.heap:
            return None
        return -self.heap[0]

    def print_heap(self):
        print([-x for x in self.heap])

# Example usage
mh = MaxHeap()
mh.insert(10)
mh.insert(20)
mh.insert(15)
mh.print_heap()
print("Extracted Max:", mh.extract_max())
mh.print_heap()

Summary

Max-Heaps are an important data structure used for efficiently managing and retrieving the maximum element in a collection. They are particularly useful for implementing priority queues and for algorithms like heap sort. Understanding Max-Heaps and their operations can enhance your ability to handle a variety of computational problems.
