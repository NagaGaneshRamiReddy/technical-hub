CSS Grid

What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It allows you to create complex and responsive grid-based designs easily. With CSS Grid, you can control both rows and columns, making it a powerful tool for layout design.

Grid Properties

Container Properties:

  • display: grid; - Defines a grid container.
  • grid-template-columns - Defines the number and size of columns.
  • grid-template-rows - Defines the number and size of rows.
  • gap - Defines the spacing between grid items.

Item Properties:

  • grid-column - Defines how many columns an item should span.
  • grid-row - Defines how many rows an item should span.
  • grid-area - Defines where an item should be placed in the grid.

Grid template areas allow you to define the layout by naming the areas. This makes it easy to place items in specific parts of the grid.

Code Example


/* Grid Container */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
    border: 2px solid #ddd;
    padding: 10px;
}

/* Grid Items */
.grid-item {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    border: 2px solid #333;
    text-align: center;
}

/* Grid Template Areas */
.grid-template-areas {
    display: grid;
    grid-template-areas: 
        "header header header"
        "sidebar content content"
        "footer footer footer";
    gap: 10px;
    border: 2px solid #ddd;
    padding: 10px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }
