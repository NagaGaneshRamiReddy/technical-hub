HTML Input Types

Understanding HTML Input Types

HTML provides several input types that allow users to enter different kinds of data. These input types enhance the user experience by enabling appropriate controls for each data type.

Text Input

The text input type is used to collect text data from users:

<label for="username">Username:</label>
<input type="text" id="username" name="username">

Email Input

The email input type is used to ensure that the data entered follows the email address format:

<label for="email">Email:</label>
<input type="email" id="email" name="email">

Expected Output:

Password Input

The password input type hides the entered characters, commonly used for sensitive data:

<label for="password">Password:</label>
<input type="password" id="password" name="password">

Expected Output:

Number Input

The number input type allows the user to enter numerical data:

<label for="age">Age:</label>
<input type="number" id="age" name="age" min="18" max="99">

Expected Output:

Date Input

The date input type provides a date picker control for selecting a date:

<label for="birthdate">Birthdate:</label>
<input type="date" id="birthdate" name="birthdate">

Checkbox Input

The checkbox input type allows users to select or deselect options:

<label for="subscribe">Subscribe to newsletter:</label>
<input type="checkbox" id="subscribe" name="subscribe">

Radio Button Input

The radio input type allows users to select one option from a list:

<p>Gender:</p>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>
