File Handling in PHP

Overview

File handling in PHP allows you to read from and write to files on the server. This is essential for tasks such as logging, data storage, and file management.

Opening Files

To open a file, you can use the fopen() function:

    <?php
    $file = fopen("example.txt", "r"); // Opens the file for reading
    ?>

Reading Files

You can read the contents of a file using several functions:

1. fread()

    <?php
    $content = fread($file, filesize("example.txt")); // Reads the entire file
    echo $content;
    ?>

2. fgets()

    <?php
    while(!feof($file)) {
        echo fgets($file) . "<br>"; // Reads line by line
    }
    ?>

Writing to Files

To write data to a file, use the fwrite() function. You must open the file in write mode:

    <?php
    $file = fopen("example.txt", "w"); // Opens the file for writing
    fwrite($file, "Hello, World!"); // Writes data to the file
    fclose($file); // Close the file
    ?>

Appending to Files

If you want to add data without erasing the existing content, open the file in append mode:

    <?php
    $file = fopen("example.txt", "a"); // Opens the file for appending
    fwrite($file, "\nNew line added.");
    fclose($file);
    ?>

Closing Files

Always remember to close the file after you are done with it using fclose():

    <?php
    fclose($file);
    ?>

Error Handling

It's important to handle errors during file operations. You can check if a file was opened successfully:

    <?php
    $file = fopen("example.txt", "r");
    if (!$file) {
        die("Unable to open the file!");
    }
    ?>

Summary

File handling in PHP provides a way to manage files, allowing you to read, write, and manipulate file contents effectively. Mastering these functions will enhance your ability to develop robust applications.
