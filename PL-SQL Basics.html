PL/SQL Basics

What is PL/SQL?

PL/SQL (Procedural Language/SQL) is Oracle's extension of SQL that enables the writing of complex database interactions in a procedural way. It combines SQL's data manipulation power with the procedural capabilities of programming languages, making it suitable for developing applications that require business logic and data handling.

PL/SQL Block Structure

PL/SQL programs are structured as blocks. A PL/SQL block consists of three sections:

  1. Declaration Section: Where variables, constants, and cursors are defined.
  2. Execution Section: Contains SQL statements and PL/SQL code to be executed.
  3. Exception Handling Section: Handles exceptions that occur during execution.

Basic Example of a PL/SQL Block


DECLARE
    v_message VARCHAR2(100);
BEGIN
    v_message := 'Hello, PL/SQL!';
    DBMS_OUTPUT.PUT_LINE(v_message);
END;

Variables and Data Types

PL/SQL supports several data types, including:

  • NUMBER: Numeric values.
  • VARCHAR2: Variable-length character strings.
  • DATE: Date and time values.
  • BOOLEAN: TRUE or FALSE values.

Declaring Variables


DECLARE
    v_count NUMBER := 0;
    v_name VARCHAR2(50);
BEGIN
    SELECT COUNT(*) INTO v_count FROM employees;
    DBMS_OUTPUT.PUT_LINE('Total Employees: ' || v_count);
END;

Control Structures

PL/SQL includes control structures to manage the flow of execution:

  • IF Statements: Conditional execution of code.
  • LOOP Statements: Execute code repeatedly.
  • CASE Statements: Multi-way branching based on conditions.

Example of an IF Statement


DECLARE
    v_salary NUMBER := 3000;
BEGIN
    IF v_salary > 2500 THEN
        DBMS_OUTPUT.PUT_LINE('Salary is above average.');
    ELSE
        DBMS_OUTPUT.PUT_LINE('Salary is average or below.');
    END IF;
END;

Cursors

Cursors are used to handle SQL query results. There are two types:

  • Implicit Cursors: Automatically created by PL/SQL for single SQL statements.
  • Explicit Cursors: Defined by the user for queries returning multiple rows.

Example of an Explicit Cursor


DECLARE
    CURSOR emp_cursor IS SELECT first_name FROM employees;
    v_name employees.first_name%TYPE;
BEGIN
    OPEN emp_cursor;
    LOOP
        FETCH emp_cursor INTO v_name;
        EXIT WHEN emp_cursor%NOTFOUND;
        DBMS_OUTPUT.PUT_LINE('Employee Name: ' || v_name);
    END LOOP;
    CLOSE emp_cursor;
END;

Conclusion

PL/SQL is an essential tool for Oracle database developers, providing a powerful way to manage and manipulate data with procedural programming constructs. By mastering PL/SQL basics, you can enhance your database applications and improve their functionality.
