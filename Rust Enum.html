Rust Enums

Understanding Enums in Rust

Enums (short for enumerations) in Rust allow you to define a type that can be one of several variants. They are a powerful feature for creating types that can hold different kinds of data.

Defining an Enum

You can define an enum using the enum keyword followed by the name of the enum and its variants:

enum Direction {
            Up,
            Down,
            Left,
            Right,
        }

In this example, we defined an enum called Direction with four variants: Up, Down, Left, and Right.

Using Enums

To use an enum, you can create an instance of one of its variants:

fn main() {
            let move_direction = Direction::Up;
        
            match move_direction {
                Direction::Up => println!("Moving up!"),
                Direction::Down => println!("Moving down!"),
                Direction::Left => println!("Moving left!"),
                Direction::Right => println!("Moving right!"),
            }
        }

Enums with Data

Enums can also hold data associated with their variants. You can define variants with different types of data:

enum Shape {
            Circle(f64), // Holds radius
            Rectangle(f64, f64), // Holds width and height
        }
        
        fn main() {
            let circle = Shape::Circle(2.5);
            let rectangle = Shape::Rectangle(2.0, 3.0);
        
            match circle {
                Shape::Circle(radius) => println!("Circle with radius: {}", radius),
                Shape::Rectangle(width, height) => println!("Rectangle {} x {}", width, height),
            }
        }

Enums and Pattern Matching

Pattern matching is a powerful feature that allows you to handle different enum variants effectively:

fn main() {
            let shape = Shape::Rectangle(3.0, 4.0);
            
            match shape {
                Shape::Circle(radius) => println!("Circle with radius: {}", radius),
                Shape::Rectangle(width, height) => println!("Rectangle {} x {}", width, height),
            }
        }

Summary

Enums in Rust provide a way to define types that can have multiple variants, including variants that hold data. They are often used with pattern matching, making it easier to handle complex data types in a safe and expressive way.
