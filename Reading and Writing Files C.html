Reading and Writing files in C

Introduction

File handling in C allows you to read from and write to files. To perform file operations, you use file pointers along with standard library functions provided in stdio.h. This section will explain how to read from and write to files in C.

Opening Files

Before you can read from or write to a file, you need to open it using the fopen function. This function returns a file pointer which is used for subsequent file operations.

#include <stdio.h>

int main() {
    FILE *filePointer;
    filePointer = fopen("example.txt", "r"); // Open file for reading

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Perform file operations here

    fclose(filePointer);
    return 0;
}

In this example, fopen opens example.txt for reading. If the file cannot be opened, fopen returns NULL.

Reading from Files

There are several functions for reading data from a file. The most commonly used functions are:

  • fgetc: Reads a single character from the file.
  • fgets: Reads a string of characters from the file.
  • fread: Reads a block of data from the file.

Example: Reading Characters

#include <stdio.h>

int main() {
    FILE *filePointer;
    char ch;

    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Read and print each character from the file
    while ((ch = fgetc(filePointer)) != EOF) {
        printf("%c", ch);
    }

    fclose(filePointer);
    return 0;
}

Example: Reading Strings

#include <stdio.h>

int main() {
    FILE *filePointer;
    char str[100];

    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Read and print each line from the file
    while (fgets(str, sizeof(str), filePointer) != NULL) {
        printf("%s", str);
    }

    fclose(filePointer);
    return 0;
}

Writing to Files

Similarly, several functions are available for writing data to a file:

  • fputc: Writes a single character to the file.
  • fputs: Writes a string of characters to the file.
  • fwrite: Writes a block of data to the file.

Example: Writing Characters

#include <stdio.h>

int main() {
    FILE *filePointer;

    filePointer = fopen("example.txt", "w");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Write characters to the file
    fputc('H', filePointer);
    fputc('e', filePointer);
    fputc('l', filePointer);
    fputc('l', filePointer);
    fputc('o', filePointer);

    fclose(filePointer);
    return 0;
}

Example: Writing Strings

#include <stdio.h>

int main() {
    FILE *filePointer;

    filePointer = fopen("example.txt", "w");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Write a string to the file
    fputs("Hello, World!\n", filePointer);

    fclose(filePointer);
    return 0;
}

Appending to Files

To append data to an existing file without overwriting its contents, open the file in append mode using the "a" mode with fopen:

#include <stdio.h>

int main() {
    FILE *filePointer;

    filePointer = fopen("example.txt", "a");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Append a string to the file
    fputs("Appended text.\n", filePointer);

    fclose(filePointer);
    return 0;
}

Closing Files

Always close the file after performing file operations using the fclose function. This ensures that all data is properly written and resources are released.

Summary

File handling in C is crucial for reading from and writing to files. By using file pointers and the appropriate functions from the stdio.h library, you can manage file operations effectively. Remember to open files with the correct mode, perform necessary operations, and close files to ensure data integrity and proper resource management.
