Structures and Unions in C

Structures in C

Structures in C are user-defined data types that allow you to group different types of variables under a single name. Each variable in a structure is called a member. Structures are useful for organizing complex data and managing related data items.

Declaration and Definition

To declare a structure, use the struct keyword followed by the structure name and its members. Here’s an example:

#include <stdio.h>

struct Person {
    char name[50];
    int age;
    float height;
};

This defines a structure named Person with three members: name, age, and height.

Creating and Accessing Structure Variables

You can create variables of the structure type and access its members using the dot operator:

#include <stdio.h>

struct Person {
    char name[50];
    int age;
    float height;
};

int main() {
    struct Person person1;
    // Assign values to members
    strcpy(person1.name, "Alice");
    person1.age = 30;
    person1.height = 5.6;
    
    // Print values of members
    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);
    printf("Height: %.1f\n", person1.height);
    
    return 0;
}

Unions in C

Unions in C are similar to structures but with a key difference: all members of a union share the same memory location. This means that a union can store only one member value at a time, but the size of the union is the size of its largest member.

Declaration and Definition

To declare a union, use the union keyword followed by the union name and its members. Here’s an example:

#include <stdio.h>

union Data {
    int intValue;
    float floatValue;
    char charValue;
};

This defines a union named Data with three members: intValue, floatValue, and charValue.

Creating and Accessing Union Variables

You can create variables of the union type and access its members using the dot operator, but remember that only one member can hold a value at a time:

#include <stdio.h>

union Data {
    int intValue;
    float floatValue;
    char charValue;
};

int main() {
    union Data data;
    
    // Assign value to intValue
    data.intValue = 10;
    printf("Int Value: %d\n", data.intValue);
    
    // Assign value to floatValue (overwrites intValue)
    data.floatValue = 3.14;
    printf("Float Value: %.2f\n", data.floatValue);
    
    // Assign value to charValue (overwrites floatValue)
    data.charValue = 'A';
    printf("Char Value: %c\n", data.charValue);
    
    return 0;
}

Summary

Structures and unions are powerful tools in C programming for managing complex data. Structures allow you to group different types of data under a single name, while unions enable you to use the same memory location for different data types, optimizing memory usage. Understanding these concepts is crucial for writing efficient and organized C programs.
