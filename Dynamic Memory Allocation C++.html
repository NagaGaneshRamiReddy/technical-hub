Dynamic memory allocation in C++

Introduction

Dynamic memory allocation in C++ allows you to allocate and deallocate memory at runtime using pointers. This is useful when you need to allocate memory based on runtime conditions or handle data structures that change size during execution. The C++ Standard Library provides several operators and functions for managing dynamic memory.

Using new and delete

The new operator is used to allocate memory for a variable or an array dynamically. The delete operator is used to deallocate memory that was previously allocated using new.

#include <iostream>

int main() {
    // Dynamically allocate memory for a single integer
    int *p = new int;  // Allocate memory
    *p = 10;  // Assign a value to the allocated memory
    std::cout << "Value: " << *p << std::endl;
    delete p;  // Deallocate memory

    // Dynamically allocate memory for an array of integers
    int size = 5;
    int *arr = new int[size];  // Allocate memory for an array
    for (int i = 0; i < size; ++i) {
        arr[i] = i * 10;
    }

    std::cout << "Array elements:" << std::endl;
    for (int i = 0; i < size; ++i) {
        std::cout << "Element " << i << ": " << arr[i] << std::endl;
    }

    delete[] arr;  // Deallocate memory for the array

    return 0;
}

In this example:

  • new int; allocates memory for a single integer.
  • delete p; deallocates the memory.
  • new int[size]; allocates memory for an array of integers.
  • delete[] arr; deallocates memory for the array.

Memory Management Best Practices

Proper management of dynamically allocated memory is crucial to avoid memory leaks and undefined behavior. Here are some best practices:

  • Always deallocate memory: Use delete and delete[] to free memory that is no longer needed.
  • Set pointers to nullptr after deletion: After calling delete, set the pointer to nullptr to avoid dangling pointers.
  • Use smart pointers: Consider using smart pointers from the memory library (e.g., std::unique_ptr and std::shared_ptr) to manage dynamic memory automatically.

Smart Pointers

Smart pointers provide automatic memory management and help prevent memory leaks. They are part of the C++ Standard Library and include std::unique_ptr and std::shared_ptr.

#include <iostream>
#include <memory>  // For smart pointers

int main() {
    // Unique pointer
    std::unique_ptr uniquePtr = std::make_unique(10);
    std::cout << "Unique pointer value: " << *uniquePtr << std::endl;

    // Shared pointer
    std::shared_ptr sharedPtr1 = std::make_shared(20);
    std::shared_ptr sharedPtr2 = sharedPtr1;  // Both pointers share ownership
    std::cout << "Shared pointer 1 value: " << *sharedPtr1 << std::endl;
    std::cout << "Shared pointer 2 value: " << *sharedPtr2 << std::endl;

    return 0;
}

In this example:

  • std::unique_ptr uniquePtr = std::make_unique(10); creates a unique pointer to an integer.
  • std::shared_ptr sharedPtr1 = std::make_shared(20); creates a shared pointer to an integer.
  • std::shared_ptr sharedPtr2 = sharedPtr1; shares ownership of the integer with another shared pointer.

Summary

Dynamic memory allocation allows for flexible and efficient management of memory during runtime. Understanding how to use new and delete operators, as well as smart pointers, is essential for effective memory management in C++. Following best practices for memory management helps avoid common pitfalls like memory leaks and dangling pointers.
