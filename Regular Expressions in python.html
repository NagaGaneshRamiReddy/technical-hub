Regular Expressions in Python

What are Regular Expressions?

Regular expressions (regex or regexp) are sequences of characters that define a search pattern. They are used for pattern matching, searching, and text manipulation in strings. Regular expressions provide a powerful and flexible way to process text data, validate inputs, and extract information.

Using Regular Expressions in Python

In Python, regular expressions are handled by the re module. This module provides functions and methods for matching patterns and performing operations on strings.

Basic Syntax

Here are some basic components of regular expressions:

  • Literal Characters: Matches the exact characters. Example: hello matches "hello".
  • Dot (.): Matches any single character except a newline. Example: a.c matches "abc", "a1c", etc.
  • Star (*): Matches zero or more of the preceding element. Example: a* matches "", "a", "aa", etc.
  • Plus (+): Matches one or more of the preceding element. Example: a+ matches "a", "aa", "aaa", etc.
  • Question Mark (?): Matches zero or one of the preceding element. Example: a? matches "", "a".
  • Brackets ([]): Matches any one of the characters inside the brackets. Example: [abc] matches "a", "b", or "c".
  • Caret (^): Matches the start of the string. Example: ^abc matches "abc" at the beginning of the string.
  • Dollar ($): Matches the end of the string. Example: abc$ matches "abc" at the end of the string.
  • Pipes (|): Matches either the pattern before or after the pipe. Example: abc|def matches "abc" or "def".

Common Functions

Some common functions provided by the re module include:

  • re.match(pattern, string): Checks if the pattern matches at the beginning of the string.
  • re.search(pattern, string): Searches for the first occurrence of the pattern in the string.
  • re.findall(pattern, string): Returns a list of all matches of the pattern in the string.
  • re.finditer(pattern, string): Returns an iterator yielding match objects for all matches of the pattern in the string.
  • re.sub(pattern, repl, string): Replaces occurrences of the pattern with the replacement string.

Examples

Here are some practical examples of using regular expressions in Python:

import re

# Example: Find all email addresses in a text
text = "Contact us at support@example.com or sales@example.org"
pattern = r'\b[\w\.-]+@[\w\.-]+\.\w+\b'
emails = re.findall(pattern, text)
print(emails)  # Output: ['support@example.com', 'sales@example.org']

# Example: Replace all digits in a string with a '#'
text = "My phone number is 123-456-7890"
pattern = r'\d'
replacement = '#'
result = re.sub(pattern, replacement, text)
print(result)  # Output: 'My phone number is ###-###-####'

Key Points about Regular Expressions

  • Powerful Tool: Regular expressions are a powerful tool for text processing and pattern matching.
  • Flexibility: Regular expressions offer flexibility in defining patterns, making them suitable for complex text processing tasks.
  • Readability: While regular expressions are powerful, they can sometimes be difficult to read and understand. Itâ€™s important to use them judiciously.
  • Performance: Regular expressions can be computationally expensive, so optimize patterns and usage when dealing with large datasets.
