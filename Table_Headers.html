Table Headers in HTML

Understanding Table Headers

Table headers in HTML help define the titles of each column in a table. They are important for improving the accessibility and readability of tables.

Creating Table Headers

Table headers are created using the <th> tag. The headers can be defined for each column of the table.

Example: A Simple Table with Headers

Here’s how to create a table with headers:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>25</td>
        <td>Los Angeles</td>
    </tr>
</table>

Expected Output:

Name Age City
Alice 30 New York
Bob 25 Los Angeles

Example: Table with Multiple Headers

Here’s an example of a table with multiple headers that span multiple columns:

<table>
    <tr>
        <th colspan="2">Employee Details</th>
        <th>Department</th>
    </tr>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Department</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
        <td>Marketing</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>25</td>
        <td>Development</td>
    </tr>
</table>

Expected Output:

Employee Details Department
Name Age Department
Alice 30 Marketing
Bob 25 Development

Example: Table with Rowspan

You can also use the rowspan attribute to combine multiple rows under a single header:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th rowspan="2">City</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>25</td>
    </tr>
</table>

Expected Output:

Name Age City
Alice 30
Bob 25

Summary

Table headers are crucial for defining the structure of your data in tables. They enhance the table's readability and accessibility, making it easier for users to understand the content.
