Lambda expressions in C++

Introduction

Lambda expressions are a feature introduced in C++11 that allows you to define anonymous functions directly within your code. These functions can be used for short, localized tasks, making them particularly useful for algorithms and callbacks. Lambda expressions provide a way to write concise and efficient code, especially for operations on containers and other data structures.

Basic Syntax

The syntax of a lambda expression in C++ is as follows:

[capture](parameters) -> return_type { body }
  • Capture: Specifies which variables from the surrounding scope are available inside the lambda.
  • Parameters: The input parameters for the lambda function, similar to function parameters.
  • Return Type: The type of value the lambda returns (optional). If omitted, the return type is deduced from the return statements.
  • Body: The code that defines what the lambda does.

Examples

Here are some examples of lambda expressions:

Basic Lambda

#include <iostream>

int main() {
    auto greet = []() {
        std::cout << "Hello, World!" << std::endl;
    };

    greet();  // Calls the lambda function

    return 0;
}

In this example:

  • [] is the capture clause (empty in this case, meaning no variables are captured).
  • () are the parameters (none in this case).
  • std::cout << "Hello, World!" << std::endl; is the body of the lambda function.

Lambda with Parameters

#include <iostream>

int main() {
    auto add = [](int a, int b) -> int {
        return a + b;
    };

    int result = add(5, 3);  // Calls the lambda function with arguments 5 and 3
    std::cout << "Result: " << result << std::endl;

    return 0;
}

In this example:

  • [ ] captures no variables.
  • (int a, int b) specifies the parameters of the lambda.
  • -> int indicates the return type of the lambda.
  • return a + b; is the body of the lambda function, which adds the two parameters.

Lambda with Capture

#include <iostream>

int main() {
    int x = 10;
    auto print_x = [x]() {
        std::cout << "x = " << x << std::endl;
    };

    print_x();  // Calls the lambda function

    return 0;
}

In this example:

  • [x] captures the variable x by value. The lambda can access the value of x but cannot modify it.

Advanced Usage

Lambda expressions also support more advanced features such as:

  • Capture by Reference: Capture variables by reference using [&] so that modifications to captured variables are reflected outside the lambda.
  • Mutable Lambdas: Use the mutable keyword to allow modification of captured variables.
  • Generic Lambdas: Use auto in parameters to create generic lambdas that work with different types.

Capture by Reference

#include <iostream>

int main() {
    int x = 10;
    auto increment = [&x]() {
        x++;
        std::cout << "x = " << x << std::endl;
    };

    increment();  // Calls the lambda function
    std::cout << "x after increment = " << x << std::endl;

    return 0;
}

In this example:

  • [&x] captures x by reference. The lambda function modifies the value of x directly.

Mutable Lambdas

#include <iostream>

int main() {
    int x = 10;
    auto print_and_modify = [x]() mutable {
        x += 5;
        std::cout << "Modified x = " << x << std::endl;
    };

    print_and_modify();  // Calls the lambda function

    return 0;
}

In this example:

  • mutable allows the lambda to modify the captured value of x.

Generic Lambdas

#include <iostream>

int main() {
    auto print = [](auto value) {
        std::cout << "Value = " << value << std::endl;
    };

    print(10);       // Prints integer
    print(3.14);     // Prints double
    print("Hello");  // Prints string

    return 0;
}

In this example:

  • auto in the parameter list allows the lambda to accept arguments of any type.
