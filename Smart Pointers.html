Smart Pointers in Rust

Understanding Smart Pointers

Smart pointers in Rust are data structures that provide more functionality than regular pointers. They manage memory automatically, ensuring memory safety and preventing common issues such as dangling pointers and memory leaks.

Types of Smart Pointers

Rust has several types of smart pointers, each with unique features:

1. Box

Box<T> is used for heap allocation. It allows you to store data on the heap while providing ownership semantics.

fn main() {
            let b = Box::new(5);
            println!("Boxed value: {}", b);
        }

2. Rc

Rc<T> stands for "Reference Counted". It enables multiple ownership of data by keeping track of the number of references to a value. This is useful for sharing data without copying it.

use std::rc::Rc;
        
        fn main() {
            let rc1 = Rc::new(5);
            let rc2 = Rc::clone(&rc1);
            
            println!("Value: {}", rc1);
            println!("Reference Count: {}", Rc::strong_count(&rc1));
        }

3. Arc

Arc<T> (Atomic Reference Counted) is similar to Rc but is thread-safe. It is designed for concurrent programming where data needs to be shared across threads.

use std::sync::Arc;
        
        fn main() {
            let arc1 = Arc::new(5);
            let arc2 = Arc::clone(&arc1);
            
            println!("Value: {}", arc1);
            println!("Reference Count: {}", Arc::strong_count(&arc1));
        }

4. RefCell

RefCell<T> allows for interior mutability, enabling you to mutate data even if the RefCell itself is immutable. It checks borrow rules at runtime.

use std::cell::RefCell;
        
        fn main() {
            let x = RefCell::new(5);
            *x.borrow_mut() = 10; // Mutable borrow
            println!("Value: {}", x.borrow());
        }

Combining Smart Pointers

Smart pointers can be used together to create complex data structures. For example, combining Rc with RefCell allows multiple ownership with mutable access:

use std::rc::Rc;
        use std::cell::RefCell;
        
        fn main() {
            let value = Rc::new(RefCell::new(5));
            {
                let mut val = value.borrow_mut();
                *val += 5;
            }
            println!("Updated Value: {}", value.borrow());
        }
