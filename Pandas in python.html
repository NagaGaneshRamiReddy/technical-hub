Pandas in Python

What is Pandas ?

Pandas is a powerful and flexible open-source data manipulation and analysis library for Python. It provides data structures and functions needed to work with structured data seamlessly. Pandas is widely used in data science and machine learning for data cleaning, transformation, and analysis.

Basics of Pandas

To use Pandas, you first need to install it (if not already installed) and import it into your Python script:

pip install pandas
import pandas as pd

Data Structures

Pandas primarily provides two data structures:

  • Series: A one-dimensional labeled array capable of holding any data type.
  • DataFrame: A two-dimensional labeled data structure with columns of potentially different types.

Creating a Series

Series can be created from lists, dictionaries, or scalar values.

# Create a Series from a list
series = pd.Series([10, 20, 30, 40])
print("Series:\n", series)

# Create a Series from a dictionary
data = {'a': 1, 'b': 2, 'c': 3}
series_from_dict = pd.Series(data)
print("Series from Dictionary:\n", series_from_dict)

Creating a DataFrame

DataFrames can be created from dictionaries, lists of lists, or other DataFrames.

# Create a DataFrame from a dictionary
data = {'Name': ['Alice', 'Bob', 'Charlie'],
        'Age': [25, 30, 35],
        'City': ['New York', 'Los Angeles', 'Chicago']}
df = pd.DataFrame(data)
print("DataFrame:\n", df)

# Create a DataFrame from a list of lists
data = [['Alice', 25, 'New York'],
        ['Bob', 30, 'Los Angeles'],
        ['Charlie', 35, 'Chicago']]
df_from_list = pd.DataFrame(data, columns=['Name', 'Age', 'City'])
print("DataFrame from List:\n", df_from_list)

DataFrame Operations

Here are some common operations you can perform with DataFrames:

# Selecting columns
print("Select 'Name' column:\n", df['Name'])

# Filtering rows
filtered_df = df[df['Age'] > 30]
print("Filtered DataFrame:\n", filtered_df)

# Adding a new column
df['Occupation'] = ['Engineer', 'Doctor', 'Artist']
print("DataFrame with New Column:\n", df)

# Dropping a column
df_dropped = df.drop(columns=['Occupation'])
print("DataFrame with Column Dropped:\n", df_dropped)

Data Aggregation and Grouping

Pandas provides powerful tools for data aggregation and grouping.

# Group by a column and aggregate
grouped_df = df.groupby('City').mean()
print("Grouped DataFrame:\n", grouped_df)

# Aggregating data
agg_df = df.agg({'Age': ['min', 'max', 'mean']})
print("Aggregated DataFrame:\n", agg_df)

Handling Missing Data

Pandas provides functions to handle missing data effectively.

# Fill missing values
df_with_nan = df.copy()
df_with_nan.loc[1, 'Age'] = None
df_filled = df_with_nan.fillna(df['Age'].mean())
print("DataFrame with Missing Values Filled:\n", df_filled)

# Drop rows with missing values
df_dropped_nan = df_with_nan.dropna()
print("DataFrame with Missing Values Dropped:\n", df_dropped_nan)

Summary

Pandas is a fundamental library for data manipulation and analysis in Python. It provides powerful data structures like Series and DataFrame and offers extensive functionality for data cleaning, transformation, and aggregation. Mastering Pandas is essential for data analysis and data science tasks.
