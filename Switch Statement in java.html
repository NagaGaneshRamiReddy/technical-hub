switch statement in Java

Introduction

The switch statement in Java provides a way to execute different blocks of code based on the value of an expression. It is often used as an alternative to multiple if-else statements when dealing with a single variable that can take on multiple discrete values. This guide covers the syntax and usage of the switch statement, including its variations and enhancements in recent Java versions.

1. Basic Syntax

The basic syntax of a switch statement includes a switch expression, multiple case labels, and an optional default case:

int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    case 4:
        System.out.println("Thursday");
        break;
    case 5:
        System.out.println("Friday");
        break;
    case 6:
        System.out.println("Saturday");
        break;
    case 7:
        System.out.println("Sunday");
        break;
    default:
        System.out.println("Invalid day");
}

In this example, the switch statement checks the value of day and executes the corresponding block of code. If day does not match any of the case labels, the default case is executed.

2. The break Statement

The break statement is used to terminate the switch block and prevent the execution from falling through to the next case. Without break, the code will continue executing the subsequent case blocks until a break is encountered or the switch block ends:

int number = 2;

switch (number) {
    case 1:
        System.out.println("One");
        // No break here
    case 2:
        System.out.println("Two");
        break;
    case 3:
        System.out.println("Three");
        break;
    default:
        System.out.println("Not One, Two, or Three");
}

In this example, the output will be "Two" and "Not One, Two, or Three" because there is no break after case 1.

3. Multiple Cases

You can group multiple case labels to execute the same block of code for different values:

char grade = 'B';

switch (grade) {
    case 'A':
    case 'B':
    case 'C':
        System.out.println("Passed");
        break;
    case 'D':
    case 'E':
        System.out.println("Failed");
        break;
    default:
        System.out.println("Invalid grade");
}

Here, grades 'A', 'B', and 'C' will all print "Passed" because they share the same code block.

4. Switch Expression (Java 12 and Later)

Starting with Java 12, the switch statement can also be used as an expression, which allows for a more concise syntax with the use of the -> operator:

int number = 2;

String result = switch (number) {
    case 1 -> "One";
    case 2 -> "Two";
    case 3 -> "Three";
    default -> "Unknown";
};

System.out.println(result); // Output: Two

This new syntax is more compact and can return a value directly from the switch expression.

5. Enhanced Switch Statement (Java 14 and Later)

Java 14 introduced a more expressive form of the switch statement, which includes the yield keyword for returning values from a switch block:

int number = 2;

String result = switch (number) {
    case 1 -> "One";
    case 2 -> "Two";
    case 3 -> "Three";
    default -> {
        System.out.println("Default case");
        yield "Unknown";
    }
};

System.out.println(result); // Output: Two

This syntax allows you to execute multiple statements in a case block before yielding a result.

Summary

The switch statement is a powerful tool for handling multiple discrete values in Java. It provides a clear and efficient way to manage conditional logic based on the value of a single expression. With enhancements in recent Java versions, the switch statement has become even more versatile and expressive, making it an important feature in modern Java programming.
