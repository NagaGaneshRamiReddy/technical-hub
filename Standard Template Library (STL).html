Standard Template Library (STL) in C++

Introduction

The Standard Template Library (STL) is a powerful set of C++ template classes to provide general-purpose classes and functions with templates. It includes a collection of algorithms and data structures that are fundamental to efficient software development.

Components of STL

The STL consists of four main components:

  • Containers: These are data structures that store objects. Common containers include vectors, lists, and maps.
  • Algorithms: These are functions that operate on containers to perform operations such as sorting, searching, and modifying data.
  • Iterators: These are objects that provide a way to access the elements of containers. They work like pointers to navigate through the data in containers.
  • Function Objects: Also known as functors, these are objects that can be called as if they were functions. They are used to customize algorithms.

Common STL Containers

STL provides several container classes:

  • std::vector: A dynamic array that can resize itself automatically. It provides fast random access to elements.
  • std::list: A doubly linked list that allows fast insertion and deletion of elements.
  • std::deque: A double-ended queue that allows fast insertions and deletions at both ends.
  • std::set: A collection of unique elements, usually implemented as a balanced tree.
  • std::map: A collection of key-value pairs, where each key is unique. It is typically implemented as a balanced tree.
  • std::unordered_set: A collection of unique elements implemented using hash tables, providing average constant-time complexity for insertions and lookups.
  • std::unordered_map: Similar to `std::unordered_set` but with key-value pairs.

Example Usage of STL Containers

Here’s how you can use some common STL containers:

1. Using std::vector

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    for (int num : numbers) {
        std::cout << num << " ";
    }

    return 0;
}

2. Using std::map

#include <iostream>
#include <map>

int main() {
    std::map<std::string, int> age;
    age["Alice"] = 30;
    age["Bob"] = 25;

    for (const auto &pair : age) {
        std::cout << pair.first << " is " << pair.second << " years old" << std::endl;
    }

    return 0;
}

STL Algorithms

STL algorithms operate on containers and iterators. Here are a few examples:

1. Sorting Elements

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {5, 3, 8, 1, 2};

    std::sort(numbers.begin(), numbers.end());

    for (int num : numbers) {
        std::cout << num << " ";
    }

    return 0;
}

2. Finding Elements

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {5, 3, 8, 1, 2};
    auto it = std::find(numbers.begin(), numbers.end(), 8);

    if (it != numbers.end()) {
        std::cout << "Element found: " << *it << std::endl;
    } else {
        std::cout << "Element not found" << std::endl;
    }

    return 0;
}

Iterators

Iterators provide a way to traverse the elements of a container. Here are some common types:

  • Input Iterators: Read elements from a container.
  • Output Iterators: Write elements to a container.
  • Forward Iterators: Read and write elements and move forward through the container.
  • Bidirectional Iterators: Move forward and backward through the container.
  • Random Access Iterators: Move to any position in the container efficiently.

Function Objects (Functors)

Function objects are objects that can be used as if they were functions. They are useful for customizing algorithms. Here’s an example:

#include <iostream>
#include <algorithm>
#include <vector>

struct GreaterThanFive {
    bool operator()(int x) const {
        return x > 5;
    }
};

int main() {
    std::vector<int> numbers = {2, 7, 5, 10, 3};
    auto it = std::find_if(numbers.begin(), numbers.end(), GreaterThanFive());

    if (it != numbers.end()) {
        std::cout << "First number greater than 5: " << *it << std::endl;
    }

    return 0;
}
