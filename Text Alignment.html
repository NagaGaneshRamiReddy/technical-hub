Text Alignment in CSS

What is Text Alignment in CSS?

Text alignment in CSS is used to control the horizontal alignment of text within an element. It determines how text is positioned in relation to the containing block, allowing you to align text to the left, center, right, or justify it across the width of its container.

Key Text Alignment Properties

1. Text Align: Left

The text-align property with the value left aligns the text to the left edge of its container.


/* Align text to the left */
.left-align {
    text-align: left;
}

This paragraph is aligned to the left.

2. Text Align: Center

The text-align property with the value center centers the text within its container.


/* Align text to the center */
.center-align {
    text-align: center;
}

This paragraph is centered.

3. Text Align: Right

The text-align property with the value right aligns the text to the right edge of its container.


/* Align text to the right */
.right-align {
    text-align: right;
}

This paragraph is aligned to the right.

4. Text Align: Justify

The text-align property with the value justify adjusts the text so that it stretches to align with both the left and right edges of its container, creating a clean, uniform look.


/* Justify text */
.justify-align {
    text-align: justify;
}

This paragraph is justified. Notice how the text stretches to fit both the left and right edges of the container, creating a neat, aligned look on both sides.

Summary

Text alignment in CSS allows you to control how text is positioned within its container. By using properties like text-align with values such as left, center, right, and justify, you can create various text alignments to enhance the readability and appearance of your web pages.
