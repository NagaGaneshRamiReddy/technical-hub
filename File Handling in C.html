File Handling in C

What is File Handling?

File handling in C programming involves performing operations such as reading from and writing to files. C provides a set of standard library functions to work with files, enabling you to manage file input and output efficiently.

Basic File Operations

The main file operations in C include opening a file, reading from it, writing to it, and closing it. These operations are performed using standard library functions defined in the stdio.h header file.

Opening a File

To open a file, use the fopen function. This function returns a file pointer that is used to perform further operations on the file. The fopen function takes two arguments: the file name and the mode in which the file is to be opened:

FILE *fopen(const char *filename, const char *mode);

Common modes include:

  • "r": Read mode. Opens the file for reading. The file must exist.
  • "w": Write mode. Opens the file for writing. If the file exists, its contents are erased. If it does not exist, a new file is created.
  • "a": Append mode. Opens the file for writing at the end of the file. If the file does not exist, a new file is created.
  • "r+": Read and write mode. Opens the file for both reading and writing. The file must exist.
  • "w+": Write and read mode. Opens the file for both reading and writing. If the file exists, its contents are erased. If it does not exist, a new file is created.

Example: Opening a File

Here’s an example of opening a file for reading:

#include <stdio.h>

int main() {
    FILE *filePointer;
    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // File operations here

    fclose(filePointer);
    return 0;
}

This example opens a file named example.txt in read mode. If the file cannot be opened, it prints an error message and returns a non-zero value.

Reading from a File

To read data from a file, use functions like fgetc, fgets, or fread:

  • fgetc: Reads a single character from the file.
  • fgets: Reads a string of characters from the file.
  • fread: Reads a block of data from the file.

Example: Reading from a File

The following example reads characters from a file and prints them:

#include <stdio.h>>

int main() {
    FILE *filePointer;
    char ch;

    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    while ((ch = fgetc(filePointer)) != EOF) {
        printf("%c", ch);
    }

    fclose(filePointer);
    return 0;
}

This program reads characters one by one from example.txt and prints them until the end of the file (EOF) is reached.

Writing to a File

To write data to a file, use functions like fputc, fputs, or fwrite:

  • fputc: Writes a single character to the file.
  • fputs: Writes a string of characters to the file.
  • fwrite: Writes a block of data to the file.

Example: Writing to a File

The following example writes a string to a file:

#include <stdio.h>

int main() {
    FILE *filePointer;

    filePointer = fopen("output.txt", "w");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    fprintf(filePointer, "Hello, World!\n");

    fclose(filePointer);
    return 0;
}

This program creates or overwrites output.txt and writes the string "Hello, World!" to it.

Closing a File

After performing file operations, it’s important to close the file using the fclose function to release the file resources:

int fclose(FILE *stream);

This function closes the file associated with the file pointer stream and returns 0 if successful, or EOF if an error occurs.

Example: Closing a File

The fclose function is used in the previous examples to close the file once the operations are complete.

Handling File Errors

It’s good practice to check for errors while performing file operations. Functions like fopen return NULL if the file cannot be opened, and you can use feof, ferror, and clearerr to handle errors:

  • feof: Checks if the end-of-file indicator is set.
  • ferror: Checks for errors during file operations.
  • clearerr: Clears the end-of-file and error indicators.

Summary

File handling is an essential aspect of C programming that allows you to read from and write to files. Understanding how to open, read, write, and close files, as well as handling file errors, is crucial for effective file management in your programs.
