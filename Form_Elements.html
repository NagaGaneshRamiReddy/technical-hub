HTML Form Elements

Understanding HTML Form Elements

HTML provides various form elements to gather input from users. These elements are essential for building interactive and dynamic web applications.

Text Input

Text input fields allow users to enter single-line text data:

<label for="name">Name:</label>
<input type="text" id="name" name="name" placeholder="Enter your name">

Password Input

Password fields mask the input, hiding the text as it's entered:

<label for="password">Password:</label>
<input type="password" id="password" name="password" placeholder="Enter your password">

Expected Output:

Email Input

Email input ensures that the user enters a valid email format:

<label for="email">Email:</label>
<input type="email" id="email" name="email" placeholder="Enter your email">

Expected Output:

Number Input

Number input fields accept only numeric values:

<label for="age">Age:</label>
<input type="number" id="age" name="age" min="1" max="120" placeholder="Enter your age">

Expected Output:

Radio Buttons

Radio buttons allow users to select one option from a set:

<label>Gender:</label>
<input type="radio" id="male" name="gender" value="male"> <label for="male">Male</label>
<input type="radio" id="female" name="gender" value="female"> <label for="female">Female</label>

Expected Output:

Checkboxes

Checkboxes allow users to select multiple options:

<label>Interests:</label>
<input type="checkbox" id="sports" name="interest" value="sports"> <label for="sports">Sports</label>
<input type="checkbox" id="music" name="interest" value="music"> <label for="music">Music</label>

Expected Output:

Dropdown Menu (Select)

A dropdown menu lets users choose one option from a list:

<label for="country">Country:</label>
<select id="country" name="country">
    <option value="us">United States</option>
    <option value="uk">United Kingdom</option>
    <option value="ca">Canada</option>
</select>

Text Area

A text area allows users to enter multi-line text:

<label for="comments">Comments:</label>
<textarea id="comments" name="comments" rows="4" cols="50"></textarea>

Submit Button

The submit button submits the form data:

Summary

HTML form elements are critical for collecting user input. They include text fields, radio buttons, checkboxes, dropdowns, and more. These elements allow you to create interactive forms that capture and submit data efficiently.
