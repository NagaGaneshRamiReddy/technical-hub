Control Statements in Python

1. Conditional Statements

Conditional statements allow you to execute certain blocks of code based on specific conditions:

1.1. if Statement

The if statement executes a block of code if its condition evaluates to True:

age = 20

if age >= 18:
    print("You are an adult.")

1.2. if-else Statement

The if-else statement allows for an alternative block of code if the condition is False:

age = 16

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

1.3. if-elif-else Statement

The if-elif-else statement allows multiple conditions to be checked:

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: D")

2. Loops

Loops are used to execute a block of code repeatedly based on a condition:

2.1. for Loop

The for loop iterates over a sequence (like a list, tuple, or string) and executes a block of code for each item:

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

2.2. while Loop

The while loop repeatedly executes a block of code as long as its condition remains True:

count = 0

while count < 5:
    print(count)
    count += 1

2.3. break Statement

The break statement exits the nearest enclosing loop, regardless of the loop's condition:

for i in range(10):
    if i == 5:
        break
    print(i)

2.4. continue Statement

The continue statement skips the rest of the code inside the current loop iteration and proceeds to the next iteration:

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

2.5. pass Statement

The pass statement is a placeholder that does nothing and is used where code is syntactically required but not needed:

for i in range(10):
    if i % 2 == 0:
        pass  # Placeholder for future code
    else:
        print(i)

3. Summary

Control statements in Python, including conditional statements and loops, allow you to control the flow of your program based on conditions and repetitions. Mastering these constructs is crucial for creating dynamic and responsive code.
