Media Queries


Introduction to Media Queries

Media queries are a powerful feature of CSS that allow you to apply styles based on the conditions of the viewport, such as its width, height, and resolution. They enable responsive web design by adapting the layout and styles to different devices, ensuring that your website is user-friendly on all screens.

Basic Syntax

The basic syntax of a media query consists of the @media rule followed by a media type and condition:

@media screen and (max-width: 600px) {
    /* CSS rules here */
}

In this example, the CSS rules within the curly braces will only apply if the viewport width is 600 pixels or less.

Example: Changing Background Color

Here's an example of how to change the background color of the body based on the viewport size:

body {
    background-color: lightblue;
}

@media screen and (max-width: 600px) {
    body {
        background-color: lightcoral;
    }
}

Expected Output:

This is a sample paragraph. Resize the window to see the background color change!

Combining Multiple Conditions

You can also combine multiple conditions in a single media query using and:

@media screen and (max-width: 600px) and (orientation: landscape) {
    /* CSS rules here */
}

Conclusion

Media queries are essential for creating responsive web designs that adapt to various screen sizes and orientations. By using media queries, you can improve the user experience and accessibility of your website on different devices.
