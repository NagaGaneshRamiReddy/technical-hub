CSS Syntax

Introduction to CSS Syntax

Cascading Style Sheets (CSS) syntax is the way you write CSS rules. It consists of selectors and declaration blocks. Understanding CSS syntax is essential for applying styles effectively to HTML elements.

Basic Syntax Structure

CSS rules are composed of a selector and a declaration block. The syntax looks like this:


selector {
    property: value;
}

Selector: Specifies the HTML element(s) you want to style.

Property: Defines what aspect of the element you want to style (e.g., color, font-size).

Value: The value assigned to the property (e.g., blue, 16px).

Example of CSS Syntax

Hereâ€™s an example of a CSS rule that changes the color and font size of all <h1> elements:


h1 {
    color: blue;
    font-size: 24px;
}

Selectors

Selecting elements correctly is crucial for applying styles. Here are some common types of selectors:

  • Type Selector: Targets elements by their type, such as h1, p.
  • Class Selector: Targets elements with a specific class attribute, such as .example.
  • ID Selector: Targets an element with a specific ID, such as #header.

Declaration Block

A declaration block is enclosed in curly braces { } and contains one or more declarations. Each declaration is composed of a property and a value, separated by a colon :, and ended with a semicolon ;. For example:


p {
    color: red;
    margin: 10px;
}

Getting Started with CSS

To start working with CSS, follow these steps:

  • Learn Basic Syntax: Understand how selectors, properties, and values work together.
  • Create a CSS File: Write your CSS in a separate file with a .css extension.
  • Link CSS to HTML: Connect the CSS file to your HTML document using the <link> tag within the <head> section.
  • Apply Styles: Use CSS rules to style HTML elements according to your design needs.
