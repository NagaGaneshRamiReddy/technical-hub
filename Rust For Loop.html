Rust For Loop

Understanding For Loops in Rust

The for loop in Rust is a control flow construct that allows you to iterate over a range, array, or collection. It is particularly useful for processing items in a collection or executing a block of code a specific number of times.

Basic Syntax

The syntax for a for loop is as follows:

for variable in collection {
    // Code to execute
}

Iterating Over a Range

One common use of the for loop is to iterate over a range of numbers. Here’s an example:

for number in 1..6 {
    println!("Number: {}", number);
}

This loop will print numbers from 1 to 5. The range 1..6 is inclusive of the start (1) and exclusive of the end (6).

Iterating Over Collections

You can also use a for loop to iterate over arrays or vectors. Here’s an example using an array:

let numbers = [1, 2, 3, 4, 5];

for &number in &numbers {
    println!("Number: {}", number);
}

In this example, the loop iterates over the array numbers, printing each element. The & symbol is used to create a reference to the array.

Enumerating Over Collections

The enumerate() method can be used to obtain the index along with the value when iterating:

let fruits = ["apple", "banana", "cherry"];

for (index, fruit) in fruits.iter().enumerate() {
    println!("Fruit {}: {}", index, fruit);
}

This will print each fruit along with its index in the array.

Using Range with Steps

You can also specify a step in the range by using the step_by() method:

for number in (1..11).step_by(2) {
    println!("Odd number: {}", number);
}

This example will print every odd number from 1 to 10.

Summary

The for loop in Rust provides a convenient and efficient way to iterate over ranges and collections. It enhances code readability and reduces the potential for errors compared to traditional loop constructs.
