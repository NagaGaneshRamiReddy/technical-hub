Ordered Lists

Understanding Ordered Lists in HTML

An ordered list is used for items that should be displayed in a specific sequence. This is created using the <ol> tag, and each item within the list is defined using the <li> tag.

Basic Ordered List Example

Hereâ€™s how to create a simple ordered list:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Expected Output:

  1. First item
  2. Second item
  3. Third item

Ordered List with Different Types

You can specify different list styles for ordered lists using the type attribute. Here are some options:

<ol type="1">  
    <li>Item 1</li>
    <li>Item 2</li>
    </ol>

<ol type="A">  
    <li>Item A</li>
    <li>Item B</li>
    </ol>

<ol type="i">  
    <li>Item i</li>
    <li>Item ii</li>
    </ol>

Expected Output:

  1. Item 1
  2. Item 2
  1. Item A
  2. Item B
  1. Item i
  2. Item ii

Nesting Ordered Lists

Ordered lists can be nested within one another. This is useful for creating sub-items:

<ol>
    <li>Main Item 1
        <ol>
            <li>Sub Item 1.1</li>
            <li>Sub Item 1.2</li>
        </ol>
    </li>
    <li>Main Item 2</li>
</ol>

Expected Output:

  1. Main Item 1
    1. Sub Item 1.1
    2. Sub Item 1.2
  2. Main Item 2

Styling Ordered Lists with CSS

Ordered lists can also be styled using CSS to improve their appearance:

ol {
    list-style-type: decimal;  /* Change to 'upper-alpha', 'lower-alpha', etc. */
    padding: 10px;
    margin: 0;
    color: #333; /* Change text color */
}

Expected Output: The list style will change according to the CSS applied.

Ordered lists in HTML are essential for displaying items that need to be in a specific sequence. By using different types and nesting lists, you can create structured and organized content effectively.
