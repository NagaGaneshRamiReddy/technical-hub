Stack and Positioned in Flutter

Introduction to Stack and Positioned Widgets

In Flutter, the Stack widget allows you to overlay multiple widgets on top of each other. The Positioned widget is used within a Stack to precisely control the position of its child widgets.

Stack Widget

The Stack widget is a powerful layout tool that places its children on top of each other. You can use it to create complex designs where elements need to overlap, such as floating buttons, background images, or custom layouts.

Example of a Stack Widget

Stack(
  children: [
    Container(
      width: 200,
      height: 200,
      color: Colors.blue,
    ),
    Container(
      width: 100,
      height: 100,
      color: Colors.red,
    ),
  ],
)

Expected Output:

A blue box of 200x200 pixels, with a red box of 100x100 pixels stacked on top of it, centered by default.

In this example, two containers are stacked on top of each other. The red container overlaps the blue container.

Positioned Widget

The Positioned widget allows you to precisely place a widget within a Stack. You can specify the distance from the edges (top, bottom, left, or right) to control the position of a widget inside the Stack.

Example of a Positioned Widget

Stack(
  children: [
    Container(
      width: 200,
      height: 200,
      color: Colors.blue,
    ),
    Positioned(
      top: 50,
      left: 50,
      child: Container(
        width: 100,
        height: 100,
        color: Colors.red,
      ),
    ),
  ],
)

Expected Output:

A blue box of 200x200 pixels, with a red box of 100x100 pixels placed 50 pixels from the top and 50 pixels from the left inside the blue box.

In this example, the Positioned widget is used to place the red container inside the blue container with an offset of 50 pixels from the top and left edges.

Stack and Positioned Properties

  • alignment: Aligns all children of the stack. Default is center alignment.
  • fit: Controls how the non-positioned children are sized. Options include StackFit.loose, StackFit.expand, etc.
  • Positioned properties:
    • top: The distance of the widget from the top of the stack.
    • left: The distance of the widget from the left of the stack.
    • right: The distance of the widget from the right of the stack.
    • bottom: The distance of the widget from the bottom of the stack.

Example with Multiple Positioned Widgets

Stack(
  children: [
    Container(
      width: 300,
      height: 300,
      color: Colors.green,
    ),
    Positioned(
      top: 20,
      left: 20,
      child: Container(
        width: 100,
        height: 100,
        color: Colors.yellow,
      ),
    ),
    Positioned(
      bottom: 20,
      right: 20,
      child: Container(
        width: 100,
        height: 100,
        color: Colors.orange,
      ),
    ),
  ],
)

Expected Output:

A green box of 300x300 pixels, with a yellow box placed 20 pixels from the top left and an orange box placed 20 pixels from the bottom right inside the green box.

This example demonstrates how multiple Positioned widgets can be used within a Stack to place child widgets precisely at different locations.

Summary

The Stack widget allows you to overlay multiple widgets on top of each other, creating complex layouts where elements can overlap. The Positioned widget is a key tool for controlling the exact placement of widgets within a Stack. These widgets are fundamental for creating dynamic and layered UI designs in Flutter.
