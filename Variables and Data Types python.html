Variables and Data Types

1. Variables

In Python, variables are used to store data that can be referenced and manipulated later in the program. Unlike some other programming languages, Python does not require explicit declaration of variable types. You simply assign a value to a variable name:

age = 30
name = "John"
height = 5.9
is_student = True

In this example:

  • age stores an integer.
  • name stores a string.
  • height stores a float.
  • is_student stores a boolean value.

2. Data Types

Python has several built-in data types that are used to store different kinds of data. Here are the most commonly used data types:

2.1. Integers

Integers are whole numbers, both positive and negative:

number = 10

2.2. Floats

Floats are numbers with decimal points:

price = 19.99

2.3. Strings

Strings are sequences of characters enclosed in quotes (single, double, or triple quotes):

message = "Hello, World!"
multiline_message = """This is
a multiline string."""

2.4. Booleans

Booleans represent truth values: True or False:

is_valid = True
is_expired = False

2.5. Lists

Lists are ordered collections of items, which can be of different data types. Lists are mutable:

fruits = ["apple", "banana", "cherry"]

2.6. Tuples

Tuples are similar to lists but are immutable. Once created, their values cannot be changed:

coordinates = (10, 20)

2.7. Dictionaries

Dictionaries store key-value pairs. Keys are unique and used to access corresponding values:

student = {
    "name": "Alice",
    "age": 21,
    "is_graduated": False
}

2.8. Sets

Sets are collections of unique items, without duplicate values:

unique_numbers = {1, 2, 3, 4}

3. Type Conversion

You can convert between different data types using built-in functions:

number = 10
number_as_str = str(number)  # Converts integer to string
float_number = float("3.14") # Converts string to float

4. Summary

Understanding variables and data types is fundamental in Python programming. Variables store data, and Python’s built-in data types allow you to work with various kinds of information, including integers, floats, strings, booleans, lists, tuples, dictionaries, and sets. Properly managing these types will help you write efficient and effective Python code.
