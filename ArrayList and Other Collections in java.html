Introduction to Collections in Java

Introduction to Collections

In Java, collections are frameworks that provide a way to store and manipulate groups of objects. The Collections Framework provides a set of interfaces and classes that implement various types of collections, such as lists, sets, and maps. These collections are dynamic and flexible, offering a wide range of operations and functionalities.

ArrayList

ArrayList is a part of the Java Collections Framework and provides a dynamic array that can grow as needed. It implements the List interface and allows for efficient random access and modification of elements. Here’s how you can work with an ArrayList:

Creating an ArrayList

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
    }
}

Adding Elements

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");
    }
}

Accessing and Modifying Elements

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");

        // Accessing elements
        String fruit = list.get(1); // Banana
        
        // Modifying elements
        list.set(2, "Orange"); // Replace Cherry with Orange
    }
}

Removing Elements

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");

        // Removing elements
        list.remove("Banana"); // Removes Banana
        list.remove(1); // Removes element at index 1 (Orange)
    }
}

Iterating Over an ArrayList

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");

        // Using a for loop
        for (int i = 0; i < list.size(); i++) {
            System.out.println(list.get(i));
        }

        // Using an enhanced for loop
        for (String fruit : list) {
            System.out.println(fruit);
        }
    }
}

Other Collections

Besides ArrayList, the Java Collections Framework provides other collection types, each serving different purposes:

HashSet

HashSet is a collection that implements the Set interface. It does not allow duplicate elements and does not guarantee the order of elements:

import java.util.HashSet;

public class HashSetExample {
    public static void main(String[] args) {
        HashSet set = new HashSet<>();
        set.add("Apple");
        set.add("Banana");
        set.add("Cherry");
        set.add("Apple"); // Duplicate element, will not be added
    }
}

HashMap

HashMap is a collection that implements the Map interface. It stores key-value pairs and does not allow duplicate keys. The order of elements is not guaranteed:

import java.util.HashMap;

public class HashMapExample {
    public static void main(String[] args) {
        HashMap map = new HashMap<>();
        map.put("Apple", 1);
        map.put("Banana", 2);
        map.put("Cherry", 3);
    }
}

TreeSet

TreeSet is a collection that implements the Set interface. It stores elements in a sorted order:

import java.util.TreeSet;

public class TreeSetExample {
    public static void main(String[] args) {
        TreeSet set = new TreeSet<>();
        set.add("Apple");
        set.add("Banana");
        set.add("Cherry");
    }
}

LinkedList

LinkedList implements both the List and Deque interfaces. It provides efficient insertion and deletion operations:

import java.util.LinkedList;

public class LinkedListExample {
    public static void main(String[] args) {
        LinkedList list = new LinkedList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");
    }
}

Summary

The Java Collections Framework provides a range of collection types to handle various data storage and manipulation needs. Understanding how to use ArrayList, HashSet, HashMap, TreeSet, and LinkedList effectively allows you to choose the right collection for your specific requirements and optimize your code for performance and functionality.
