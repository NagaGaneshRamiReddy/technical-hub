DELETE Statement

SQL DELETE Statement

The DELETE statement in SQL is used to remove one or more rows from a table based on a specified condition. This command can be used to delete specific records or all records from a table, depending on the condition provided.

Basic Syntax

    DELETE FROM table_name
    WHERE condition;
  • table_name - The name of the table from which to delete records.
  • condition - The condition that specifies which rows should be deleted. If omitted, all rows in the table will be deleted.

Examples of Using the DELETE Statement

1. Deleting a Single Record

To delete a specific record based on a condition:

    DELETE FROM employees
    WHERE emp_id = 101;

This query deletes the employee record with ID 101 from the "employees" table.

2. Deleting Multiple Records

To delete multiple records based on a condition:

    DELETE FROM employees
    WHERE department = 'Marketing';

This query deletes all records from the "employees" table where the department is "Marketing".

3. Deleting All Records

To delete all records from a table (use with caution):

    DELETE FROM employees;

This query deletes all rows from the "employees" table but leaves the table structure intact.

4. Deleting Records with Subqueries

You can use a subquery to specify which rows to delete based on data from other tables:

    DELETE FROM employees
    WHERE department IN (SELECT department FROM departments WHERE location = 'New York');

This query deletes employees who belong to departments located in "New York".

5. Using JOIN in DELETE (for certain SQL databases)

Some SQL databases support deleting records using a JOIN clause:

    DELETE e
    FROM employees e
    JOIN departments d ON e.department_id = d.department_id
    WHERE d.location = 'New York';

This query deletes employees associated with departments located in "New York" using a JOIN operation.

Summary

The DELETE statement is used to remove data from a SQL table. By specifying conditions, you can delete specific rows, multiple rows, or even all rows from a table. It is crucial to use this statement carefully, especially when deleting all records or using conditions that may affect many rows.
