String Functions

String Functions in SQL

String Functions are used to perform operations on string data types. These functions can manipulate, extract, and format text data, making it easier to handle and analyze textual information within SQL queries.

Common String Functions

Function Description Example
CONCAT() Concatenates two or more strings into a single string. SELECT CONCAT(FirstName, ' ', LastName) AS FullName FROM Employees;
LENGTH() Returns the number of characters in a string. SELECT LENGTH(FirstName) FROM Employees;
SUBSTRING() Extracts a portion of a string based on specified starting position and length. SELECT SUBSTRING(FirstName, 1, 3) FROM Employees;
UPPER() Converts all characters in a string to uppercase. SELECT UPPER(FirstName) FROM Employees;
LOWER() Converts all characters in a string to lowercase. SELECT LOWER(FirstName) FROM Employees;
TRIM() Removes leading and trailing spaces from a string. SELECT TRIM(FirstName) FROM Employees;
REPLACE() Replaces occurrences of a specified substring with another substring. SELECT REPLACE(Address, 'Street', 'Avenue') FROM Customers;
CHARINDEX() Returns the starting position of a specified substring within a string. SELECT CHARINDEX('New', Address) FROM Customers;
LEFT() Returns a specified number of characters from the left side of a string. SELECT LEFT(FirstName, 3) FROM Employees;
RIGHT() Returns a specified number of characters from the right side of a string. SELECT RIGHT(LastName, 4) FROM Employees;

Examples of String Functions in Action

Concatenating Names

To create a full name by concatenating first and last names:

SELECT CONCAT(FirstName, ' ', LastName) AS FullName
FROM Employees;

Finding String Length

To find the length of the address of customers:

SELECT LENGTH(Address) AS AddressLength
FROM Customers;

Extracting Substrings

To extract the first three characters of the first name:

SELECT SUBSTRING(FirstName, 1, 3) AS ShortName
FROM Employees;

Changing Case

To convert all first names to uppercase:

SELECT UPPER(FirstName) AS UpperFirstName
FROM Employees;

Removing Spaces

To remove leading and trailing spaces from a string:

SELECT TRIM(Address) AS CleanAddress
FROM Customers;

Summary

String functions in SQL are powerful tools for manipulating and analyzing text data. Functions such as CONCAT(), LENGTH(), SUBSTRING(), and REPLACE() enable you to perform a variety of operations on strings, from concatenation and length calculation to case conversion and substring extraction.
