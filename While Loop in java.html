while loop in Java

Introduction

The while loop in Java is a control flow statement that allows you to execute a block of code repeatedly as long as a specified condition remains true. It is particularly useful when the number of iterations is not known in advance and the loop needs to continue based on dynamic conditions. This guide will explain the syntax and usage of the while loop with examples.

1. Basic Syntax

The basic syntax of a while loop is as follows:

while (condition) {
    // Code to be executed
}
  • Condition: Evaluated before each iteration. If the condition evaluates to true, the loop executes the code block; otherwise, the loop terminates.

2. Example: Counting from 0 to 4

Hereâ€™s a simple example that uses a while loop to count from 0 to 4:

int i = 0;

while (i < 5) {
    System.out.println("Count: " + i);
    i++;
}

In this example, the loop starts with i initialized to 0. The condition i < 5 is checked before each iteration. If true, the loop prints the value of i and increments i by 1. The loop terminates when i reaches 5.

3. Example: User Input Loop

The `while` loop can also be used to repeatedly prompt the user for input until a valid input is received:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int number = 0;

while (number < 1 || number > 10) {
    System.out.print("Enter a number between 1 and 10: ");
    number = scanner.nextInt();
}
System.out.println("Valid number entered: " + number);

In this example, the loop continues to prompt the user for a number until a value between 1 and 10 is entered. Once a valid number is provided, the loop exits.

4. Example: Infinite Loop

A `while` loop can run indefinitely if the condition always evaluates to true. This is known as an infinite loop:

while (true) {
    System.out.println("This will run forever.");
}

In this example, the loop will run indefinitely because the condition true is always true. Be cautious with infinite loops as they can cause programs to become unresponsive.

5. Breaking Out of a While Loop

You can use the break statement to exit a `while` loop prematurely based on a specific condition:

int i = 0;

while (true) {
    if (i >= 5) {
        break; // Exit the loop when i is 5 or greater
    }
    System.out.println("Count: " + i);
    i++;
}

In this example, the loop will terminate when i reaches 5, even though the condition of the while loop is always true.

6. Continue Statement in While Loop

You can use the continue statement to skip the remaining code in the current iteration and proceed with the next iteration of the loop:

int i = 0;

while (i < 10) {
    if (i % 2 == 0) {
        i++;
        continue; // Skip printing for even numbers
    }
    System.out.println("Odd Count: " + i);
    i++;
}

In this example, the continue statement causes the loop to skip printing even numbers and only print odd numbers.

Summary

The while loop is a versatile tool for handling repetitive tasks when the number of iterations is not known beforehand. It evaluates the loop condition before each iteration, making it suitable for scenarios where the loop should continue based on dynamic conditions. Mastery of the while loop, including handling infinite loops and using break and continue statements, will enhance your programming skills and allow you to solve a variety of problems effectively.
