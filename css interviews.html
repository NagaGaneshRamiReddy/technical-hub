CSS Interview Questions

What is CSS? ›
CSS stands for Cascading Style Sheets. It is used to style and layout web pages, including the design, layout, and variations in display for different devices and screen sizes.
What are the different ways to apply CSS to a web page? ›
CSS can be applied to HTML in three different ways:
  1. Inline CSS: Using the style attribute directly inside HTML elements.
  2. Internal CSS: Using a <style> element within the <head> of an HTML document.
  3. External CSS: Linking to an external CSS file using the <link> element.
Example of inline CSS:
<p style="color: red;">This is red text.</p>
What is the CSS box model? ›
The CSS box model is a box that wraps around every HTML element. It consists of:
  • Content: The actual content of the box, such as text or images.
  • Padding: Clears an area around the content inside the box.
  • Border: A border around the padding (if any is defined).
  • Margin: Clears an area outside the border, creating space between elements.
Example:
.element {
    width: 200px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}
What is the difference between classes and IDs in CSS? ›
Classes are reusable and can be applied to multiple elements, whereas IDs must be unique within a page. In CSS, classes are prefixed with a dot . and IDs are prefixed with a hash #.
Example:
.example-class { color: blue; }
#example-id { color: red; }
What is the difference between padding and margin? ›
Padding is the space inside the element, between the content and the border. Margin is the space outside the element, creating distance between the element and its neighbors.
What is Flexbox in CSS? ›
Flexbox is a layout model in CSS that makes it easy to align elements vertically and horizontally, distribute space, and handle dynamic resizing of elements within a container. It uses properties like justify-content, align-items, and flex-direction.
What is CSS Grid? ›
CSS Grid is a powerful layout system that allows you to design complex web layouts easily by using rows and columns. It is more flexible than Flexbox when it comes to creating two-dimensional layouts.
What are pseudo-classes in CSS? ›
A pseudo-class is used to define a special state of an element. For example, :hover applies a style when the user hovers over an element.
Example:
a:hover { color: red; }
What are pseudo-elements in CSS? ›
A pseudo-element is used to style specific parts of an element. For example, ::before can insert content before an element's content.
Example:
p::before { content: "Prefix"; }
What is the difference between block, inline, and inline-block elements? ›
  • Block elements: Take up the full width and start on a new line (e.g., <div>, <p>).
  • Inline elements: Take up only the space their content needs and do not start on a new line (e.g., <span>, <a>).
  • Inline-block elements: Behave like inline elements but respect width and height properties.
How can you center an element in CSS? ›
There are different ways to center elements horizontally and vertically. Here is a common method using Flexbox:
Example:
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}
What is the difference between relative, absolute, fixed, and sticky positioning? ›
  • Relative: Positioned relative to its normal position in the document flow.
  • Absolute: Positioned relative to its nearest positioned ancestor (if any), otherwise the document body.
  • Fixed: Positioned relative to the browser window, staying in the same place even when the page is scrolled.
  • Sticky: Toggles between relative and fixed positioning depending on the user's scroll position.
How does the z-index property work in CSS? ›
The z-index property determines the stack order of elements. Elements with a higher z-index appear in front of elements with a lower z-index. The property only works on positioned elements (relative, absolute, fixed).
What is the difference between CSS transition and animation? ›
  • Transitions: Changes an element’s property values over a specified duration. It requires a trigger such as hover or focus.
  • Animations: Allow more complex changes and can be looped. You can define multiple keyframes to control the state of an element at different stages.
How do media queries work in CSS? ›
Media queries are used to apply different styles to different devices or screen sizes. You can specify different CSS rules for different breakpoints using the @media rule.
Example:
@media (max-width: 768px) { body { font-size: 14px; } }
What are CSS preprocessors like Sass or Less? ›
CSS preprocessors like Sass or Less add features that don’t exist in regular CSS, such as variables, nested rules, and mixins. The preprocessor syntax is compiled into standard CSS before being used in a webpage.
What is the difference between rem and em units? ›
rem: Refers to the font-size of the root element (usually html).
em: Refers to the font-size of the current element's parent. It is relative to the element's inherited size.
How do you create a responsive design? ›
Responsive design is created by using flexible grids, layouts, images, and CSS media queries. It ensures that the webpage adjusts to different screen sizes and devices.
What is the purpose of the :nth-child() selector? ›
The :nth-child() selector allows you to style elements based on their position in a parent element, such as styling every odd or even child.
Example:
li:nth-child(odd) { background-color: lightgray; }
What is the purpose of the display property in CSS? ›
The display property controls the layout of an element. Common values include block, inline, flex, and grid.
What is the purpose of the calc() function in CSS? ›
The calc() function allows you to perform calculations to determine CSS property values. For example, you can dynamically calculate width or margins.
Example:
width: calc(100% - 50px);
How do you make a CSS rule apply only to a specific screen size? ›
You can use media queries to apply CSS rules based on the screen size or device type. For example:
@media (min-width: 768px) { .container { width: 70%; } }
What is the ::after pseudo-element? ›
The ::after pseudo-element inserts content after an element's content. It's commonly used for adding decorative elements or icons.
Example:
p::after { content: "*"; color: red; }
What are vendor prefixes and why are they used? ›
Vendor prefixes (like -webkit-, -moz-, -o-, -ms-) are used to ensure CSS works in specific browsers that don't fully support new CSS features yet.
What is the difference between visibility: hidden and display: none? ›
  • visibility: hidden: The element remains in the layout but is invisible.
  • display: none: The element is removed from the layout entirely, as if it doesn't exist.
How do you optimize CSS for better performance? ›
Some ways to optimize CSS include:
  • Minifying CSS files.
  • Using shorthand properties where possible.
  • Combining multiple CSS files into one to reduce HTTP requests.
  • Using efficient selectors and reducing specificity.
What is the purpose of the content property in CSS? ›
The content property is used with pseudo-elements like ::before and ::after to insert generated content into an element. It can be text, images, or symbols.
What is specificity in CSS? ›
Specificity is a rule in CSS that determines which styles are applied to an element when there are conflicting styles. It's calculated based on the types of selectors used (inline, ID, class, or element).
How do you target elements with CSS that don’t have a specific class or ID? ›
You can target elements without classes or IDs by using element selectors (e.g., div, p) or pseudo-classes (e.g., :nth-child(), :first-of-type).
What is the difference between hover and focus pseudo-classes? ›
The :hover pseudo-class applies when the user hovers over an element with their mouse. The :focus pseudo-class applies when an element gains focus, such as when a user clicks on an input field.
