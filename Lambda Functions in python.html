Lambda Function in Python

What is a Lambda Function?

A lambda function in Python is an anonymous function expressed as a single statement. Lambda functions are useful for creating small, one-off functions without the need to define a named function using the `def` keyword. They are often used in situations where a small function is required temporarily and will not be reused elsewhere in the code.

Syntax of Lambda Functions

The syntax for a lambda function is:

lambda arguments: expression

Here, lambda is the keyword, arguments are the inputs to the function, and expression is a single expression that the function evaluates and returns. Lambda functions can have multiple arguments but can only have a single expression.

Examples of Lambda Functions

Basic Example

Hereâ€™s a simple example of a lambda function that adds two numbers:

# Lambda function to add two numbers
add = lambda x, y: x + y

# Using the lambda function
result = add(5, 3)
print(result)  # Output: 8

Lambda Function with No Arguments

A lambda function can also be defined with no arguments:

# Lambda function with no arguments
greeting = lambda: "Hello, World!"

# Using the lambda function
print(greeting())  # Output: Hello, World!

Lambda Functions with Filter, Map, and Reduce

Lambda functions are often used with functions like filter, map, and reduce to perform operations on collections of data:

# Using lambda with map to square numbers
numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Output: [1, 4, 9, 16]

# Using lambda with filter to get even numbers
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # Output: [2, 4]

Lambda Function with Reduce

The reduce function from the functools module can be used with lambda functions to perform cumulative operations:

# Using lambda with reduce to calculate the product of numbers
from functools import reduce

numbers = [1, 2, 3, 4]
product = reduce(lambda x, y: x * y, numbers)
print(product)  # Output: 24

Key Points

  • Anonymous Functions: Lambda functions are anonymous and do not require a name.
  • Single Expression: Lambda functions can only have a single expression, making them less flexible than regular functions.
  • Use Cases: Ideal for small, short-term functions where defining a full function might be overkill.
  • Integration with Functions: Frequently used with functions like map, filter, and reduce for concise data manipulation.

Summary

Lambda functions provide a concise way to create small, anonymous functions in Python. They are especially useful for short-term operations and functional programming paradigms. Understanding how to use lambda functions effectively can help you write more efficient and readable code.
