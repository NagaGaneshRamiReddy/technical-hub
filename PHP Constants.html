PHP Constants

Overview

Constants are identifiers (names) for simple values that remain the same throughout the execution of a script. Unlike variables, constants cannot be changed or undefined once they are set.

Defining Constants

You can define a constant in PHP using the define() function:

    define("CONSTANT_NAME", "Value");

Example:

    define("PI", 3.14);

Using Constants

Once defined, constants can be used throughout your code without the dollar sign:

    echo PI; // Outputs: 3.14

Constant Naming Rules

When naming constants, follow these rules:

  • Names should be uppercase by convention (e.g., MY_CONSTANT).
  • Names can contain letters, numbers, and underscores.
  • Constants cannot start with a number.

Magic Constants

PHP has predefined constants known as magic constants. These include:

  • __LINE__: The current line number in the script.
  • __FILE__: The full path and filename of the file.
  • __DIR__: The directory of the file.
  • __FUNCTION__: The name of the function.
  • __CLASS__: The name of the class.

Example usage:

    echo "This is line " . __LINE__; // Outputs the current line number

Case Sensitivity

Constant names are case-insensitive. However, it is recommended to use uppercase for better readability:

    define("SAMPLE_CONSTANT", "Hello");
    echo sample_constant; // Outputs: Hello

Undefined Constants

Accessing an undefined constant will produce a notice, and the constant will be evaluated as a string:

    echo UNDEFINED_CONSTANT; // Outputs: UNDEFINED_CONSTANT (as a string)

Summary

Constants are useful for defining values that should remain unchanged throughout your script. Understanding how to use and define constants effectively can enhance the clarity and maintainability of your code.
