Gradients

Introduction to Gradients

Gradients are a way to create smooth transitions between two or more colors. CSS gradients can be used as background images, creating visually appealing effects without the need for image files. There are two main types of gradients in CSS: linear gradients and radial gradients.

1. Linear Gradients

Linear gradients transition colors along a straight line. You can control the direction and color stops of the gradient:

linear-gradient(to right, #ff7e5f, #feb47b)
linear-gradient(to bottom, #00c6ff, #0072ff)
linear-gradient(45deg, #ff9a9e, #fad0c4)

2. Radial Gradients

Radial gradients transition colors from the center outward. You can control the shape, size, and color stops:

radial-gradient(circle, #ff7e5f, #feb47b)
radial-gradient(ellipse at center, #00c6ff, #0072ff)
radial-gradient(circle, #ff9a9e, #fad0c4)

3. Using Multiple Colors

You can use more than two colors in a gradient to create complex and rich visual effects:

linear-gradient(to right, #ff7e5f, #feb47b, #f8ff00)
radial-gradient(circle, #ff7e5f, #feb47b, #f8ff00)

Code Example


/* Linear Gradient */
background: linear-gradient(to right, #ff7e5f, #feb47b);

/* Radial Gradient */
background: radial-gradient(circle, #ff7e5f, #feb47b);

/* Multiple Colors */
background: linear-gradient(to right, #ff7e5f, #feb47b, #f8ff00);

Summary

CSS gradients are a powerful tool for creating visually appealing designs. By using linear and radial gradients, along with multiple color stops, you can enhance the aesthetics of your web pages and achieve a range of effects without relying on external images.
