HTML Attributes

Overview

HTML attributes provide additional information about HTML elements. They are always specified in the opening tag and come in name/value pairs like name="value".

Common HTML Attributes

1. id Attribute

The id attribute specifies a unique identifier for an HTML element:

<div id="uniqueId">This is a unique div</div>

2. class Attribute

The class attribute specifies one or more class names for an HTML element, allowing for CSS styling:

<div class="container">This is a styled div</div>

3. style Attribute

The style attribute allows you to add inline CSS styles to an HTML element:

<p style="color: blue; font-size: 20px;">This is a styled paragraph.</p>

4. href Attribute

The href attribute specifies the URL of a link:

<a href="https://www.example.com">This is a link</a>

5. src Attribute

The src attribute specifies the path to an image or other media:

<img src="image.jpg" alt="Description">

6. alt Attribute

The alt attribute provides alternative text for images:

<img src="image.jpg" alt="A beautiful scenery">

7. title Attribute

The title attribute provides additional information about an element, typically displayed as a tooltip:

<a href="#" title="More information">Hover over me</a>

Global Attributes

Global attributes can be applied to any HTML element:

  • class: The class attribute is used to assign one or more class names to an HTML element. These classes are typically utilized for CSS styling and can also be targeted by JavaScript for manipulation or event handling.
    Example: 
    <div class="container highlight">This div is styled with two classes.</div>
    Use Case: Helps to apply the same styles to multiple elements without repeating CSS rules.
  • id: The id attribute provides a unique identifier for an HTML element. An id should be unique within a page, ensuring that no two elements share the same identifier.
    Example: 
    <h1 id="main-title">Welcome to My Website</h1>
    Use Case: Useful for linking to specific sections of a page or for styling and JavaScript targeting.
  • style: The style attribute allows you to apply inline CSS styles directly to an HTML element. While convenient for quick styling, it is generally recommended to use external CSS for maintainability.
    Example: 
    <p style="color: red; font-size: 20px;">This text is red and 20 pixels in size.</p>
    Use Case: Useful for making quick styling changes or overriding external CSS temporarily.
  • title: The title attribute offers additional information about an element. A tooltip containing the title text appears when a user hovers over the element.
    Example: 
    <a href="#" title="Click here for more information">Learn more</a>
    Use Case: Enhances user experience by providing extra context without cluttering the interface.
  • data-* (custom data attributes): Custom data attributes allow you to store extra information on HTML elements without using additional properties. They begin with the prefix data- and can be accessed in JavaScript.
    Example: 
    <div data-user-id="12345" data-role="admin">User Info</div>
    Use Case: Useful for storing additional information that can be dynamically accessed through JavaScript, especially in single-page applications or interactive web pages.

Example of Custom Data Attribute

<div data-user-id="12345">User Info</div>
