INNER JOIN

SQL INNER JOIN

The INNER JOIN operation in SQL is used to combine rows from two or more tables based on a related column that exists in both tables. It returns only the rows that have matching values in both tables, filtering out non-matching rows.

Basic Syntax

    SELECT columns
    FROM table1
    INNER JOIN table2
    ON table1.common_column = table2.common_column;
  • columns - The columns you want to retrieve from both tables.
  • table1 - The first table in the join operation.
  • table2 - The second table in the join operation.
  • common_column - The column used to match rows between the two tables.

Examples of INNER JOIN

1. Basic INNER JOIN

Retrieve a list of employees and their corresponding department names:

    SELECT employees.emp_name, departments.dept_name
    FROM employees
    INNER JOIN departments
    ON employees.dept_id = departments.dept_id;

This query joins the employees table with the departments table using the dept_id column. It returns the names of employees and their respective department names where there is a match.

2. INNER JOIN with Multiple Tables

Retrieve student names, their courses, and the instructor names:

    SELECT students.student_name, courses.course_name, instructors.instructor_name
    FROM students
    INNER JOIN enrollments ON students.student_id = enrollments.student_id
    INNER JOIN courses ON enrollments.course_id = courses.course_id
    INNER JOIN instructors ON courses.instructor_id = instructors.instructor_id;

This query joins the students, enrollments, courses, and instructors tables. It provides a list of student names, their enrolled courses, and the names of the instructors for those courses.

3. INNER JOIN with Aliases

Use table aliases to simplify complex queries:

    SELECT e.emp_name, d.dept_name
    FROM employees e
    INNER JOIN departments d ON e.dept_id = d.dept_id;

This query uses aliases e for the employees table and d for the departments table. It achieves the same result as the previous example but with a more concise syntax.

Advanced Usage of INNER JOIN

1. INNER JOIN with Filtering

Combine INNER JOIN with additional filters:

    SELECT employees.emp_name, departments.dept_name
    FROM employees
    INNER JOIN departments
    ON employees.dept_id = departments.dept_id
    WHERE departments.dept_name = 'Sales';

This query retrieves employees who are in the "Sales" department by adding a WHERE clause to filter the results.

2. INNER JOIN with Sorting

Combine INNER JOIN with sorting results:

    SELECT employees.emp_name, departments.dept_name
    FROM employees
    INNER JOIN departments
    ON employees.dept_id = departments.dept_id
    ORDER BY employees.emp_name;

This query retrieves employees and their department names and sorts the results by employee names in ascending order.

Summary

The INNER JOIN is a powerful SQL operation that combines rows from two or more tables based on a related column. It returns only the rows that have matching values in both tables, making it essential for querying related data across multiple tables.
