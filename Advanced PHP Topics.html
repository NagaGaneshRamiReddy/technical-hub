Advanced PHP Topics

Overview

This section delves into advanced topics in PHP, including regular expressions, handling JSON data, and working with RESTful APIs. These concepts are crucial for building sophisticated applications.

PHP Regular Expressions

Regular expressions (regex) allow you to search and manipulate strings based on patterns. In PHP, you can use functions like preg_match(), preg_replace(), and preg_split().

Example Usage

    <?php
    $pattern = "/\d+/"; // Pattern to match one or more digits
    $string = "There are 123 apples.";

    if (preg_match($pattern, $string, $matches)) {
        echo "Found a number: " . $matches[0];
    }
    ?>

Handling JSON Data

JSON (JavaScript Object Notation) is a lightweight data interchange format. PHP provides json_encode() and json_decode() functions to work with JSON data.

Example Usage

    <?php
    // PHP array
    $data = ["name" => "John", "age" => 30];

    // Convert array to JSON
    $json = json_encode($data);
    echo $json; // Outputs: {"name":"John","age":30}

    // Convert JSON back to PHP array
    $decoded = json_decode($json, true);
    echo $decoded['name']; // Outputs: John
    ?>

PHP RESTful APIs

Building RESTful APIs with PHP allows your application to communicate with other services over HTTP. You can create APIs that return data in JSON format.

Example Structure

    <?php
    header("Content-Type: application/json");
    
    // Sample data
    $data = [
        ["id" => 1, "name" => "John"],
        ["id" => 2, "name" => "Jane"]
    ];
    
    echo json_encode($data);
    ?>

This API can be accessed using a client (like Postman) and will return the JSON array.
