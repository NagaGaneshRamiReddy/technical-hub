Pointer and Structures in C

Introduction

In C programming, structures are used to group variables of different types into a single unit. Pointers can be used with structures to dynamically allocate memory and access structure members efficiently.

1. Basics of Structures

Structures allow you to create complex data types that group together variables under a single name.

Defining a Structure

Structures are defined using the struct keyword, followed by the structure name and the members it contains.


#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    struct Person person1;
    person1.age = 30;
    snprintf(person1.name, sizeof(person1.name), "John Doe");

    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);

    return 0;
}

Accessing Structure Members

Structure members are accessed using the dot operator (.).

2. Pointers to Structures

Pointers can be used to point to structures, allowing you to dynamically manage memory and access structure members.

Declaring and Using Pointers to Structures

You can declare pointers to structures and use them to access structure members via the arrow operator (->).


#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    struct Person person1;
    struct Person *ptr = &person1;

    ptr->age = 30;
    snprintf(ptr->name, sizeof(ptr->name), "Jane Smith");

    printf("Name: %s\n", ptr->name);
    printf("Age: %d\n", ptr->age);

    return 0;
}

Dynamic Memory Allocation for Structures

Using pointers with dynamic memory allocation allows you to create structures at runtime.


#include <stdio.h>
#include <stdlib.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    struct Person *ptr = (struct Person *)malloc(sizeof(struct Person));
    if (ptr == NULL) {
        printf("Memory allocation failed\n");
        return 1;
    }

    ptr->age = 40;
    snprintf(ptr->name, sizeof(ptr->name), "Alice Johnson");

    printf("Name: %s\n", ptr->name);
    printf("Age: %d\n", ptr->age);

    free(ptr); // Free allocated memory
    return 0;
}

3. Pointers to Structures in Functions

Pointers to structures can be passed to functions, allowing the function to modify the structure directly.

Passing Structure Pointers to Functions

By passing a structure pointer to a function, you can modify the structure’s contents from within the function.


#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

void updatePerson(struct Person *p) {
    p->age += 1;
    snprintf(p->name, sizeof(p->name), "Updated Name");
}

int main() {
    struct Person person1;
    person1.age = 25;
    snprintf(person1.name, sizeof(person1.name), "Robert Brown");

    updatePerson(&person1);

    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);

    return 0;
}

4. Structures and Arrays

You can create arrays of structures and use pointers to manage and access the elements of these arrays.

Array of Structures

Arrays of structures can be used to store multiple records of the same type.


#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    struct Person people[3] = {
        {"Alice", 30},
        {"Bob", 25},
        {"Charlie", 35}
    };

    for (int i = 0; i < 3; i++) {
        printf("Name: %s, Age: %d\n", people[i].name, people[i].age);
    }

    return 0;
}
