Font Properties in CSS

What are Font Properties in CSS?

Font properties in CSS are used to control the appearance of text on a web page. These properties allow you to set the font type, size, weight, style, and more, giving you precise control over text styling.

Key Font Properties

1. Font Family

The font-family property specifies the typeface for text. You can use specific font names or generic font families like serif and sans-serif.


/* Set font family */
p {
    font-family: 'Georgia', serif;
}

2. Font Size

The font-size property controls the size of the text. You can set it in various units like pixels (px), ems (em), or percentages (%).


/* Set font size */
h1 {
    font-size: 36px;
}

3. Font Weight

The font-weight property adjusts the thickness of the text. Values include normal, bold, and numerical values from 100 to 900.


/* Set font weight */
strong {
    font-weight: 700;
}

4. Font Style

The font-style property allows you to set the style of the text to normal, italic, or oblique.


/* Set font style */
em {
    font-style: italic;
}

5. Line Height

The line-height property sets the amount of space between lines of text. It can be specified in units like pixels or as a unitless number.


/* Set line height */
p {
    line-height: 2;
}

6. Text Transform

The text-transform property changes the capitalization of text. You can set it to uppercase, lowercase, or capitalize.


/* Transform text to uppercase */
.uppercase {
    text-transform: uppercase;
}

7. Letter Spacing

The letter-spacing property adjusts the space between characters in a text. You can set it in units like pixels or ems.


/* Set letter spacing */
h2 {
    letter-spacing: 2px;
}

8. Font Variant

The font-variant property provides access to various typographic features such as small-caps. It is often used for stylistic effects.


/* Set font variant */
.small-caps {
    font-variant: small-caps;
}

Summary

Font properties in CSS allow you to customize the appearance of text on a web page. By using properties like font-family, font-size, font-weight, and others, you can create visually engaging and readable text that fits the design of your website.
