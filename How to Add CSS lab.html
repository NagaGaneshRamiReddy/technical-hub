Web Development Lab: Task - How to Add CSS

Task: Add CSS in Three Different Ways (30 Marks)

In this task, you need to demonstrate the three ways to add CSS to a webpage:

  • Using **inline styles**
  • Using **internal styles** (within a <style> block in the <head>)
  • Using **external styles** (linking to an external stylesheet)

Instructions

Create an HTML document that demonstrates all three ways to add CSS. Your document should include the following:

  • An inline style applied to a single element.
  • Internal styles that style multiple elements on the page.
  • An external stylesheet (create a separate file, and ensure the page links to it).

Example Output

Your page should look like this:

This is an inline styled heading

This paragraph is styled using internal CSS.
