Rust Strings

Understanding Strings in Rust

Rust provides two primary string types: String and &str (string slices). The String type is a growable, heap-allocated data structure, while &str is a fixed-size reference to a string slice.

Creating a String

You can create a string using the String::from function or by using the to_string() method:

fn main() {
            let string1 = String::from("Hello, Rust!");
            let string2 = "Hello, World!".to_string();
        
            println!("{}", string1); // Outputs: Hello, Rust!
            println!("{}", string2); // Outputs: Hello, World!
        }

Modifying a String

Strings are mutable, allowing you to modify their contents:

fn main() {
            let mut my_string = String::from("Hello");
            my_string.push_str(", Rust!"); // Appending a string slice
        
            println!("{}", my_string); // Outputs: Hello, Rust!
        }

String Concatenation

You can concatenate strings using the + operator or the format! macro:

fn main() {
            let s1 = String::from("Hello");
            let s2 = String::from(", Rust!");
            let s3 = s1 + &s2; // s1 is moved, so it can no longer be used
        
            println!("{}", s3); // Outputs: Hello, Rust!
            // println!("{}", s1); // This would cause a compile error
        
            let s4 = format!("{}{}", s2, " is awesome!"); // s2 remains usable
            println!("{}", s4); // Outputs: , Rust! is awesome!
        }

Accessing Characters

You can access individual characters in a string using indexing, but be cautious, as it may panic if the index is out of bounds:

fn main() {
            let my_string = String::from("Rust");
        
            let first_char = my_string.chars().nth(0).unwrap(); // Accessing first character
            println!("First character: {}", first_char); // Outputs: First character: R
        }

Iterating Over a String

You can iterate over the characters in a string using a loop:

fn main() {
            let my_string = String::from("Hello");
        
            for c in my_string.chars() {
                println!("{}", c); // Outputs each character on a new line
            }
        }

Summary

Strings in Rust are versatile and powerful, providing mutable and immutable options for handling text. Understanding the difference between String and &str is crucial for effective string manipulation.
