Control statements in Java

Introduction

Control statements in Java are used to control the flow of execution in a program. They allow you to make decisions, execute code conditionally, and repeat code. This guide covers the primary control statements in Java, including if statements, switch statements, and various types of loops.

1. If Statement

The if statement allows you to execute code blocks conditionally based on whether a boolean expression evaluates to true or false.

int age = 20;

if (age >= 18) {
    System.out.println("You are an adult.");
} else {
    System.out.println("You are a minor.");
}

Output

This is the output from the if-else condition:

You are an adult.

Explanation of the Output

age = 20 — The variable age is assigned the value 20.

if (age >= 18) — The condition checks whether the value of age is greater than or equal to 18. Since age is 20, which is greater than 18, the condition evaluates to true.

System.out.println("You are an adult.") — Since the condition is true, the program prints "You are an adult.".

If the age had been less than 18, the program would have executed the else block, printing "You are a minor." instead.

2. Switch Statement

The switch statement allows you to select one of many code blocks to execute based on the value of an expression. It is often used as an alternative to multiple if-else statements when dealing with multiple conditions.

int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    case 4:
        System.out.println("Thursday");
        break;
    case 5:
        System.out.println("Friday");
        break;
    case 6:
        System.out.println("Saturday");
        break;
    case 7:
        System.out.println("Sunday");
        break;
    default:
        System.out.println("Invalid day");
}

Output

This is the output from the switch statement:

Wednesday

Explanation of the Output

day = 3 — The variable day is assigned the value 3, representing Wednesday.

switch (day) — The switch statement evaluates the value of day and executes the case block that matches the value.

case 3: — Since day is 3, the program matches the case case 3: and executes the corresponding code, printing "Wednesday".

break; — The break statement exits the switch block after printing the result. This prevents the program from executing any subsequent cases.

If day had been a different value (like 1, 2, 4, etc.), the corresponding day of the week would have been printed. If the value did not match any case, the default block would have executed and printed "Invalid day".

3. For Loop

The for loop is used to execute a block of code a specific number of times. It is often used when the number of iterations is known beforehand.

for (int i = 0; i < 5; i++) {
    System.out.println("Count: " + i);
}

Output

This is the output from the for loop:

Count: 0
Count: 1
Count: 2
Count: 3
Count: 4

Explanation of the Output

for (int i = 0; i < 5; i++) — This is a for loop that initializes i to 0 and runs the loop as long as i is less than 5. After each iteration, the value of i is incremented by 1.

System.out.println("Count: " + i); — In each iteration, the current value of i is printed to the console, showing the current count.

The loop executes 5 times: when i is 0, 1, 2, 3, and 4, and prints the value of i in each iteration.

When i reaches 5, the loop condition i < 5 becomes false, and the loop terminates.

4. While Loop

The while loop repeatedly executes a block of code as long as a specified condition is true. It is used when the number of iterations is not known and depends on a condition.

int i = 0;
while (i < 5) {
    System.out.println("Count: " + i);
    i++;
}

Output

This is the output from the while loop:

Count: 0
Count: 1
Count: 2
Count: 3
Count: 4

Explanation of the Output

int i = 0; — The variable i is initialized to 0, which serves as the starting point for the loop.

while (i < 5) — This is the condition for the while loop. The loop will continue to execute as long as i is less than 5. When i reaches 5, the loop will terminate.

System.out.println("Count: " + i); — In each iteration, the current value of i is printed to the console, showing the count of the loop at that moment.

i++; — The value of i is incremented by 1 after each iteration to eventually make the condition i < 5 false and stop the loop.

The loop starts with i = 0 and increments by 1 until it reaches 5, at which point the loop condition is no longer satisfied, and the loop stops.

5. Do-While Loop

The do-while loop is similar to the while loop but guarantees that the block of code will be executed at least once, regardless of the condition.

int i = 0;
do {
    System.out.println("Count: " + i);
    i++;
} while (i < 5);

Output

This is the output from the do-while loop:

Count: 0
Count: 1
Count: 2
Count: 3
Count: 4

Explanation of the Output

int i = 0; — The variable i is initialized to 0.

do { ... } while (i < 5); — The do-while loop executes the block inside the do statement at least once before checking the condition in the while statement.

System.out.println("Count: " + i); — In each iteration, the current value of i is printed to the console with the message `"Count: "`.

i++; — After printing the value of i, the value of i is incremented by 1.

The loop will execute at least once, and it will continue executing as long as the condition i < 5 is true.

Summary

Control statements are fundamental to programming as they allow you to direct the flow of execution and handle various scenarios in your code. Mastering these control structures will enable you to write more flexible and effective Java programs.
