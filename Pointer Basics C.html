Basics of Pointer in C

Introduction to Pointers

In C programming, a pointer is a variable that stores the memory address of another variable. Pointers are crucial for efficient memory management and manipulation of data structures. They provide a way to directly access and modify memory, making them a powerful feature of the C language.

1. Declaring Pointers

To declare a pointer, you specify the type of data it will point to, followed by an asterisk (*) before the pointer's name.


int *ptr; // Pointer to an integer
char *chPtr; // Pointer to a character

In these examples, ptr is a pointer to an int, and chPtr is a pointer to a char.

2. Initializing Pointers

To initialize a pointer, assign it the address of a variable using the address-of operator (&).


int num = 10;
int *ptr = # // ptr holds the address of num

Here, ptr is initialized to point to the address of the variable num.

3. Dereferencing Pointers

Dereferencing a pointer means accessing the value stored at the memory address the pointer is pointing to. This is done using the dereference operator (*).


int num = 10;
int *ptr = #

printf("Value of num: %d\n", num); // Output: 10
printf("Value pointed to by ptr: %d\n", *ptr); // Output: 10

In this example, *ptr gives us the value stored at the address held by ptr, which is 10.

4. Pointer to Pointer

You can also have pointers that point to other pointers. This is known as a pointer to a pointer.


int num = 10;
int *ptr = #
int **ptrToPtr = &ptr; // Pointer to pointer

printf("Value of num: %d\n", num); // Output: 10
printf("Value pointed to by ptr: %d\n", *ptr); // Output: 10
printf("Value pointed to by ptrToPtr: %d\n", **ptrToPtr); // Output: 10

Here, ptrToPtr is a pointer that holds the address of the pointer ptr. Dereferencing twice gives us the value of num.

5. Null Pointers

A null pointer is a pointer that does not point to any valid memory location. It is used to indicate that the pointer is not currently pointing to a valid object.


int *ptr = NULL; // Null pointer

if (ptr == NULL) {
    printf("Pointer is NULL\n");
}

Using null pointers helps in avoiding accidental access to invalid memory locations.

6. Pointer Size and Arithmetic

The size of a pointer depends on the system architecture (e.g., 32-bit or 64-bit). Pointer arithmetic involves operations like incrementing or decrementing pointers to navigate through memory locations, especially useful in array manipulation.


int arr[] = {10, 20, 30};
int *ptr = arr; // Pointer to the first element of the array

printf("First element: %d\n", *ptr); // Output: 10
ptr++; // Move to the next element
printf("Second element: %d\n", *ptr); // Output: 20

Pointer arithmetic allows you to efficiently iterate over elements of an array.
