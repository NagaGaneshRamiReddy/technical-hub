Python Tutorial

Introduction to Python

What is Python?

Python is a high-level, interpreted programming language known for its readability and simplicity. It is widely used for web development, data analysis, artificial intelligence, scientific computing, and more.

Why Learn Python?

Here are some reasons to learn Python:

  • Readability: Python's syntax is clean and easy to read, which makes it a great language for beginners.
  • Versatility: Python can be used in various domains such as web development, data science, automation, and more.
  • Community: Python has a large and active community, providing a wealth of libraries, frameworks, and resources.

Basic Structure of a Python Program

Python programs typically consist of functions and statements. Here’s a simple example:

print("Hello, World!")

In this example:

  • print("Hello, World!") - This prints "Hello, World!" to the screen.

Using Variables

Variables are used to store information. For example:

age = 10   # Stores a whole number
height = 5.9  # Stores a decimal number
initial = 'A'  # Stores a single letter

In Python:

  • int - Used for whole numbers (e.g., 1, 100). (No explicit type declaration needed in Python.)
  • float - Used for numbers with decimals (e.g., 3.14).
  • str - Used for strings (e.g., "A").

Making Decisions with if

Sometimes you need to make decisions in your program. You can use an if statement to do this:

age = 18

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

This checks if the age is 18 or more and prints a message accordingly.

Repeating Actions with Loops

Loops help you repeat actions. Here’s how you can use a loop to count from 1 to 5:

for i in range(1, 6):
    print(f"Count: {i}")

This loop prints numbers from 1 to 5, each on a new line.

Creating Functions

Functions are blocks of code that perform specific tasks. Here’s how you can create a function to add two numbers:

def add(a, b):
    return a + b

result = add(5, 3)
print(f"Sum: {result}")

The add function takes two numbers, adds them, and returns the result.

Using Libraries

Python has a rich standard library and many third-party libraries. Here’s an example of using the math library:

import math

print(math.sqrt(16))  # Prints the square root of 16

This example demonstrates how to use a library function to perform a task.

Summary

Python is a versatile and user-friendly programming language that is ideal for beginners and experts alike. Understanding its basics like variables, control structures, functions, and libraries will help you build effective programs and explore more advanced topics.
