Web Development Lab: Task - CSS Syntax

Task: Understanding CSS Syntax (30 Marks)

In this task, you need to apply CSS syntax correctly. Your CSS should include the following:

  • Create rules for styling an element using a class selector
  • Create rules for styling an element using an ID selector
  • Style multiple elements with a single selector
  • Use a universal selector to apply basic styles across the page
  • Apply styles using pseudo-classes (e.g., :hover)
  • Use at least one font styling property (e.g., font-size, font-family)
