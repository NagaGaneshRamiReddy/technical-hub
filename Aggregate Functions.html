Aggregate Functions

Aggregate Functions in SQL

Aggregate Functions perform a calculation on a set of values and return a single value. They are commonly used in conjunction with the GROUP BY clause to summarize data across multiple rows. Aggregate functions help in performing operations like summing, averaging, counting, and finding maximum or minimum values.

Common Aggregate Functions

Function Description Example
COUNT() Counts the number of rows in a set. Useful for determining the number of records. SELECT COUNT(*) FROM Orders;
SUM() Calculates the sum of a numeric column. Useful for totaling amounts. SELECT SUM(Salary) FROM Employees;
AVG() Calculates the average value of a numeric column. Useful for finding the mean value. SELECT AVG(Salary) FROM Employees;
MIN() Returns the minimum value of a column. Useful for finding the lowest value. SELECT MIN(Salary) FROM Employees;
MAX() Returns the maximum value of a column. Useful for finding the highest value. SELECT MAX(Salary) FROM Employees;

Using Aggregate Functions with GROUP BY

Aggregate functions are often used with the GROUP BY clause to group rows that have the same values into summary rows. Here’s an example:

SELECT Department, AVG(Salary) AS AverageSalary
FROM Employees
GROUP BY Department;

This query calculates the average salary for each department.

Examples of Aggregate Functions in Action

Counting Orders

To count the total number of orders in the Orders table:

SELECT COUNT(*) AS TotalOrders
FROM Orders;

Total Sales

To calculate the total sales amount from the Sales table:

SELECT SUM(SalesAmount) AS TotalSales
FROM Sales;

Average Employee Salary

To find the average salary of employees:

SELECT AVG(Salary) AS AverageSalary
FROM Employees;

Highest and Lowest Salaries

To find the highest and lowest salaries among employees:

SELECT MAX(Salary) AS HighestSalary, MIN(Salary) AS LowestSalary
FROM Employees;

Summary

Aggregate functions are essential tools for summarizing and analyzing data in SQL. By using functions such as COUNT(), SUM(), AVG(), MIN(), and MAX(), you can perform various calculations and gain valuable insights from your data. When combined with the GROUP BY clause, these functions provide powerful ways to group and summarize data.
