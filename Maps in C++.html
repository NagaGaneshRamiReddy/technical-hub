Maps in C++

Introduction

A map in C++ is a standard library container that stores key-value pairs in a sorted order. Each key in a map is unique and is associated with a specific value. The C++ Standard Library provides the map functionality through the std::map class. Maps are useful for situations where you need to associate unique keys with values and perform quick lookups.

Including the Map Header

To use maps, you need to include the header file <map>:

#include <map>

int main() {
    std::map<std::string, int> myMap;  // Declares an empty map with string keys and integer values
    return 0;
}

Creating and Initializing Maps

You can create and initialize a map using the std::map class. Keys are unique, and the map will automatically sort the keys:

#include <iostream>
#include <map>

int main() {
    std::map<std::string, int> myMap = {
        {"Alice", 30},
        {"Bob", 25},
        {"Charlie", 35}
    };

    // Displaying elements
    for (const auto &pair : myMap) {
        std::cout << pair.first << " is " << pair.second << " years old" << std::endl;
    }

    return 0;
}

Basic Operations

Maps support several key operations:

  • Insertion: Add key-value pairs using insert() or the subscript operator.
  • Deletion: Remove key-value pairs using erase().
  • Find: Locate a value by key using find().
  • Access: Access values using the subscript operator or at().
  • Size: Get the number of elements using size().
  • Clear: Remove all elements using clear().

Example Program

Hereâ€™s a sample program demonstrating basic operations with a map:

#include <iostream>
#include <map>

int main() {
    std::map<std::string, int> myMap;

    // Adding key-value pairs
    myMap["Alice"] = 30;
    myMap["Bob"] = 25;
    myMap["Charlie"] = 35;

    // Displaying elements
    std::cout << "Map contents:" << std::endl;
    for (const auto &pair : myMap) {
        std::cout << pair.first << " is " << pair.second << " years old" << std::endl;
    }

    // Finding an element
    auto search = myMap.find("Bob");
    if (search != myMap.end()) {
        std::cout << "Bob's age is " << search->second << std::endl;
    }

    // Removing an element
    myMap.erase("Charlie");

    // Displaying elements after removal
    std::cout << "Map contents after removing Charlie:" << std::endl;
    for (const auto &pair : myMap) {
        std::cout << pair.first << " is " << pair.second << " years old" << std::endl;
    }

    return 0;
}

Common Use Cases

Maps are useful in various scenarios:

  • Key-Value Associations: Store data where each key is unique and maps to a specific value.
  • Quick Lookups: Efficiently find values associated with specific keys.
  • Sorting: Maintain keys in a sorted order automatically.
