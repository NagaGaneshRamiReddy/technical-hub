if-else statement in java

Introduction

The if-else statement is one of the fundamental control structures in Java. It allows you to execute different blocks of code based on whether a specified condition is true or false. This guide covers the syntax and usage of if-else statements, including variations like if-else if-else and nested if-else statements.

1. Basic If Statement

The simplest form of an if statement executes a block of code if a specified condition is true:

int number = 10;

if (number > 0) {
    System.out.println("The number is positive.");
}

In this example, the message will be printed only if `number` is greater than 0.

2. If-Else Statement

An if-else statement allows you to specify a block of code to execute if the condition is true and another block if the condition is false:

int number = -5;

if (number > 0) {
    System.out.println("The number is positive.");
} else {
    System.out.println("The number is not positive.");
}

Here, the program will print "The number is not positive." because number is less than 0.

3. If-Else If-Else Statement

You can chain multiple conditions using if-else if-else statements. This allows for more complex decision-making:

int number = 0;

if (number > 0) {
    System.out.println("The number is positive.");
} else if (number < 0) {
    System.out.println("The number is negative.");
} else {
    System.out.println("The number is zero.");
}

In this example, the program will check each condition in order and print the appropriate message based on the value of number.

4. Nested If-Else Statements

Nested if-else statements are if statements inside other if statements. This allows for more granular decision-making:

int number = 15;

if (number != 0) {
    if (number > 0) {
        System.out.println("The number is positive.");
    } else {
        System.out.println("The number is negative.");
    }
} else {
    System.out.println("The number is zero.");
}

Here, the program first checks if number is not zero. If true, it then checks whether number is positive or negative.

5. Ternary Operator

The ternary operator (? :) is a shorthand for the if-else statement. It provides a concise way to assign values based on a condition:

int number = 10;
String result = (number > 0) ? "Positive" : "Not Positive";
System.out.println(result); // Output: Positive

In this example, result will be assigned "Positive" if number is greater than 0, otherwise it will be assigned "Not Positive".

Summary

If-else statements are a crucial part of Java programming that allow you to control the flow of your program based on conditions. By mastering if-else, if-else if-else, nested if-else, and the ternary operator, you'll be able to handle various decision-making scenarios effectively in your code.
