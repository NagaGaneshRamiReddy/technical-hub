Rust Vectors

Understanding Vectors in Rust

Vectors are dynamic arrays that allow you to store a sequence of elements. They provide efficient access and modification, making them one of the most commonly used collection types in Rust.

Creating a Vector

You can create a vector using the vec! macro or by using the Vec::new() method:

fn main() {
    // Using the vec! macro
    let numbers = vec![1, 2, 3, 4, 5];

    // Using Vec::new()
    let mut empty_vector: Vec = Vec::new();
    empty_vector.push(6);
    empty_vector.push(7);
    println!("{:?}", numbers); // Outputs: [1, 2, 3, 4, 5]
    println!("{:?}", empty_vector); // Outputs: [6, 7]
}

Accessing Elements

Elements in a vector can be accessed using indexing:

fn main() {
    let numbers = vec![10, 20, 30, 40];
    
    // Accessing elements
    let first = &numbers[0];
    let second = numbers.get(1);
    
    println!("First: {}", first); // Outputs: First: 10
    match second {
        Some(value) => println!("Second: {}", value), // Outputs: Second: 20
        None => println!("No second value."),
    }
}

Modifying a Vector

You can add or remove elements from a vector easily:

fn main() {
    let mut vec = vec![1, 2, 3];

    // Adding elements
    vec.push(4);
    vec.push(5);
    
    // Removing the last element
    vec.pop();

    println!("{:?}", vec); // Outputs: [1, 2, 3, 4]
}

Iterating Over a Vector

You can iterate over the elements in a vector using a for loop:

fn main() {
    let vec = vec![10, 20, 30];

    for value in &vec {
        println!("{}", value); // Outputs: 10, 20, 30 (each on a new line)
    }
}

Summary

Vectors are a powerful and flexible way to store ordered collections of data in Rust. They support dynamic sizing and provide methods for adding, removing, and accessing elements efficiently.
