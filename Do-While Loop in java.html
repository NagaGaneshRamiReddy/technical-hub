Do-while loop in Java

Introduction

The do-while loop in Java is a control flow statement that repeatedly executes a block of code as long as a specified condition remains true. Unlike the while loop, the do-while loop guarantees that the code block will execute at least once before the condition is evaluated. This guide will explain the syntax and usage of the do-while loop with examples.

1. Basic Syntax

The basic syntax of a do-while loop is as follows:

do {
    // Code to be executed
} while (condition);
  • Code to be Executed: This block of code is executed once before the condition is checked.
  • Condition: Evaluated after each iteration. If the condition evaluates to true, the loop executes the code block again; otherwise, the loop terminates.

2. Example: Counting from 0 to 4

Here’s a simple example that uses a do-while loop to count from 0 to 4:

int i = 0;

do {
    System.out.println("Count: " + i);
    i++;
} while (i < 5);

In this example, the loop starts with i initialized to 0. The code block is executed first, printing the value of i and then incrementing i by 1. The condition i < 5 is checked after each iteration. The loop terminates when i reaches 5.

3. Example: User Input Loop

The do-while loop can also be used to repeatedly prompt the user for input, ensuring the code block executes at least once:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int number;

do {
    System.out.print("Enter a number between 1 and 10: ");
    number = scanner.nextInt();
} while (number < 1 || number > 10);

System.out.println("Valid number entered: " + number);

In this example, the loop prompts the user for a number and ensures that the prompt occurs at least once, even if the user provides an invalid number initially.

4. Example: Infinite Loop

A do-while loop can run indefinitely if the condition always evaluates to true, similar to a while loop:

do {
    System.out.println("This will run forever.");
} while (true);

In this example, the loop will run indefinitely because the condition true is always true. Be cautious with infinite loops as they can cause programs to become unresponsive.

5. Breaking Out of a Do-While Loop

You can use the break statement to exit a do-while loop prematurely based on a specific condition:

int i = 0;

do {
    if (i >= 5) {
        break; // Exit the loop when i is 5 or greater
    }
    System.out.println("Count: " + i);
    i++;
} while (true);

In this example, the loop will terminate when i reaches 5, even though the condition of the do-while loop is always true.

6. Continue Statement in Do-While Loop

You can use the continue statement to skip the remaining code in the current iteration and proceed with the next iteration of the loop:

int i = 0;

do {
    if (i % 2 == 0) {
        i++;
        continue; // Skip printing for even numbers
    }
    System.out.println("Odd Count: " + i);
    i++;
} while (i < 10);

In this example, the continue statement causes the loop to skip printing even numbers and only print odd numbers.

Summary

The do-while loop is a useful control flow statement in Java for scenarios where you need to ensure that the loop body is executed at least once. It evaluates the loop condition after each iteration, making it suitable for cases where the initial execution of the loop is necessary. Understanding how to use the do-while loop effectively, including handling infinite loops and using break and continue statements, will enhance your programming skills and provide greater flexibility in handling repetitive tasks.
