Command line arguments in C

Introduction

Command line arguments are parameters passed to a program when it is executed from the command line. They provide a way to pass data into the program at runtime. In C, command line arguments are handled using the argc and argv parameters of the main function.

Understanding argc and argv

In C, the main function can take two parameters:

  • int argc: The argument count, which represents the number of command line arguments passed to the program.
  • char *argv[]: The argument vector, which is an array of strings (character arrays) representing each command line argument.

Example Program

The following example demonstrates how to use argc and argv to process command line arguments:

#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Number of arguments: %d\n", argc);

    for (int i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }

    return 0;
}

In this example:

  • argc holds the number of arguments passed to the program.
  • argv is an array of strings where each element represents a command line argument.

Running the Program

To run the program with command line arguments, compile and execute it from the command line:

$ gcc program.c -o program
$ ./program arg1 arg2 arg3

Output:

Number of arguments: 4
Argument 0: ./program
Argument 1: arg1
Argument 2: arg2
Argument 3: arg3

The first argument (argv[0]) is always the name of the program itself. Subsequent arguments are those passed by the user.

Handling Command Line Arguments

Command line arguments can be used to customize the behavior of a program. For instance, you can use them to specify file names, set options, or provide input data. Proper handling of these arguments can improve the flexibility and usability of your programs.

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    if (argc != 3) {
        printf("Usage: %s  \n", argv[0]);
        return 1;
    }

    int num1 = atoi(argv[1]);
    int num2 = atoi(argv[2]);
    printf("Sum: %d\n", num1 + num2);

    return 0;
}

In this example, the program expects exactly two arguments. It converts them to integers using atoi and computes their sum.

Summary

Command line arguments are a powerful feature in C programming that allow users to provide input to programs at runtime. By understanding and utilizing argc and argv, you can create more flexible and interactive programs. Mastering command line arguments can enhance the versatility and usability of your C applications.
