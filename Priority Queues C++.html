priority queue in C++

Introduction

A priority queue is a specialized type of queue where each element has a priority associated with it. Elements are removed from the queue based on their priority rather than the order in which they were added. The highest priority elements are served before lower priority ones. Priority queues are commonly used in scenarios like scheduling tasks and implementing algorithms such as Dijkstra's shortest path.

Including the Priority Queue Header

To use priority queues in C++, include the header file <queue>. The priority queue is implemented as a container adaptor that uses a underlying container (by default a max-heap) to maintain the priority order:

#include <queue>

int main() {
    std::priority_queue<int> pq;  // Declares an empty priority queue of integers
    return 0;
}

Initializing and Adding Elements

You can initialize a priority queue and add elements using the push() method. By default, std::priority_queue is a max-heap, meaning the largest element has the highest priority:

#include <iostream>
#include <queue>

int main() {
    std::priority_queue<int> pq;

    // Adding elements
    pq.push(10);
    pq.push(30);
    pq.push(20);

    // Displaying elements
    while (!pq.empty()) {
        std::cout << pq.top() << " ";  // Display the highest priority element
        pq.pop();  // Remove the highest priority element
    }
    std::cout << std::endl;

    return 0;
}

Accessing and Removing Elements

Priority queues provide several methods to access and remove elements:

  • top: Returns a reference to the highest priority element in the queue.
  • pop: Removes the highest priority element from the queue.
  • push: Adds a new element to the queue.
  • empty: Checks if the queue is empty.
  • size: Returns the number of elements in the queue.

Custom Comparison

You can customize the priority queue to use different comparison logic. For example, to create a min-heap instead of a max-heap, you can provide a custom comparison function:

#include <iostream>
#include <queue>>
#include <vector>>

struct Compare {
    bool operator()(int a, int b) {
        return a > b;  // Min-heap: smaller elements have higher priority
    }
};

int main() {
    std::priority_queue<int, std::vector<int>, Compare> pq;

    pq.push(10);
    pq.push(30);
    pq.push(20);

    // Displaying elements
    std::cout << "Priority queue elements:" << std::endl;
    while (!pq.empty()) {
        std::cout << pq.top() << " ";
        pq.pop();
    }
    std::cout << std::endl;

    return 0;
}

Common Use Cases

Priority queues are useful in a variety of scenarios:

  • Task Scheduling: Managing tasks where certain tasks have higher importance or urgency.
  • Graph Algorithms: Implementing algorithms like Dijkstra's shortest path where nodes are processed based on distance.
  • Event Simulation: Handling events in simulations where events occur at different times with different priorities.
