How to Add CSS

Introduction to Adding CSS

CSS (Cascading Style Sheets) can be added to HTML documents in several ways. Each method offers different benefits depending on the size and scope of your project. Hereâ€™s how you can add CSS to your HTML:

1. Inline CSS

Inline CSS involves adding styles directly within an HTML element using the style attribute. This method is best for quick, specific styles but is not suitable for larger projects.


<p style="color: red; font-size: 20px;">This text is red and 20px in size.</p>

2. Internal CSS

Internal CSS is added within a <style> tag in the <head> section of your HTML document. This method is useful for styling a single page.


<head>
    <style>
        body {
            background-color: #f4f4f4;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: blue;
        }
    </style>
</head>

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Internal CSS Example</title>
    <style>
        body {
            background-color: #f4f4f4;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: blue;
        }
    </style>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is a sample paragraph styled with internal CSS.</p>
</body>
</html>

3. External CSS

External CSS involves creating a separate CSS file and linking it to your HTML document. This is the most efficient method for larger projects and for applying consistent styles across multiple pages.

Steps:

  1. Create a CSS File: Create a file named styles.css and write your CSS rules in this file.
  2. Link the CSS File: Use the <link> tag in the <head> section of your HTML document to include the CSS file.

<link rel="stylesheet" href="styles.css">

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>External CSS Example</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is a sample paragraph styled with external CSS.</p>
</body>
</html>

4. Using CSS Frameworks

CSS frameworks like Bootstrap and Tailwind CSS provide pre-designed styles and components that can speed up development. You include these frameworks by adding a link to their CSS file in the <head> section of your HTML document.


<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">

Example with Bootstrap:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Example</title>
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
</head>
<body>
    <div class="container">
        <h1 class="text-primary">Welcome to My Website</h1>
        <p>This is a sample paragraph styled with Bootstrap.</p>
    </div>
</body>
</html>

Summary

Adding CSS to your HTML document can be done through inline styles, internal styles, external stylesheets, or CSS frameworks. Each method has its own use cases, and choosing the right one depends on the needs of your project.
