HTML Basics

Overview

HTML (Hypertext Markup Language) is the standard language used to create webpages. It defines the structure of a webpage by using various elements and tags. In this section, we'll cover the basics of HTML, including common tags and how they are used to build simple web pages.

Basic HTML Structure

The foundation of any HTML document consists of specific tags that define the document’s structure. Here's an example of a simple HTML page:

    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My First HTML Page</title>
      </head>
      <body>
        <h1>Hello, World!</h1>
        <p>This is a simple paragraph of text on my first HTML page.</p>
      </body>
    </html>

Common HTML Tags

Below are some of the most commonly used HTML tags to structure and format content:

  • <h1> to <h6>: Heading tags, used to define headers. <h1> is the largest, and <h6> is the smallest.
  • <p>: Defines a paragraph of text.
  • <a>: Defines a hyperlink. Example: <a href="https://example.com">Click here</a>.
  • <img>: Embeds an image. Example: <img src="image.jpg" alt="Description">.
  • <ul> and <ol>: Define unordered (bulleted) and ordered (numbered) lists, respectively.
  • <li>: Defines list items, used inside <ul> or <ol>.
  • <div>: A container element for grouping HTML content. Useful for organizing and styling.

Attributes in HTML

HTML elements can have attributes that provide additional information about the element. Attributes are written inside the opening tag and usually come in name/value pairs. For example:

    <a href="https://www.example.com">Visit Example</a>

In this case, the href attribute specifies the URL the link points to.

HTML Formatting Tags

HTML also includes tags to format text content. Some commonly used formatting tags include:

  • <strong>: Makes text bold. Example: <strong>Bold text</strong>.
  • <em>: Italicizes text (emphasized). Example: <em>Italic text</em>.
  • <br>: Inserts a line break, creating a new line in the text.
  • <hr>: Inserts a horizontal line (a divider).

Best Practices for Writing HTML

  • Proper Indentation: Indent your HTML code for better readability and structure.
  • Use Semantic Tags: Use tags that describe the content, such as <header>, <footer>, and <article> for better organization.
  • Validate Your Code: Use an HTML validator to check for errors and ensure your code follows HTML standards.
  • Keep HTML Clean: Avoid unnecessary or outdated tags (such as <font>), and focus on clean, modern code.

Summary

HTML is the foundation of web development. By learning the basic structure, common tags, and best practices, you can begin creating structured and accessible web pages. As you become more familiar with HTML, you’ll be able to build more complex and feature-rich websites.
