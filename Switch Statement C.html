switch statement in C

Introduction

The switch statement in C programming is used to execute one block of code among multiple possible blocks based on the value of an expression. It provides a more readable and efficient way to handle multiple conditions compared to a series of if-else statements.

1. Basic Syntax

The basic syntax of a switch statement is as follows:


switch (expression) {
    case value1:
        // code to be executed if expression == value1
        break;
    case value2:
        // code to be executed if expression == value2
        break;
    // more cases...
    default:
        // code to be executed if expression doesn't match any case
}

Flowchart

  • Start
    • Evaluate Expression
      • Case 1
        • Execute Case 1 Block
        • Break or End
      • Case 2
        • Execute Case 2 Block
        • Break or End
      • Default Case
        • Execute Default Block
        • End

2. Example of Switch Statement

Here is an example of a switch statement that evaluates an integer variable and prints the corresponding day of the week:


int day = 3;

switch (day) {
    case 1:
        printf("Monday\n");
        break;
    case 2:
        printf("Tuesday\n");
        break;
    case 3:
        printf("Wednesday\n");
        break;
    case 4:
        printf("Thursday\n");
        break;
    case 5:
        printf("Friday\n");
        break;
    case 6:
        printf("Saturday\n");
        break;
    case 7:
        printf("Sunday\n");
        break;
    default:
        printf("Invalid day\n");
}

In this example, "Wednesday" will be printed because the value of day is 3.

3. Using Switch with Characters

The switch statement can also be used with characters. Here's an example:


char grade = 'B';

switch (grade) {
    case 'A':
        printf("Excellent\n");
        break;
    case 'B':
        printf("Good\n");
        break;
    case 'C':
        printf("Average\n");
        break;
    case 'D':
        printf("Below Average\n");
        break;
    case 'F':
        printf("Fail\n");
        break;
    default:
        printf("Invalid grade\n");
}

In this case, "Good" will be printed because the value of grade is 'B'.

4. Fall-Through Behavior

If a case statement does not have a break statement, the control will fall through to the next case. This can be useful in some scenarios:


int number = 2;

switch (number) {
    case 1:
    case 2:
        printf("Number is 1 or 2\n");
        break;
    case 3:
        printf("Number is 3\n");
        break;
    default:
        printf("Number is not 1, 2, or 3\n");
}

In this example, "Number is 1 or 2" will be printed because number is 2, and the control falls through from case 1 to case 2.

5. Using Switch for Menu Selection

The switch statement is often used to handle menu selections in console applications. Here's a simple example:


int choice;

printf("Enter your choice (1-3):\n");
printf("1. Option 1\n");
printf("2. Option 2\n");
printf("3. Option 3\n");
scanf("%d", &choice);

switch (choice) {
    case 1:
        printf("You selected Option 1\n");
        break;
    case 2:
        printf("You selected Option 2\n");
        break;
    case 3:
        printf("You selected Option 3\n");
        break;
    default:
        printf("Invalid choice\n");
}

This program displays a menu and performs an action based on the user's selection.
