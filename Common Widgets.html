Common Widgets: Text, Button, Image

Introduction to Common Widgets

In Flutter, widgets are the building blocks of the UI. Some of the most commonly used widgets in Flutter applications include Text, Button, and Image widgets. These widgets allow developers to create user interfaces that display text, capture user actions, and display images easily.

1. Text Widget

The Text widget in Flutter is used to display text in the UI. It allows you to customize the style, alignment, and behavior of the text.

Example of Text Widget

Text(
  'Hello, Flutter!',
  style: TextStyle(
    fontSize: 24,
    color: Colors.blue,
    fontWeight: FontWeight.bold,
  ),
)

Explanation:

  • The Text widget displays the text "Hello, Flutter!".
  • The TextStyle allows us to customize the appearance, setting the font size, color, and weight.

2. Button Widgets

Flutter provides several types of buttons, such as ElevatedButton, TextButton, and IconButton, for user interactions.

Example of ElevatedButton

ElevatedButton(
  onPressed: () {
    print('Button Pressed!');
  },
  child: Text('Press Me'),
)

Explanation:

  • The ElevatedButton widget creates a button that triggers an action when pressed.
  • The onPressed callback is triggered when the button is clicked, and in this example, it prints a message to the console.
  • The Text widget inside the button displays "Press Me".

3. Image Widget

The Image widget is used to display images in a Flutter application. Flutter supports images from assets, the network, or directly from memory.

Example of Image Widget (Network Image)

Image.network(
  'https://flutter.dev/images/flutter-logo-sharing.png',
  width: 100,
  height: 100,
)

Explanation:

  • The Image.network widget loads and displays an image from the provided URL.
  • The width and height properties set the size of the image to 100x100 pixels.

Combining Widgets

In Flutter, you can combine widgets to build more complex UIs. Below is an example of a simple UI that uses the Text, ElevatedButton, and Image widgets together.

Example of Combining Widgets

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Common Widgets Example')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: < Widget >[
              Text(
                'Hello, Flutter!',
                style: TextStyle(fontSize: 24, fontWeight: FontWeight.bold),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  print('Button Pressed!');
                },
                child: Text('Press Me'),
              ),
              SizedBox(height: 20),
              Image.network(
                'https://flutter.dev/images/flutter-logo-sharing.png',
                width: 100,
                height: 100,
              ),
            ],
          ),
        ),
      ),
    );
  }
}

Explanation:

  • The Column widget arranges the child widgets vertically.
  • It contains a Text widget, an ElevatedButton, and an Image widget.
  • The SizedBox widgets add spacing between the elements.
