Bootstrap Utilities


Introduction to Bootstrap Utilities

Bootstrap provides a variety of utility classes to simplify common CSS tasks such as margin, padding, display, and more. These utilities help you quickly apply styles without writing custom CSS.

Margin and Padding Utilities

Bootstrap includes utility classes for margins and padding that follow the format: .m{t|r|b|l|x|y}-{breakpoint}-{value}. The values can range from 0 to 5, and you can use auto for margins.

<div class="m-3">This div has a margin of 3 units.</div>
<div class="p-2">This div has padding of 2 units.</div>

Expected Output:

This div has a margin of 3 units.
This div has padding of 2 units.

Text Utilities

Bootstrap provides classes to manage text alignment and color.

<p class="text-center">This text is centered.</p>
<p class="text-danger">This text is red.</p>

Expected Output:

This text is centered.

This text is red.

Display Utilities

Display utility classes can be used to control the display property of an element.

<div class="d-none">This div is hidden.</div>
<div class="d-block">This div is displayed as a block element.</div>

Expected Output:

This div is displayed as a block element.
