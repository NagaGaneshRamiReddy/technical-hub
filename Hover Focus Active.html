Hover, Focus, and Active Pseudo-Classes

Hover Pseudo-Class

The :hover pseudo-class applies a style when the user hovers over an element.


/* Change button and link background on hover */
button:hover, a:hover {
    background-color: #0066cc;
    color: white;
}

Focus Pseudo-Class

The :focus pseudo-class applies a style when an element gains focus, such as when clicked or tabbed into.


/* Change input and button background and outline on focus */
button:focus, input:focus {
    outline: 3px solid #33cc33;
    background-color: #e0ffe0;
}

Active Pseudo-Class

The :active pseudo-class applies a style when the user activates an element (usually by clicking it).


/* Change button and link background on active */
button:active, a:active {
    background-color: #cc3300;
    color: white;
}

Summary

The :hover, :focus, and :active pseudo-classes are used to style elements when users interact with them. These interactions enhance the user experience by providing visual feedback during various states.
