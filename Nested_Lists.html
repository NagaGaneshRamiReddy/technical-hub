Nested Lists

Understanding Nested Lists in HTML

A nested list is a list that contains another list as a sub-item. You can create nested lists using the <ul> or <ol> tags inside another list item (<li>).

Example of Nested Unordered Lists

Here’s how to create a nested unordered list:

<ul>
    <li>Fruits
        <ul>
            <li>Apple</li>
            <li>Banana</li>
            <li>Orange</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrot</li>
            <li>Broccoli</li>
        </ul>
    </li>
</ul>

Expected Output:

  • Fruits
    • Apple
    • Banana
    • Orange
  • Vegetables
    • Carrot
    • Broccoli

Example of Nested Ordered Lists

You can also nest ordered lists. Here’s an example:

<ol>
    <li>Plan a Trip
        <ol>
            <li>Choose a Destination</li>
            <li>Book Flights</li>
        </ol>
    </li>
    <li>Pack Bags</li>
</ol>

Expected Output:

  1. Plan a Trip
    1. Choose a Destination
    2. Book Flights
  2. Pack Bags

Nesting Mixed Lists

You can mix ordered and unordered lists together for more complex structures:

<ul>
    <li>Travel
        <ol>
            <li>Countries to Visit
                <ul>
                    <li>France</li>
                    <li>Japan</li>
                </ul>
            </li>
            <li>Travel Essentials</li>
        </ol>
    </li>
</ul>

Expected Output:

  • Travel
    1. Countries to Visit
      • France
      • Japan
    2. Travel Essentials
