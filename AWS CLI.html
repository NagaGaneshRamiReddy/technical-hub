AWS CLI

Getting Started with AWS CLI

The AWS CLI is a handy tool from Amazon that lets you interact with AWS services using simple text commands. It’s like having a remote control for AWS, letting you manage your resources right from your terminal or command prompt. This makes it super easy to automate tasks and keep your AWS environment running smoothly.

Why Use AWS CLI?

  • Command-Based Interface: Run commands to manage your AWS services. Need to launch an instance or set up a bucket? Just type a command!
  • Automate Tasks: Write scripts to automate repetitive tasks and integrate AWS into your workflows. Save time and reduce errors.
  • Consistent Experience: Use a consistent command structure across all AWS services, making it easier to learn and use.
  • Works on Any Platform: Available for Windows, macOS, and Linux, so you can use it no matter what system you’re on.
  • Customizable: Configure settings and manage multiple profiles to handle different AWS accounts and regions.

How to Install AWS CLI

  1. Download: Visit the AWS CLI official page and get the installer for your operating system.
  2. Install: Follow the installation guide for your platform. Windows users can use the installer, while macOS and Linux users can use a package manager.
  3. Set Up: After installing, open your terminal and run aws configure to set up your credentials, default region, and output format.

Basic Commands You’ll Use

  • List S3 Buckets: aws s3 ls - This command shows all the S3 buckets in your account.
  • Describe EC2 Instances: aws ec2 describe-instances - Get details about your EC2 instances.
  • Create an S3 Bucket: aws s3 mb s3://my-bucket - Creates a new S3 bucket called "my-bucket".
  • Start an EC2 Instance: aws ec2 start-instances --instance-ids i-1234567890abcdef0 - Starts an EC2 instance with the specified ID.
  • Stop an EC2 Instance: aws ec2 stop-instances --instance-ids i-1234567890abcdef0 - Stops the EC2 instance with the given ID.

Example: Launching an EC2 Instance

Here’s a quick guide to launching an EC2 instance using the AWS CLI:

  1. Open your terminal or command prompt.
  2. Type the command: aws ec2 run-instances --image-id ami-0abcdef1234567890 --count 1 --instance-type t2.micro --key-name MyKeyPair to start a new instance.
  3. Replace ami-0abcdef1234567890 with your AMI ID, t2.micro with your instance type, and MyKeyPair with your key pair name.
