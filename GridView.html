GridView in Flutter

Introduction to GridView

The GridView widget in Flutter allows you to create a scrollable grid of widgets. It is similar to a ListView, but instead of being a single column or row, it arranges its children in a grid format. You can use it to create layouts like galleries or display items in a matrix form.

Types of GridView

There are multiple types of GridView widgets in Flutter:

  • GridView.count: Defines a fixed number of tiles in each row or column.
  • GridView.builder: Dynamically builds its child widgets based on the data source.
  • GridView.extent: Specifies the maximum pixel width or height for a tile, and Flutter will calculate how many tiles fit.
  • GridView.custom: Allows you to create a custom grid layout with your own delegate.

1. GridView.count Example

This type of GridView is used when you want to have a specific number of items per row or column.

GridView.count(
  crossAxisCount: 2,
  children: List.generate(6, (index) {
    return Container(
      color: Colors.blue,
      margin: EdgeInsets.all(8.0),
      child: Center(
        child: Text(
          'Item \$index',
          style: TextStyle(color: Colors.white, fontSize: 18),
        ),
      ),
    );
  }),
)

Expected Output:

A grid with two columns displaying six items, each represented by a blue box with text inside (e.g., "Item 1", "Item 2"). The boxes are separated by a small margin.

2. GridView.builder Example

GridView.builder is useful when you have a large data set and want to create child widgets dynamically. It builds items on demand as the user scrolls.

GridView.builder(
  gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
    crossAxisCount: 3,
    crossAxisSpacing: 8.0,
    mainAxisSpacing: 8.0,
  ),
  itemCount: 10,
  itemBuilder: (context, index) {
    return Container(
      color: Colors.green,
      child: Center(
        child: Text(
          'Item \$index',
          style: TextStyle(color: Colors.white, fontSize: 16),
        ),
      ),
    );
  },
)

Expected Output:

A grid with three columns displaying 10 items, each represented by a green box with text inside (e.g., "Item 1", "Item 2"). The boxes have spacing between them.

3. GridView.extent Example

The GridView.extent widget allows you to define the maximum size of each tile, and Flutter will figure out how many tiles can fit in the grid.

GridView.extent(
  maxCrossAxisExtent: 150,
  padding: EdgeInsets.all(8.0),
  children: List.generate(8, (index) {
    return Container(
      color: Colors.orange,
      child: Center(
        child: Text(
          'Item \$index',
          style: TextStyle(color: Colors.white, fontSize: 18),
        ),
      ),
    );
  }),
)

Expected Output:

A grid where each item is a maximum of 150 pixels wide, displaying eight items in orange boxes with text inside (e.g., "Item 1", "Item 2").

4. GridView.custom Example

GridView.custom allows you to define your own grid layout using SliverGridDelegate and other custom properties.

GridView.custom(
  gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
    crossAxisCount: 2,
    crossAxisSpacing: 8.0,
    mainAxisSpacing: 8.0,
  ),
  childrenDelegate: SliverChildListDelegate(
    List.generate(4, (index) {
      return Container(
        color: Colors.purple,
        child: Center(
          child: Text(
            'Item \$index',
            style: TextStyle(color: Colors.white, fontSize: 16),
          ),
        ),
      );
    }),
  ),
)

Expected Output:

A grid with two columns, displaying four items in purple boxes with text inside (e.g., "Item 1", "Item 2"). The layout is fully customizable.

Summary

GridView is a powerful widget that allows you to display items in a grid format, with the ability to customize the number of rows and columns, spacing, and much more. It is particularly useful for creating layouts like photo galleries or product lists, and its various types such as GridView.count and GridView.builder offer flexibility for different scenarios.
