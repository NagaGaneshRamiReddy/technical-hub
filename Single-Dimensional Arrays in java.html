Single-Dimensional Array in Java

What is a Single-Dimensional Array?

A single-dimensional array in Java is a linear array that holds a fixed number of values of the same type. It is essentially a list of elements that can be accessed using a single index. Each element is stored in contiguous memory locations, making it easy to access and manipulate data sequentially.

Declaring a Single-Dimensional Array

To declare a single-dimensional array in Java, you specify the type of elements the array will hold, followed by square brackets. Here’s the basic syntax:

type[] arrayName;

For example, to declare an array of integers:

int[] numbers;

Initializing a Single-Dimensional Array

After declaring an array, you need to initialize it to allocate memory and optionally set its values. There are two main ways to initialize an array:

  • Direct Initialization: Provide values at the time of declaration.
  • Dynamic Initialization: Allocate memory and then assign values.

Direct Initialization

int[] numbers = {1, 2, 3, 4, 5};

Dynamic Initialization

// Declare and allocate memory
int[] numbers = new int[5];

// Assign values
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

Accessing and Modifying Elements

Elements in a single-dimensional array are accessed using their index, which starts from 0. You can also modify array elements using their index. For example:

// Accessing elements
int firstNumber = numbers[0]; // Accesses the first element

// Modifying elements
numbers[2] = 10; // Changes the third element to 10

Traversing a Single-Dimensional Array

To process or display each element of the array, you can use a loop to traverse it. The most common loop used is the for loop:

// Traverse and print all elements
for (int i = 0; i < numbers.length; i++) {
    System.out.println("Element at index " + i + ": " + numbers[i]);
}

Example: Working with Single-Dimensional Arrays

Here’s a complete example demonstrating the declaration, initialization, and usage of a single-dimensional array:

public class ArrayExample {
    public static void main(String[] args) {
        // Declare and initialize an array
        int[] numbers = {10, 20, 30, 40, 50};

        // Access and modify elements
        numbers[2] = 35; // Change the third element

        // Print all elements
        for (int i = 0; i < numbers.length; i++) {
            System.out.println("Element at index " + i + ": " + numbers[i]);
        }
    }
}

Summary

Single-dimensional arrays in Java are a fundamental data structure that allows you to store and manage a fixed number of elements of the same type. They provide a simple and efficient way to organize and access data. Understanding how to declare, initialize, and use single-dimensional arrays is essential for effective Java programming.
