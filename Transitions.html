CSS Transitions

What are CSS Transitions?

CSS transitions provide a way to make gradual changes between property values. They allow you to define the duration and timing of the transition, enabling smooth animations for CSS properties when their values change.

Basic Transition Example

Hover Me

Transition Properties

Transition Properties:

  • transition - A shorthand property for transition-property, transition-duration, transition-timing-function, and transition-delay.
  • transition-property - Specifies the CSS properties to which the transition is applied.
  • transition-duration - Defines how long the transition should take.
  • transition-timing-function - Describes how the intermediate property values are calculated.
  • transition-delay - Specifies when the transition should start.

Code Example


/* Basic Transition */
.transition-box {
    width: 100px;
    height: 100px;
    margin: 10px;
    background-color: #4CAF50;
    border: 2px solid #333;
    transition: all 0.5s ease;
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    font-weight: bold;
}

/* Transition Effects on Hover and Active */
.transition-box:hover {
    background-color: #FFC107;
    transform: scale(1.2);
    border-color: #F44336;
}
.transition-box:active {
    background-color: #FF5722;
    transform: scale(0.9);
}

Summary

CSS transitions offer a simple way to create smooth changes between property values. By using transitions, you can enhance the user experience with gradual animations that respond to user interactions.
