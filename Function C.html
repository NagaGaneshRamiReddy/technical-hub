Function in C

Introduction

In C programming, functions are used to encapsulate code into reusable blocks. Functions help in organizing code, making it modular, and improving readability. They allow you to perform specific tasks and can be called from other parts of the program.

1. Function Definition

A function in C is defined with a return type, a function name, and a parameter list (optional). The function body contains the code to be executed when the function is called.

Syntax


return_type function_name(parameter_list) {
    // function body
}

Example

Here’s an example of a simple function that adds two numbers:


#include <stdio.h>

// Function declaration
int add(int a, int b);

int main() {
    int result = add(5, 3);
    printf("Sum: %d\n", result);
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

In this example:

  • The function add is declared before main() and defined after it.
  • It takes two integer parameters and returns their sum.
  • The function is called from main(), and its result is printed to the console.

2. Function Declaration and Definition

Functions must be declared before they are used in a program. The declaration provides the function's name, return type, and parameters. The definition provides the actual implementation of the function.

Function Declaration


return_type function_name(parameter_list);

Function Definition


return_type function_name(parameter_list) {
    // function body
}

3. Function Parameters and Return Values

Functions can take parameters (input values) and return a result (output value). Parameters are specified in the function definition, and the return value is specified by the function’s return type.

Parameters

Parameters are specified in the function declaration and definition. They act as placeholders for the values that are passed to the function.

Return Values

The return value is specified by the function's return type. If a function does not return a value, its return type should be void.

4. Recursion

Recursion is a technique where a function calls itself. It is useful for solving problems that can be broken down into smaller, similar problems.

Example

Here’s an example of a recursive function that calculates the factorial of a number:


#include <stdio.h>

// Function declaration
int factorial(int n);

int main() {
    int result = factorial(5);
    printf("Factorial: %d\n", result);
    return 0;
}

// Function definition
int factorial(int n) {
    if (n == 0) {
        return 1; // Base case
    } else {
        return n * factorial(n - 1); // Recursive case
    }
}

In this example:

  • The function factorial calculates the factorial of a number using recursion.
  • The base case is when n is 0, returning 1.
  • The recursive case multiplies n by the result of factorial(n - 1).

5. Function Overloading

C does not support function overloading. However, different functions with different names can be created to perform similar tasks.
