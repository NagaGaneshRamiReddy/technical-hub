PHP Arrays

Overview

Arrays in PHP are data structures that allow you to store multiple values in a single variable. They are incredibly versatile and widely used to manage collections of data efficiently.

Types of Arrays in PHP

PHP supports three types of arrays:

  • Indexed Arrays: Arrays with a numeric index.
  • Associative Arrays: Arrays where keys are named instead of being numeric.
  • Multidimensional Arrays: Arrays that contain other arrays as elements.

Indexed Arrays

In indexed arrays, the index is automatically assigned, starting from 0.

    $fruits = array("Apple", "Banana", "Cherry");
    echo $fruits[0]; // Outputs: Apple

Adding Elements to Indexed Arrays

    $fruits[] = "Orange"; // Adds a new element to the array

Associative Arrays

Associative arrays use named keys that you assign to each value, making it easier to retrieve data by name.

    $person = array("name" => "John", "age" => 30, "profession" => "Developer");
    echo $person["name"]; // Outputs: John

Multidimensional Arrays

Multidimensional arrays allow arrays to contain other arrays. This is useful for representing more complex data.

    $matrix = array(
        array(1, 2, 3),
        array(4, 5, 6),
        array(7, 8, 9)
    );
    echo $matrix[1][2]; // Outputs: 6

Common Array Functions

PHP provides many built-in functions for working with arrays. Here are some commonly used ones:

count()

Returns the number of elements in an array.

    echo count($fruits); // Outputs: 4

array_merge()

Merges one or more arrays into one.

    $arr1 = array("a" => "red", "b" => "green");
    $arr2 = array("c" => "blue", "d" => "yellow");
    $result = array_merge($arr1, $arr2);
    print_r($result);
    // Outputs: Array ( [a] => red [b] => green [c] => blue [d] => yellow )

array_push()

Adds one or more elements to the end of an array.

    array_push($fruits, "Grape", "Mango");
    print_r($fruits);
    // Outputs: Array ( [0] => Apple [1] => Banana [2] => Cherry [3] => Orange [4] => Grape [5] => Mango )

array_pop()

Removes and returns the last element of an array.

    $lastFruit = array_pop($fruits);
    echo $lastFruit; // Outputs: Mango

array_key_exists()

Checks if a specified key exists in an array.

    if (array_key_exists("name", $person)) {
        echo "The name key exists!";
    }

Looping Through Arrays

You can loop through arrays using a foreach loop:

    foreach($fruits as $fruit) {
        echo $fruit . " ";
    }
    // Outputs: Apple Banana Cherry Orange

In associative arrays, you can loop through both keys and values:

    foreach($person as $key => $value) {
        echo $key . ": " . $value . "<br>";
    }
    // Outputs: 
    // name: John
    // age: 30
    // profession: Developer
