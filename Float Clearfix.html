Float and Clearfix

What is Floating in CSS?

The float property is used to position elements horizontally within their containing element. Floating elements are taken out of the normal document flow, allowing other content to wrap around them.

What is Clearfix?

When using floats, a common issue is that the container may collapse if all its content is floated. The clearfix technique is used to clear floats and ensure that the container expands to fit its floated content.

Code Example


/* Floating elements */
.float-box {
    float: left;
    width: 45%;
    margin: 10px;
    padding: 10px;
    color: #fff;
    text-align: center;
    border: 2px solid #333;
}

/* Clearfix */
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

Summary

Floating elements allows for horizontal positioning and wrapping of content around them. The clearfix technique is essential for ensuring that parent containers expand to contain their floated children, preventing layout issues.
