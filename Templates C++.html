Templates in C++

Introduction

Templates in C++ are a powerful feature that allows you to write generic and reusable code. Templates enable you to define functions or classes with placeholder types, which are then substituted with specific types when the code is compiled. This helps in creating flexible and type-safe code that can handle different data types without duplication.

Function Templates

A function template allows you to create functions that operate with any data type. Hereâ€™s how you define a function template:

template <typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}

In this example, template <typename T> declares a template parameter T. The function max can now be used with different data types, such as int, float, or double.

Example: Function Template

Here is a complete example showing the usage of a function template:

#include <iostream>

// Define the function template
template <typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    std::cout << "Max of 3 and 7: " << max(3, 7) << std::endl;
    std::cout << "Max of 3.5 and 2.5: " << max(3.5, 2.5) << std::endl;
    
    return 0;
}

Class Templates

Class templates allow you to create classes that can handle different data types. The syntax for a class template is similar to that of a function template:

template <typename T>
class Stack {
private:
    std::vector<T> elements;
public:
    void push(const T& element) {
        elements.push_back(element);
    }
    
    T pop() {
        T elem = elements.back();
        elements.pop_back();
        return elem;
    }
};

In this example, the Stack class is defined with a template parameter T, allowing it to store elements of any type.

Example: Class Template

Here is an example demonstrating the use of a class template:

#include <iostream>
#include <vector>

// Define the class template
template <typename T>
class Stack {
private:
    std::vector<T> elements;
public:
    void push(const T& element) {
        elements.push_back(element);
    }
    
    T pop() {
        T elem = elements.back();
        elements.pop_back();
        return elem;
    }
};

int main() {
    Stack<int> intStack;
    intStack.push(1);
    intStack.push(2);
    std::cout << "Popped from intStack: " << intStack.pop() << std::endl;
    
    Stack<std::string> strStack;
    strStack.push("Hello");
    strStack.push("World");
    std::cout << "Popped from strStack: " << strStack.pop() << std::endl;
    
    return 0;
}

Template Specialization

Template specialization allows you to define a specific implementation of a template for a particular data type. This is useful when you need to handle a type differently than others:

template <typename T>
class Printer {
public:
    void print(const T& value) {
        std::cout << "General print: " << value << std::endl;
    }
};

// Specialization for char*
template <>
class Printer<char*> {
public:
    void print(const char* value) {
        std::cout << "Specialized print for char*: " << value << std::endl;
    }
};

In this example, the Printer class has a general template and a specialized version for char*.

Common Use Cases

  • Generic Containers: Use templates to create generic data structures like lists, stacks, and queues.
  • Algorithm Implementation: Implement algorithms that work with various data types without code duplication.
  • Type-Safe Operations: Ensure type safety while performing operations on different data types.
