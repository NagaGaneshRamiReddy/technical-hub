Graph Representation

Introduction

Graphs can be represented in various ways to efficiently support different operations and algorithms. The choice of representation depends on the specific requirements of the problem being solved. Here, we cover three common methods: adjacency matrices, adjacency lists, and edge lists.

1. Adjacency Matrix

An adjacency matrix is a 2D array where the cell at row i and column j contains a value that indicates whether there is an edge between vertex i and vertex j. This representation is particularly useful for dense graphs.


#include <iostream>
#include < adjMatrix; // Adjacency matrix

public:
    Graph(int V) : V(V), adjMatrix(V, std::vector(V, 0)) {}

    void addEdge(int u, int v) {
        adjMatrix[u][v] = 1;
        adjMatrix[v][u] = 1; // For undirected graph
    }

    void printGraph() {
        for (int i = 0; i < V; ++i) {
            for (int j = 0; j < V; ++j) {
                std::cout << adjMatrix[i][j] << " ";
            }
            std::cout << std::endl;
        }
    }
};

int main() {
    Graph g(5); // Create a graph with 5 vertices
    g.addEdge(0, 1);
    g.addEdge(0, 4);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    g.printGraph();
    return 0;
}

Adjacency Matrix in Python


class Graph:
    def __init__(self, vertices):
        self.V = vertices
        self.adjMatrix = [[0] * vertices for _ in range(vertices)]

    def add_edge(self, u, v):
        self.adjMatrix[u][v] = 1
        self.adjMatrix[v][u] = 1  # For undirected graph

    def print_graph(self):
        for row in self.adjMatrix:
            print(" ".join(map(str, row)))

# Example usage
g = Graph(5)  # Create a graph with 5 vertices
g.add_edge(0, 1)

g.add_edge(1, 2)
g.add_edge(1, 3)
g.add_edge(2, 3)
g.add_edge(3, 4)
g.print_graph()

2. Adjacency List

An adjacency list represents a graph as an array of lists. Each element in the array represents a vertex, and each list contains the vertices that are adjacent to that vertex. This representation is more memory-efficient for sparse graphs.


#include <iostream>
#include <

class Graph {
    int V; // Number of vertices
    std::vector> adjList; // Adjacency list

public:
    Graph(int V) : V(V), adjList(V) {}

    void addEdge(int u, int v) {
        adjList[u].push_back(v);
        adjList[v].push_back(u); // For undirected graph
    }

    void printGraph() {
        for (int i = 0; i < V; ++i) {
            std::cout << "Vertex " << i << ":";
            for (int j : adjList[i]) {
                std::cout << " " << j;
            }
            std::cout << std::endl;
        }
    }
};

int main() {
    Graph g(5); // Create a graph with 5 vertices
    g.addEdge(0, 1);
    g.addEdge(0, 4);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    g.printGraph();
    return 0;
}

Adjacency List in Python


class Graph:
    def __init__(self, vertices):
        self.V = vertices
        self.adjList = [[] for _ in range(vertices)]

    def add_edge(self, u, v):
        self.adjList[u].append(v)
        self.adjList[v].append(u)  # For undirected graph

    def print_graph(self):
        for i in range(self.V):
            print(f"Vertex {i}:", end="")
            for j in self.adjList[i]:
                print(f" {j}", end="")
            print()

3. Edge List

An edge list is a list of all the edges in the graph. Each edge is represented as a pair (or tuple) of vertices. This representation is simple and useful for certain types of algorithms, particularly when dealing with sparse graphs or when constructing a graph from edge data.


#include <iostream>
#include <

class Graph {
    std::vector> edgeList; // List of edges

public:
    void addEdge(int u, int v) {
        edgeList.push_back(std::make_pair(u, v));
    }

    void printGraph() {
        for (const auto& edge : edgeList) {
            std::cout << "(" << edge.first << ", " << edge.second << ")" << std::endl;
        }
    }
};

int main() {
    Graph g;
    g.addEdge(0, 1);
    g.addEdge(0, 4);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    g.printGraph();
    return 0;
}

Edge List in Python


class Graph:
    def __init__(self):
        self.edgeList = []

    def add_edge(self, u, v):
        self.edgeList.append((u, v))

    def print_graph(self):
        for edge in self.edgeList:
            print(f"({edge[0]}, {edge[1]})")

# Example usage
g = Graph()
g.add_edge(0, 1)

Summary

Graph representation is a fundamental concept in data structures, allowing us to model relationships and interactions between entities. The choice of representation—adjacency matrix, adjacency list, or edge list—depends on the nature of the graph and the operations required. Each representation has its strengths and weaknesses, making it essential to select the appropriate one based on the specific needs of the application.
