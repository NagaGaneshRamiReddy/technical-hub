Breadth-First Search (BFS)

Introduction

Breadth-First Search (BFS) is a fundamental graph traversal algorithm that explores all the vertices of a graph or nodes of a tree in breadthward motion. BFS starts from a source node and explores all its neighbors at the present depth before moving on to nodes at the next depth level.

How BFS Works

BFS uses a queue to keep track of nodes to be explored. It starts at a given node, explores all its neighboring nodes, and then moves on to nodes at the next level. BFS is useful for finding the shortest path in unweighted graphs and for solving problems that require level-order traversal.

BFS Implementation

BFS in C++


#include <iostream>
#include <vector>
#include <queue>

class Graph {
    int V; // Number of vertices
    std::vector> adjList; // Adjacency list

public:
    Graph(int V) : V(V), adjList(V) {}

    void addEdge(int u, int v) {
        adjList[u].push_back(v);
        adjList[v].push_back(u); // For undirected graph
    }

    void BFS(int start) {
        std::vector visited(V, false);
        std::queue queue;
        visited[start] = true;
        queue.push(start);

        while (!queue.empty()) {
            int vertex = queue.front();
            queue.pop();
            std::cout << vertex << " ";

            for (int neighbor : adjList[vertex]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.push(neighbor);
                }
            }
        }
    }
};

int main() {
    Graph g(5); // Create a graph with 5 vertices
    g.addEdge(0, 1);
    g.addEdge(0, 4);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    std::cout << "BFS starting from vertex 0:" << std::endl;
    g.BFS(0);
    std::cout << std::endl;
    return 0;
}

BFS in Python


from collections import deque

class Graph:
    def __init__(self, vertices):
        self.V = vertices
        self.adjList = [[] for _ in range(vertices)]

    def add_edge(self, u, v):
        self.adjList[u].append(v)
        self.adjList[v].append(u)  # For undirected graph

    def BFS(self, start):
        visited = [False] * self.V
        queue = deque([start])
        visited[start] = True

        while queue:
            vertex = queue.popleft()
            print(vertex, end=" ")

            for neighbor in self.adjList[vertex]:
                if not visited[neighbor]:
                    visited[neighbor] = True
                    queue.append(neighbor)

# Example usage
g = Graph(5)  # Create a graph with 5 vertices
g.add_edge(0, 1)
g.add_edge(0, 4)
g.add_edge(1, 2)
g.add_edge(1, 3)
g.add_edge(2, 3)
g.add_edge(3, 4)
print("BFS starting from vertex 0:")
g.BFS(0)
print()

Applications of BFS

  • Shortest Path: BFS can be used to find the shortest path in an unweighted graph.
  • Level Order Traversal: BFS is used for level-order traversal of trees.
  • Connected Components: BFS helps in finding all connected components in a graph.
  • Cycle Detection: BFS can be used to detect cycles in an undirected graph.
  • Web Crawlers: BFS is used in web crawlers to explore and index web pages.

Summary

Breadth-First Search (BFS) is a crucial graph traversal algorithm that explores nodes level by level, using a queue to keep track of nodes to be explored. It is especially useful for finding the shortest path in unweighted graphs, level-order traversal in trees, and various other applications such as cycle detection and connected component finding. Understanding BFS and its applications is key for solving complex problems involving graphs and networks.
