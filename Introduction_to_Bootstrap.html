Introduction to Bootstrap


What is Bootstrap?

Bootstrap is a popular front-end framework for developing responsive and mobile-first websites. It provides a collection of CSS and JavaScript components that help streamline the development process.

Key Features of Bootstrap

  • Responsive Design: Bootstrap's grid system allows for fluid layouts that adapt to different screen sizes.
  • Predefined Components: It comes with a variety of components such as navigation bars, buttons, forms, and modals, which can be easily customized.
  • JavaScript Plugins: Bootstrap includes built-in JavaScript plugins that enhance functionality, such as carousels, dropdowns, and tooltips.
  • Customizable: Bootstrap can be customized using Sass variables to fit the design requirements of your project.

Getting Started with Bootstrap

To use Bootstrap in your project, you can either include it via CDN or download it to host locally.

Using CDN

Include the following links in the <head> section of your HTML document:

<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">
<script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.2/dist/umd/popper.min.js"></script>
<script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

Basic Example

Here’s a simple example of a Bootstrap alert component:

<div class="alert alert-primary" role="alert">
  This is a primary alert—check it out!
</div>

Expected Output:

This is a primary alert—check it out!
