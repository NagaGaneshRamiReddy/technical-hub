String Methods in C++

Introduction

The std::string class in C++ provides a variety of methods for manipulating and working with strings. These methods cover operations such as finding substrings, modifying content, and querying properties of strings. Understanding these methods allows you to efficiently handle and manipulate text data in your C++ programs.

Common String Methods

Length and Size

Methods to determine the length or size of a string:

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";

    // Length of the string
    std::cout << "Length: " << str.length() << std::endl;

    // Size of the string (same as length)
    std::cout << "Size: " << str.size() << std::endl;

    return 0;
}

Methods:

  • length() - Returns the number of characters in the string.
  • size() - Same as length().

Accessing Characters

Methods to access and modify individual characters in a string:

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";

    // Accessing a character
    char ch = str[7];  // Indexing starts from 0
    std::cout << "Character at index 7: " << ch << std::endl;

    // Modify a character
    str[7] = 'W';
    std::cout << "Modified string: " << str << std::endl;

    return 0;
}

Methods:

  • operator[] - Accesses a character at a specific index.
  • at() - Accesses a character with bounds checking.

Substrings

Methods to extract substrings from a string:

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";

    // Extract a substring
    std::string sub = str.substr(7, 5);  // Starts at index 7, length 5
    std::cout << "Substring: " << sub << std::endl;

    return 0;
}

Methods:

  • substr(pos, len) - Extracts a substring starting from pos with length len.

Find and Replace

Methods for finding and replacing substrings:

#include <iostream>
#include <string>

int main() {
    std::string str = "The quick brown fox";

    // Find a substring
    std::size_t found = str.find("brown");
    if (found != std::string::npos) {
        std::cout << "'brown' found at index: " << found << std::endl;
    }

    // Replace a substring
    str.replace(found, 5, "red");
    std::cout << "Modified string: " << str << std::endl;

    return 0;
}

Methods:

  • find(substring) - Finds the index of the first occurrence of substring.
  • replace(pos, len, new_substring) - Replaces part of the string starting from pos with length len with new_substring.

Concatenation

Methods for combining strings:

#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello, ";
    std::string str2 = "World!";

    // Concatenate strings
    std::string result = str1 + str2;
    std::cout << "Concatenated string: " << result << std::endl;

    return 0;
}

Methods:

  • operator+ - Concatenates two strings.
  • append() - Appends a string to another string.

Comparison

Methods for comparing strings:

#include <iostream>
#include <string>

int main() {
    std::string str1 = "apple";
    std::string str2 = "banana";

    // Compare strings
    if (str1 == str2) {
        std::cout << "Strings are equal." << std::endl;
    } else if (str1 < str2) {
        std::cout << "str1 is less than str2." << std::endl;
    } else {
        std::cout << "str1 is greater than str2." << std::endl;
    }

    return 0;
}

Methods:

  • operator== - Compares two strings for equality.
  • operator< - Compares strings lexicographically.
  • compare() - Compares the string with another string or substring.

Summary

The std::string class in C++ offers a rich set of methods for managing and manipulating strings. These methods include operations for finding, replacing, accessing characters, and more. By leveraging these methods, you can efficiently handle text data in your C++ programs.
