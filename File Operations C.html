File Operations in C

Introduction to File Operations

File operations in C programming involve handling files to perform various tasks such as reading data from a file or writing data to a file. The C standard library provides several functions to facilitate these operations. All file handling functions are defined in the stdio.h header file.

Opening a File

To work with files, you first need to open them using the fopen function. This function returns a file pointer that is used for subsequent operations on the file:

FILE *fopen(const char *filename, const char *mode);

The mode argument specifies the file access mode, such as:

  • "r": Open a file for reading. The file must exist.
  • "w": Open a file for writing. If the file exists, its contents are truncated. If it does not exist, a new file is created.
  • "a": Open a file for appending. Data is written at the end of the file. If the file does not exist, it is created.
  • "r+": Open a file for both reading and writing. The file must exist.
  • "w+": Open a file for both reading and writing. If the file exists, its contents are truncated. If it does not exist, a new file is created.
  • "a+": Open a file for both reading and appending. Data is written at the end of the file. If the file does not exist, it is created.

Example: Opening a File

Here's an example of opening a file in read mode:

#include <stdio.h>

int main() {
    FILE *filePointer;

    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // File operations here

    fclose(filePointer);
    return 0;
}

This example opens a file named example.txt for reading. If the file cannot be opened, it prints an error message.

Reading from a File

To read data from a file, you can use the following functions:

  • fgetc: Reads a single character from the file.
  • fgets: Reads a string of characters from the file.
  • fread: Reads a block of data from the file.

Example: Reading from a File

The following example demonstrates reading characters from a file:

#include <stdio.h>

int main() {
    FILE *filePointer;
    char ch;

    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    while ((ch = fgetc(filePointer)) != EOF) {
        printf("%c", ch);
    }

    fclose(filePointer);
    return 0;
}

This program reads characters from example.txt one by one and prints them until the end-of-file (EOF) is reached.

Writing to a File

To write data to a file, use the following functions:

  • fputc: Writes a single character to the file.
  • fputs: Writes a string of characters to the file.
  • fwrite: Writes a block of data to the file.

Example: Writing to a File

The following example writes a string to a file:

#include <stdio.h>

int main() {
    FILE *filePointer;

    filePointer = fopen("output.txt", "w");

    if (filePointer == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    fprintf(filePointer, "Hello, World!\n");

    fclose(filePointer);
    return 0;
}

This program opens (or creates) a file named output.txt in write mode and writes "Hello, World!" to it.

Closing a File

After performing file operations, it’s important to close the file using the fclose function:

int fclose(FILE *stream);

This function closes the file associated with the file pointer stream and returns 0 if successful, or EOF if an error occurs.

Example: Closing a File

In the previous examples, the fclose function is used to close the file after performing operations.

Handling File Errors

It's essential to handle errors that may occur during file operations. Common functions for error handling include:

  • feof: Checks if the end-of-file indicator is set.
  • ferror: Checks for errors during file operations.
  • clearerr: Clears the end-of-file and error indicators.

Example: Handling Errors

The following example demonstrates basic error checking while opening a file:

#include <stdio.h>

int main() {
    FILE *filePointer;
    filePointer = fopen("example.txt", "r");

    if (filePointer == NULL) {
        perror("Error opening file");
        return 1;
    }

    // Perform file operations here

    if (ferror(filePointer)) {
        printf("Error reading file.\n");
    }

    fclose(filePointer);
    return 0;
}

This program uses perror to print a descriptive error message if the file cannot be opened and checks for errors during file operations.

Summary

File operations in C are crucial for managing file input and output. Understanding how to open, read, write, and close files, as well as handle file errors, is fundamental for effective file management in your C programs.
