Floyd-Warshall Algorithm

Introduction

The Floyd-Warshall Algorithm is an efficient method for finding shortest paths in a weighted graph with positive or negative edge weights. Unlike Dijkstra's Algorithm, which finds shortest paths from a single source, Floyd-Warshall finds shortest paths between all pairs of nodes.

How It Works

  1. Initialize a distance matrix where dist[i][j] is set to the weight of the edge from node i to node j, or infinity if there is no direct edge.
  2. Iteratively update the matrix by considering whether a path through an intermediate node k provides a shorter path between two nodes i and j than the direct edge.
  3. The matrix will contain the shortest paths between all pairs of nodes after the iterations are complete.

Implementation in Java


import java.util.Arrays;

public class FloydWarshall {
    private static final int INF = 1000000000; // A large number to represent infinity

    public static void floydWarshall(int[][] dist) {
        int V = dist.length;
        for (int k = 0; k < V; ++k) {
            for (int i = 0; i < V; ++i) {
                for (int j = 0; j < V; ++j) {
                    if (dist[i][k] < INF && dist[k][j] < INF) {
                        dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                    }
                }
            }
        }
    }

    public static void printSolution(int[][] dist) {
        int V = dist.length;
        for (int i = 0; i < V; ++i) {
            for (int j = 0; j < V; ++j) {
                if (dist[i][j] == INF) {
                    System.out.print("INF ");
                } else {
                    System.out.print(dist[i][j] + " ");
                }
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        int V = 4;
        int[][] dist = {
            {0, 3, 5, INF},
            {INF, 0, 1, 8},
            {INF, INF, 0, 2},
            {INF, INF, INF, 0}
        };

        floydWarshall(dist);
        System.out.println("Shortest distances between every pair of vertices:");
        printSolution(dist);
    }
}

Summary

The Floyd-Warshall Algorithm is a fundamental technique for solving the all-pairs shortest path problem in weighted graphs. It is particularly valuable for its simplicity and applicability to graphs with negative edge weights. Despite its higher time complexity compared to other algorithms like Dijkstra's (which handles single-source shortest paths), Floyd-Warshall's ability to handle all-pairs shortest paths makes it a powerful tool in various applications, such as network routing, geographical mapping, and any domain requiring comprehensive path analysis. Understanding and implementing this algorithm can significantly enhance one's problem-solving toolkit in computer science and operations research.
