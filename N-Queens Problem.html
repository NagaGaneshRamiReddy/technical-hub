N-Queens problem

Introduction

The N-Queens problem is a classic example of constraint satisfaction and combinatorial optimization. The objective is to place N queens on an N x N chessboard such that no two queens threaten each other. This means no two queens can be on the same row, column, or diagonal.

Problem Description

Given an integer N, the task is to find all possible ways to place N queens on an N x N chessboard so that no two queens are in the same row, column, or diagonal. This problem can be solved using backtracking, a technique for incrementally building solutions and discarding those that do not meet the criteria.

Backtracking Approach

The backtracking approach for solving the N-Queens problem involves placing queens one by one in different columns and rows. After placing a queen, the algorithm checks if the current placement is safe. If it is safe, the algorithm proceeds to place the next queen. If placing a queen leads to a solution, it is recorded; otherwise, the algorithm backtracks to try another placement.

Python Implementation

Here is a Python implementation of the N-Queens problem using backtracking:


def print_board(board):
    for row in board:
        print(' '.join('Q' if x else '.' for x in row))
    print()

def is_safe(board, row, col):
    # Check this column on the upper side
    for i in range(row):
        if board[i][col]:
            return False

    # Check upper left diagonal
    for i, j in zip(range(row, -1, -1), range(col, -1, -1)):
        if board[i][j]:
            return False

    # Check upper right diagonal
    for i, j in zip(range(row, -1, -1), range(col, len(board))):
        if board[i][j]:
            return False

    return True

def solve_n_queens_util(board, row):
    if row >= len(board):
        print_board(board)
        return True

    res = False
    for i in range(len(board)):
        if is_safe(board, row, i):
            board[row][i] = True
            res = solve_n_queens_util(board, row + 1) or res
            board[row][i] = False

    return res

def solve_n_queens(n):
    board = [[False for _ in range(n)] for _ in range(n)]
    solve_n_queens_util(board, 0)

# Example usage
n = 4
solve_n_queens(n)

Q . . .

Each 'Q' represents a queen, and each '.' represents an empty space on the board. This configuration ensures no two queens are in the same row, column, or diagonal.

Applications

The N-Queens problem is a fundamental problem in computer science and artificial intelligence. It is often used to illustrate the use of backtracking and constraint satisfaction techniques. Applications include:

  • Puzzle Solving: Demonstrates problem-solving techniques and algorithms.
  • Constraint Satisfaction Problems: Serves as a benchmark for testing algorithms designed to handle constraints.
  • Algorithm Design: Provides insights into algorithm design and optimization techniques.

Summary

The N-Queens problem is a classic example of a constraint satisfaction problem that can be efficiently solved using backtracking. By placing queens incrementally and checking for conflicts, backtracking helps in finding all possible solutions where N queens are placed on an N x N chessboard without threatening each other. This problem serves as a useful illustration of constraint satisfaction techniques and backtracking algorithms in computer science.
