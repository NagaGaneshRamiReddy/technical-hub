Pointer Arithmetic in C++

Introduction to Pointer Arithmetic

Pointer arithmetic involves performing operations on pointers to navigate through arrays or memory blocks. This is a powerful feature in C++ that allows for efficient data manipulation and access.

Basic Pointer Arithmetic Operations

Pointer arithmetic includes operations such as addition, subtraction, and comparison. These operations are performed relative to the size of the data type to which the pointer points.

Pointer Addition

Adding an integer value to a pointer moves the pointer forward by that number of elements. For example, if you have a pointer to an integer and add 1 to it, the pointer moves to the next integer (which is 4 bytes ahead on most systems).

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;

    std::cout << "Initial pointer address: " << ptr << std::endl;
    std::cout << "Value at pointer: " << *ptr << std::endl;

    // Move pointer to the next integer
    ptr = ptr + 2;

    std::cout << "Pointer address after addition: " << ptr << std::endl;
    std::cout << "Value at new pointer location: " << *ptr << std::endl;

    return 0;
}

In this example:

  • ptr = ptr + 2; moves the pointer to the third element of the array, which is 30.

Pointer Subtraction

Subtracting an integer value from a pointer moves it backward by that number of elements. This can be used to navigate to previous elements in an array or memory block.

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = &arr[4];

    std::cout << "Initial pointer address: " << ptr << std::endl;
    std::cout << "Value at pointer: " << *ptr << std::endl;

    // Move pointer backward by two integers
    ptr = ptr - 2;

    std::cout << "Pointer address after subtraction: " << ptr << std::endl;
    std::cout << "Value at new pointer location: " << *ptr << std::endl;

    return 0;
}

In this example:

  • ptr = ptr - 2; moves the pointer back to the third element of the array, which is 30.

Pointer Comparison

Pointers can be compared using relational operators (<, >, ==, etc.). This is useful for navigating through arrays and checking if pointers are pointing to the same or different elements.

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr1 = &arr[1];
    int *ptr2 = &arr[3];

    if (ptr1 < ptr2) {
        std::cout << "ptr1 points to an earlier element than ptr2." << std::endl;
    } else {
        std::cout << "ptr1 does not point to an earlier element than ptr2." << std::endl;
    }

    return 0;
}

In this example:

  • ptr1 < ptr2 compares two pointers to determine their relative positions in the array.

Pointer Arithmetic and Arrays

Pointers and arrays are closely related in C++. An array name is essentially a pointer to the first element of the array. Pointer arithmetic can be used to traverse the elements of an array efficiently.

#include <iostream>

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int *ptr = arr;

    for (int i = 0; i < 5; ++i) {
        std::cout << "Element " << i << ": " << *(ptr + i) << std::endl;
    }

    return 0;
}

In this example:

  • *(ptr + i) is used to access each element of the array using pointer arithmetic.
