Task: Basic Styling with CSS (20 Marks)

In this task, you need to apply basic CSS styling to an HTML page. Your CSS should include the following:

  • Change the background color of the body
  • Style the headings with different colors and fonts
  • Set padding and margin for the elements
  • Create a button with hover effect
