Top Interviews Questions

  1. 1. What is C?

    Answer: C is a general-purpose, procedural programming language developed in the early 1970s by Dennis Ritchie at Bell Labs. It is known for its efficiency and control over system resources, making it suitable for system programming and embedded systems.

    Information: C serves as the foundation for many modern programming languages, and its syntax has influenced languages like C++, Java, and Python.

  2. 2. What are the basic data types in C?

    Answer: The basic data types in C include:

    • int: Represents integers.
    • float: Represents floating-point numbers.
    • double: Represents double-precision floating-point numbers.
    • char: Represents single characters.

    Information: C also provides derived types like arrays, pointers, structures, and unions.

  3. 3. What is a variable in C?

    Answer: A variable in C is a named storage location in memory that holds a value, which can be changed during program execution. Each variable has a type that defines the kind of data it can store.

    Information: Variables must be declared before use, specifying their type and name (e.g., int age;).

  4. 4. What is the difference between int, float, and double?

    Answer:

    • int: Typically represents integer values without a decimal point (e.g., 10).
    • float: Represents single-precision floating-point numbers, which use less memory than doubles but can be less precise.
    • double: Represents double-precision floating-point numbers, providing more precision than float.

    Information: The sizes of these types can vary depending on the system architecture (commonly int is 4 bytes, float is 4 bytes, and double is 8 bytes).
