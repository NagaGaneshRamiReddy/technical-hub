Introduction to CSS Selectors

CSS selectors are patterns used to select the elements you want to style. They allow you to apply specific styles to elements based on their type, class, id, attributes, and other criteria. Hereâ€™s a breakdown of the different types of CSS selectors and how they work.

1. Element Selectors

Element selectors target HTML elements by their tag name. This applies the same style to all elements of that type.


/* Targets all <p> elements */
p {
    color: green;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Element Selector Example</title>
    <style>
        p {
            color: green;
        }
    </style>
</head>
<body>
    <p>This is a paragraph styled with an element selector.</p>
</body>
</html>

2. Class Selectors

Class selectors target HTML elements with a specific class attribute. This allows you to apply the same styles to multiple elements by assigning them the same class.


/* Targets all elements with the class "highlight" */
.highlight {
    background-color: yellow;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Class Selector Example</title>
    <style>
        .highlight {
            background-color: yellow;
        }
    </style>
</head>
<body>
    <p class="highlight">This text has a yellow background due to the class selector.</p>
    <p>This text does not have the background color because it does not have the class "highlight".</p>
</body>
</html>

3. ID Selectors

ID selectors target a unique element with a specific ID attribute. Each ID should be unique within the page, so this selector applies styles to a single element.


/* Targets the element with the ID "unique" */
#unique {
    color: red;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ID Selector Example</title>
    <style>
        #unique {
            color: red;
        }
    </style>
</head>
<body>
    <p id="unique">This text is styled with an ID selector and will be red.</p>
</body>
</html>

4. Attribute Selectors

Attribute selectors target elements based on their attributes or attribute values. This is useful for styling elements based on their attributes without needing to add classes or IDs.


/* Targets all input elements with a type attribute equal to "text" */
input[type="text"] {
    border: 1px solid #ccc;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Attribute Selector Example</title>
    <style>
        input[type="text"] {
            border: 1px solid #ccc;
        }
    </style>
</head>
<body>
    <input type="text" placeholder="This is a text input field">
    <input type="password" placeholder="This is a password input field">
</body>
</html>

5. Descendant Selectors

Descendant selectors target elements that are nested within other elements. This is useful for styling elements based on their hierarchical relationship.


/* Targets all <span> elements inside <div> elements */
div span {
    color: blue;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Descendant Selector Example</title>
    <style>
        div span {
            color: blue;
        }
    </style>
</head>
<body>
    <div>
        <span>This text is blue because it is inside a <div> element.</span>
    </div>
    <span>This text is not blue because it is not inside a <div> element.</span>
</body>
</html>
