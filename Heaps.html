Heap

Introduction

A Heap is a specialized binary tree-based data structure that satisfies the heap property. It is commonly used to implement priority queues and is integral to several algorithms such as heap sort. Heaps are classified into two main types: Max-Heaps and Min-Heaps, each with its own unique properties and use cases.

Heap Explanation:

A heap is a specialized tree-based data structure that satisfies the heap property. Key operations include:

  • Insertion: Adding a new element while maintaining the heap property.
  • Heapify: Re-adjusting the heap to maintain its structure after insertion or deletion.
  • Extraction: Removing the root element (the maximum for max-heaps or minimum for min-heaps) and re-adjusting the heap.

Types of Heaps

  • Max-Heap: In a max-heap, the key value of each node is greater than or equal to the key values of its children. The largest key is at the root. This property makes max-heaps useful for implementing priority queues where the highest priority element is removed first.
  • Min-Heap: In a min-heap, the key value of each node is less than or equal to the key values of its children. The smallest key is at the root. Min-heaps are used when the lowest priority element should be removed first.

Basic Operations

  • Insertion: Adding a new element to the heap while maintaining the heap property. Involves placing the element at the end and then "bubbling up" to restore heap order.
  • Deletion: Removing the root element (which is either the maximum or minimum, depending on the heap type) and then restoring the heap property by "bubbling down" the replacement element.
  • Peek: Retrieving the root element without removing it. In a max-heap, this is the maximum element, while in a min-heap, it is the minimum element.
  • Heapify: Converting an arbitrary array into a heap. This can be done efficiently using the "heapify" process.

Implementation Examples

1. Max-Heap in C++


#include <iostream>
#include <vector>

class MaxHeap {
    std::vector heap;

    void heapify(int i) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;

        if (left < heap.size() && heap[left] > heap[largest]) {
            largest = left;
        }

        if (right < heap.size() && heap[right] > heap[largest]) {
            largest = right;
        }

        if (largest != i) {
            std::swap(heap[i], heap[largest]);
            heapify(largest);
        }
    }

public:
    void insert(int key) {
        heap.push_back(key);
        int i = heap.size() - 1;

        while (i != 0 && heap[(i - 1) / 2] < heap[i]) {
            std::swap(heap[i], heap[(i - 1) / 2]);
            i = (i - 1) / 2;
        }
    }

    int extractMax() {
        if (heap.size() == 0) return -1;
        if (heap.size() == 1) {
            int root = heap[0];
            heap.pop_back();
            return root;
        }

        int root = heap[0];
        heap[0] = heap.back();
        heap.pop_back();
        heapify(0);

        return root;
    }

    void printHeap() {
        for (int val : heap) {
            std::cout << val << " ";
        }
        std::cout << std::endl;
    }
};

int main() {
    MaxHeap mh;
    mh.insert(10);
    mh.insert(20);
    mh.insert(15);
    mh.printHeap();
    std::cout << "Extracted Max: " << mh.extractMax() << std::endl;
    mh.printHeap();
    return 0;
}

2. Min-Heap in Python


import heapq

class MinHeap:
    def __init__(self):
        self.heap = []

    def insert(self, key):
        heapq.heappush(self.heap, key)

    def extract_min(self):
        if not self.heap:
            return None
        return heapq.heappop(self.heap)

    def peek(self):
        if not self.heap:
            return None
        return self.heap[0]

    def print_heap(self):
        print(self.heap)

# Example usage
mh = MinHeap()
mh.insert(10)
mh.insert(20)
mh.insert(15)
mh.print_heap()
print("Extracted Min:", mh.extract_min())
mh.print_heap()

Summary

Heaps are versatile data structures that provide efficient ways to manage a collection of elements based on their priority. Max-Heaps and Min-Heaps are crucial for various applications, including priority queues and efficient sorting algorithms like heap sort. Understanding how to implement and use heaps can greatly enhance your ability to solve complex computational problems.
