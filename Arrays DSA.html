Arrays

Introduction to Arrays

An array is a fundamental data structure in computer science that holds a fixed-size sequence of elements of the same type. Arrays provide efficient access and manipulation of elements through indexing, making them essential for various algorithms and applications.

Array Operations:

  • Add Element: Adds a new element to the end of the array.
  • Remove Element: Removes the last element from the array.
  • Highlight Element: Highlights a specific element to show access.

1. Arrays in C

Arrays in C are declared with a fixed size and can be accessed using indices. Here's a basic example of using arrays in C:


#include <stdio.h>

int main() {
    // Declare an array of integers
    int numbers[5] = {1, 2, 3, 4, 5};

    // Print array elements
    for (int i = 0; i < 5; i++) {
        printf("Element at index %d: %d\\n", i, numbers[i]);
    }

    return 0;
}

Real-World Problem: Suppose you need to store the daily temperatures for a week and print them. You can use an array to store these temperatures and iterate through the array to display them.

2. Arrays in C++

In C++, arrays are similar to C but provide additional features such as standard template libraries (STL). Here’s a basic example:


#include <iostream>

int main() {
    // Declare an array of integers
    int numbers[5] = {1, 2, 3, 4, 5};

    // Print array elements
    for (int i = 0; i < 5; i++) {
        std::cout << "Element at index " << i << ": " << numbers[i] << std::endl;
    }

    return 0;
}

Real-World Problem: Imagine a scenario where you need to store and sort student scores. Arrays in C++ can be used along with STL algorithms to sort the scores efficiently.

3. Arrays in Java

Java arrays are objects that store multiple variables of the same type. Here’s a basic example:


public class Main {
    public static void main(String[] args) {
        // Declare an array of integers
        int[] numbers = {1, 2, 3, 4, 5};

        // Print array elements
        for (int i = 0; i < numbers.length; i++) {
            System.out.println("Element at index " + i + ": " + numbers[i]);
        }
    }
}

Real-World Problem: Consider a situation where you need to manage a list of employee IDs. Arrays in Java can be used to store and access employee IDs efficiently.

4. Arrays in Python

In Python, arrays are implemented using lists, which are dynamic and can hold different types of elements. Here’s a basic example:


# Declare a list of integers
numbers = [1, 2, 3, 4, 5]

# Print list elements
for i in range(len(numbers)):
    print(f"Element at index {i}: {numbers[i]}")

Real-World Problem: Suppose you need to track user scores in a game. You can use Python lists to store and update user scores as needed.
