Bootstrap Color Utilities

Introduction to Bootstrap Colors

Bootstrap provides various utility classes for applying colors to text and backgrounds, making it easy to style your content quickly. These utilities follow a consistent naming convention for colors.

Text Color Utilities

You can change the text color of any element using Bootstrap's text color utility classes:

<p class="text-primary">This text is primary colored.</p>
<p class="text-success">This text is success colored.</p>

Expected Output:

This text is primary colored.

This text is success colored.

Background Color Utilities

You can apply background colors using the following utility classes:

<div class="bg-danger">This div has a danger background color.</div>
<div class="bg-warning">This div has a warning background color.</div>

Expected Output:

This div has a danger background color.
This div has a warning background color.

Color Palette

Bootstrap provides a standard color palette that includes a variety of colors:

<div class="color-box bg-primary"></div>
<div class="color-box bg-secondary"></div>
<div class="color-box bg-success"></div>
<div class="color-box bg-danger"></div>
<div class="color-box bg-warning"></div>
<div class="color-box bg-info"></div>
<div class="color-box bg-light"></div>
<div class="color-box bg-dark"></div>

Expected Output:

Text
