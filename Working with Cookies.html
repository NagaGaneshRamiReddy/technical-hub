Working with Cookies in PHP

Overview

Cookies are small pieces of data stored on the clientâ€™s computer by the web browser while browsing a website. PHP provides an easy way to set, retrieve, and delete cookies.

Setting Cookies

You can set a cookie using the setcookie() function. This function must be called before any output is sent to the browser:

    <?php
    setcookie("username", "JohnDoe", time() + (86400 * 30), "/"); // 86400 = 1 day
    ?>

This sets a cookie named username with the value JohnDoe, which will expire in 30 days.

Accessing Cookies

To access the value of a cookie, use the global $_COOKIE array:

    <?php
    echo $_COOKIE["username"]; // Outputs: JohnDoe
    ?>

Modifying Cookies

To modify a cookie, simply call the setcookie() function again with the same cookie name:

    <?php
    setcookie("username", "JaneDoe", time() + (86400 * 30), "/"); // Update the value
    ?>

Deleting Cookies

To delete a cookie, set its expiration time to a past timestamp:

    <?php
    setcookie("username", "", time() - 3600, "/"); // Delete the cookie
    ?>

Cookie Security

When working with cookies, consider the following security measures:

  • Use the HttpOnly and Secure flags to enhance security.
  • Validate cookie data on the server side.
  • Limit cookie data to essential information only.

Cookies are a powerful way to store user-specific data on the client side. Understanding how to work with cookies in PHP enables you to create more personalized web experiences.
