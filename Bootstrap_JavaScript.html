Bootstrap JavaScript

Introduction to Bootstrap JavaScript

Bootstrap includes several JavaScript components that enhance the interactivity of your web pages. These components require jQuery and Popper.js for functionality.

Including Bootstrap JavaScript

To use Bootstrap's JavaScript components, include the following scripts in your HTML:

<script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.2/dist/umd/popper.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

Expected Output:

The scripts included will enable Bootstrap's JavaScript components.

Example: Bootstrap Modals

Modals are a great way to display content on top of the main page without navigating away.

<button type="button" class="btn btn-primary" data-toggle="modal" data-target="#myModal">
  Launch Modal
</button>

<div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="exampleModalLabel" aria-hidden="true">
  <div class="modal-dialog" role="document">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title" id="exampleModalLabel">Modal title</h5>
        <button type="button" class="close" data-dismiss="modal" aria-label="Close">
          <span aria-hidden="true">×</span>
        </button>
      </div>
      <div class="modal-body">
        This is a simple modal body content.
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-dismiss="modal">Close</button>
      </div>
    </div>
  </div>
</div>

Expected Output:

Modal title
This is a simple modal body content.

Example: Bootstrap Tooltips

Tooltips provide a way to add extra information when users hover over an element.

<button type="button" class="btn btn-secondary" data-toggle="tooltip" title="Tooltip on top">
  Hover over me
</button>

Expected Output:

Initializing JavaScript Components

To enable tooltips and other components, you must initialize them using JavaScript:

<script>
  $(function () {
    $('[data-toggle="tooltip"]').tooltip();
  });
</script>

Expected Output:

The script initializes tooltips on elements with the data-toggle="tooltip" attribute.

Summary

Bootstrap's JavaScript components add interactivity to your web applications, enhancing user experience. By including the required scripts and initializing components, you can create dynamic interfaces with modals, tooltips, and more.
