Line Height and Spacing in CSS

What is Line Height?

Line height is a CSS property that controls the amount of space between lines of text. It is important for improving readability and visual appeal. The line height can be set using a unitless number, pixels (px), ems (em), or percentages (%).

Key Line Height Properties

1. Line Height: Unitless Number

The line height property can be set to a unitless number, which multiplies the element's font size. For example, a line height of 1.5 means the line height will be 1.5 times the font size.


/* Line height of 1.5 times the font size */
.line-height-1-5 {
    line-height: 1.5;
}

This paragraph has a line height of 1.5, which means there is extra space between lines compared to the default line height.

2. Line Height: Pixels

You can also set line height using pixel values. For example, line-height: 24px; will set the line height to 24 pixels.


/* Line height of 24 pixels */
.line-height-2 {
    line-height: 24px;
}

This paragraph has a line height of 24 pixels, providing a fixed amount of space between lines.

3. Line Height: Ems

Using ems for line height scales the line height relative to the font size of the element. For example, line-height: 1.5em; will set the line height to 1.5 times the font size.


/* Line height of 1.5 ems */
.line-height-1 {
    line-height: 1em;
}

This paragraph has a line height of 1 em, which is equal to the font size.

Spacing Properties

In addition to line height, CSS provides properties to control spacing around elements, including margins and padding.

1. Margin

Margin creates space outside the border of an element, pushing it away from other elements.


/* Margin top and bottom of 20 pixels */
.margin-top {
    margin-top: 20px;
}
.margin-bottom {
    margin-bottom: 20px;
}

This paragraph has a margin-top of 20 pixels.

This paragraph has a margin-bottom of 20 pixels.

2. Padding

Padding creates space inside the border of an element, between the border and the content.


/* Padding top and bottom of 20 pixels */
.padding-top {
    padding-top: 20px;
}
.padding-bottom {
    padding-bottom: 20px;
}

This paragraph has padding-top of 20 pixels.

This paragraph has padding-bottom of 20 pixels.
