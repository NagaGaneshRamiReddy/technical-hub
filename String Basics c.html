String Basics in C

What is a String ?

In C programming, a string is a sequence of characters terminated by a null character ('\0'). It is stored as an array of characters, with each character occupying one element in the array.

Declaration and Initialization

Strings in C are declared as arrays of characters. You can initialize a string with a string literal. Here’s how to declare and initialize a string:

char str1[6] = "Hello"; // Array size must be at least length of string + 1 for null terminator
char str2[] = "World";    // Array size is automatically determined

In the example above, str1 is explicitly sized to hold 6 characters, including the null terminator. str2 is automatically sized to fit the string literal.

Accessing String Elements

Each character in a string can be accessed using array indexing. For example:

printf("%c", str1[0]); // Output: H

This accesses the first character of the string str1.

Example Program with Strings

The following program demonstrates how to declare a string, initialize it, and print it:

#include <stdio.h>

int main() {
    char str[] = "Hello, World!";
    printf("%s\n", str); // Output: Hello, World!
    return 0;
}

This program initializes a string with "Hello, World!" and prints it to the console.

String Literals and Null Terminator

A string literal in C is a sequence of characters enclosed in double quotes. The compiler automatically appends a null character ('\0') to the end of the string to mark its termination. For example:

char str[] = "Hello"; // Stored as {'H', 'e', 'l', 'l', 'o', '\0'}

The null terminator is crucial for string functions to determine the end of the string.

String Length

The length of a string is the number of characters before the null terminator. You can determine the length of a string using the strlen function from the C standard library:

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello";
    printf("Length of str: %lu\n", strlen(str)); // Output: Length of str: 5
    return 0;
}

This program uses the strlen function to compute and print the length of the string str.

Summary

Understanding strings is fundamental in C programming. Strings are arrays of characters ending with a null terminator, and they are essential for handling text. Knowing how to declare, initialize, and manipulate strings will help you manage text data efficiently in your C programs.
