CSS Pseudo-Classes

What are Pseudo-Classes?

A pseudo-class is used to define a special state of an element. For example, you can style an element when a user hovers over it or when an element is the first child of its parent.

Commonly Used Pseudo-Classes

1. :hover

The :hover pseudo-class applies styles when the user hovers over an element (e.g., links, buttons).


a:hover {
    color: red;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Hover Pseudo-Class Example</title>
    <style>
        a:hover {
            color: red;
        }
    </style>
</head>
<body>
    <a href="#">Hover over this link to see it turn red.</a>
</body>
</html>

2. :first-child

The :first-child pseudo-class selects the first child element of its parent.


p:first-child {
    color: blue;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>First-Child Pseudo-Class Example</title>
    <style>
        p:first-child {
            color: blue;
        }
    </style>
</head>
<body>
    <div>
        <p>This is the first paragraph and will be blue.</p>
        <p>This is the second paragraph and will not be blue.</p>
    </div>
</body>
</html>

3. :nth-child()

The :nth-child() pseudo-class selects elements based on their position in a group of siblings. You can use values like odd, even, or a number.


li:nth-child(odd) {
    background-color: #f0f0f0;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Nth-Child Pseudo-Class Example</title>
    <style>
        li:nth-child(odd) {
            background-color: #f0f0f0;
        }
    </style>
</head>
<body>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ul>
</body>
</html>

4. :focus

The :focus pseudo-class applies styles to an element when it gains focus (e.g., when a user clicks or taps on a text input).


input:focus {
    border: 2px solid green;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Focus Pseudo-Class Example</title>
    <style>
        input:focus {
            border: 2px solid green;
        }
    </style>
</head>
<body>
    <form>
        <label>Enter your name:</label>
        <input type="text" name="name">
    </form>
</body>
</html>

Other Pseudo-Classes

  • :last-child - Selects the last child of an element.
  • :nth-of-type() - Selects elements based on their type (e.g., only the nth <p> element).
  • :visited - Styles links that have been visited.
  • :active - Styles elements in the active state (e.g., a link being clicked).
  • :disabled - Applies styles to disabled form elements.

Summary

CSS pseudo-classes are powerful tools that allow you to apply styles based on the state or position of an element. Using these classes, you can create more interactive and dynamic designs.
