String Manipulation in Java

Introduction to String Manipulation

String manipulation in Java involves various techniques for modifying and working with strings. Java provides a rich set of methods for operations such as concatenation, substring extraction, and character replacement. Mastering these techniques is essential for effective text processing and manipulation.

Concatenation

Concatenation is the process of joining two or more strings together. In Java, you can use the concat() method or the + operator:

// Using concat() method
String str1 = "Hello";
String str2 = "World";
String result = str1.concat(" ").concat(str2); // "Hello World"

// Using + operator
String result2 = str1 + " " + str2; // "Hello World"

Substring Extraction

Extracting a portion of a string can be done using the substring() method. This method returns a new string that is a substring of the original string:

String str = "Hello, World!";
String substr1 = str.substring(7);   // "World!"
String substr2 = str.substring(0, 5); // "Hello"

Replacing Characters and Substrings

To replace characters or substrings within a string, you can use the replace() method:

String str = "Hello, World!";
String replaced = str.replace("World", "Java"); // "Hello, Java!"

Splitting Strings

The split() method divides a string into an array of substrings based on a specified delimiter:

String str = "one,two,three";
String[] parts = str.split(","); // ["one", "two", "three"]

Trimming Whitespace

To remove leading and trailing whitespace from a string, use the trim() method:

String str = "   Hello, World!   ";
String trimmed = str.trim(); // "Hello, World!"

Converting Case

To convert the case of characters in a string, use the toLowerCase() and toUpperCase() methods:

String str = "Hello, World!";
String lower = str.toLowerCase(); // "hello, world!"
String upper = str.toUpperCase(); // "HELLO, WORLD!"

Example: Comprehensive String Manipulation

Here's an example demonstrating various string manipulation techniques:

public class StringManipulationExample {
    public static void main(String[] args) {
        String str = "   Java Programming, Learn and Grow!   ";

        // Trim whitespace
        str = str.trim();

        // Convert to uppercase
        String upper = str.toUpperCase();

        // Replace "LEARN" with "MASTER"
        String replaced = upper.replace("LEARN", "MASTER");

        // Extract substring
        String substr = replaced.substring(0, 15);

        // Split string
        String[] parts = replaced.split(",");

        // Print results
        System.out.println("Original String: " + str);
        System.out.println("Uppercase: " + upper);
        System.out.println("Replaced: " + replaced);
        System.out.println("Substring: " + substr);
        System.out.println("Parts: ");
        for (String part : parts) {
            System.out.println(part);
        }
    }
}

Summary

String manipulation in Java involves various techniques for modifying and analyzing strings. By using methods such as concatenation, substring extraction, replacement, and splitting, you can effectively handle and process text data. Understanding these operations allows you to perform complex text processing tasks efficiently.
