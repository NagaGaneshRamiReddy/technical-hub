HTML Forms

Understanding HTML Forms

HTML forms are used to collect user input and send it to a server for processing. A form can contain various input elements such as text fields, radio buttons, checkboxes, submit buttons, etc.

Basic Form Example

A simple form example with text input and a submit button:

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <br>
    <input type="submit" value="Submit">
</form>

Form Input Elements

Forms can contain different types of input elements, such as text fields, checkboxes, radio buttons, etc.

1. Text Input

Used to accept single-line text input:

3. Radio Buttons

Form with GET and POST Methods

Forms use different methods to send data: GET sends data as a query string in the URL, while POST sends data in the request body.

Example: Form using GET

<form action="/search" method="get">
    <label for="query">Search:</label>
    <input type="text" id="query" name="query">
    <input type="submit" value="Search">
</form>

Example: Form using POST

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br>

Summary

HTML forms are an essential part of web development, enabling the collection of user input and the submission of data to a server. Forms can include text fields, radio buttons, checkboxes, and more, and can use either the GET or POST method for submitting data.
