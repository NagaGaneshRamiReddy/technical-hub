Typography in CSS

What is Typography in CSS?

Typography in CSS refers to the style and appearance of text on a web page. It involves controlling the font, size, weight, style, line height, and other text-related properties to ensure that text is readable and visually appealing.

Key Typography Properties

1. Font Family

The font-family property specifies the typeface for text. You can set it to a specific font or a generic family like serif or sans-serif.


/* Set font family */
p {
    font-family: 'Arial', sans-serif;
}

2. Font Size

The font-size property controls the size of the text. It can be set in various units such as pixels (px), ems (em), or percentages (%).


/* Set font size */
h1 {
    font-size: 36px;
}

3. Font Weight

The font-weight property adjusts the thickness of the text. Common values are normal, bold, and numerical values from 100 to 900.


/* Set font weight */
strong {
    font-weight: 700;
}

4. Font Style

The font-style property allows you to set the style of the text to normal, italic, or oblique.


/* Set font style */
em {
    font-style: italic;
}

5. Text Transform

The text-transform property controls the capitalization of text. You can set it to uppercase, lowercase, or capitalize.


/* Transform text to uppercase */
.uppercase {
    text-transform: uppercase;
}

6. Letter Spacing

The letter-spacing property adjusts the space between characters in a text. You can set it in units like pixels or ems.


/* Set letter spacing */
h2 {
    letter-spacing: 2px;
}

7. Text Align

The text-align property sets the horizontal alignment of text. Possible values are left, right, center, and justify.


/* Center-align text */
.center-text {
    text-align: center;
}
