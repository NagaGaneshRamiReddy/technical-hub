Responsive Design with Bootstrap


Introduction

Bootstrap is a powerful front-end framework that makes responsive web design easier. With its built-in grid system and responsive utilities, Bootstrap allows developers to create fluid and adaptable layouts that work across various screen sizes and devices.

Bootstrap Grid System

The grid system is one of the core features of Bootstrap, providing a responsive layout structure. It consists of rows and columns that adapt to different screen sizes. Here's a basic example:

<div class="container">
    <div class="row">
        <div class="col-md-4">Column 1</div>
        <div class="col-md-4">Column 2</div>
        <div class="col-md-4">Column 3</div>
    </div>
</div>

Expected Output:

Column 1
Column 2
Column 3

Responsive Utilities

Bootstrap provides responsive utility classes that can show or hide elements based on the screen size. For example, you can use .d-none to hide elements on all screens and .d-md-block to show them on medium screens and larger:

<div class="d-none d-md-block">Visible on medium and larger screens</div>

Expected Output:

Visible on medium and larger screens

Media Queries with Bootstrap

Bootstrap uses media queries to apply different styles at different screen sizes. You can also add your custom styles using media queries:

@media (max-width: 768px) {
    body {
        background-color: #fff;
    }
}
