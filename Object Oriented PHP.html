Object-Oriented PHP

Overview

Object-oriented programming (OOP) is a programming paradigm that uses objects to design applications. PHP supports OOP concepts, allowing developers to create reusable code, organize complex programs, and implement data abstraction.

Creating a Class

A class is a blueprint for creating objects. You define a class using the class keyword:

    class Car {
        public $make;
        public $model;

        public function __construct($make, $model) {
            $this->make = $make;
            $this->model = $model;
        }

        public function getDetails() {
            return "Make: " . $this->make . ", Model: " . $this->model;
        }
    }

Creating an Object

Once a class is defined, you can create objects using the new keyword:

    $car1 = new Car("Toyota", "Camry");
    echo $car1->getDetails(); // Outputs: Make: Toyota, Model: Camry

Access Modifiers

Access modifiers control the visibility of class properties and methods:

  • public: Accessible from anywhere.
  • protected: Accessible only within the class and its subclasses.
  • private: Accessible only within the class itself.

Inheritance

Inheritance allows a class to inherit properties and methods from another class:

    class ElectricCar extends Car {
        public $batteryLife;

        public function __construct($make, $model, $batteryLife) {
            parent::__construct($make, $model);
            $this->batteryLife = $batteryLife;
        }

        public function getBatteryLife() {
            return "Battery Life: " . $this->batteryLife . " hours";
        }
    }

    $ev1 = new ElectricCar("Tesla", "Model S", 24);
    echo $ev1->getDetails(); // Outputs: Make: Tesla, Model: Model S
    echo $ev1->getBatteryLife(); // Outputs: Battery Life: 24 hours

Polymorphism

Polymorphism allows methods to do different things based on the object it is acting upon. In PHP, this is achieved through method overriding:

    class SportsCar extends Car {
        public function getDetails() {
            return "This is a sports car: " . $this->make . " " . $this->model;
        }
    }

    $sportCar = new SportsCar("Ferrari", "488");
    echo $sportCar->getDetails(); // Outputs: This is a sports car: Ferrari 488

Summary

Object-oriented programming in PHP enhances code organization and reusability. By utilizing classes, objects, inheritance, and polymorphism, developers can build more maintainable and scalable applications.
