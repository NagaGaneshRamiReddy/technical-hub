Rust Loops

Understanding Loops in Rust

Loops are essential control flow constructs that allow you to execute a block of code repeatedly. Rust provides several looping constructs, including loop, while, and for loops.

1. The Loop Statement

The loop statement creates an infinite loop that will continue executing until it is explicitly broken using the break statement:

let mut count = 0;

loop {
    count += 1;
    println!("Count is: {}", count);
    if count >= 5 {
        break;
    }
}

In this example, the loop increments count until it reaches 5, at which point the loop is terminated with break.

2. The While Loop

The while loop continues to execute as long as a specified condition is true:

let mut count = 0;

while count < 5 {
    println!("Count is: {}", count);
    count += 1;
}

This loop will print the value of count from 0 to 4. When count reaches 5, the loop condition becomes false, and the loop terminates.

3. The For Loop

The for loop is used to iterate over a range or a collection. It is particularly useful for looping through arrays, vectors, or ranges:

for number in 1..6 {
    println!("Number: {}", number);
}

In this example, the loop will print numbers from 1 to 5. The range 1..6 is inclusive of the start and exclusive of the end.

Iterating Over Collections

You can also use a for loop to iterate over collections such as arrays or vectors:

let numbers = [1, 2, 3, 4, 5];

for &number in &numbers {
    println!("Number: {}", number);
}

This example demonstrates how to iterate over an array, printing each number in the array.

4. Breaking and Continuing Loops

Within loops, you can use break to exit the loop and continue to skip to the next iteration:

for number in 1..10 {
    if number % 2 == 0 {
        continue; // Skip even numbers
    }
    println!("Odd number: {}", number);
}

In this example, only odd numbers from 1 to 9 are printed, as even numbers are skipped using continue.

Summary

Loops in Rust provide powerful mechanisms for repetitive tasks. Whether using the infinite loop, the conditional while, or the iterable for, understanding these constructs will enable you to write more efficient and readable code.
