Salesforce Connect

What is Salesforce Connect ?

Salesforce Connect is a feature that allows you to access and integrate external data sources directly within Salesforce without the need for data replication. It provides real-time access to data from external systems, enabling users to work with external records as if they were native Salesforce objects. This integration facilitates seamless data access and interaction across disparate systems.

Key Features of Salesforce Connect

  • External Data Sources: Connect Salesforce to external systems using standard protocols such as OData (Open Data Protocol) and custom adapters.
  • Real-Time Data Access: Access and view data from external systems in real-time without duplicating it in Salesforce.
  • External Objects: Create external objects in Salesforce that represent data from external systems, allowing users to interact with this data within Salesforce.
  • Custom Adapters: Develop custom adapters to connect with systems that do not support standard protocols like OData.
  • Data Integration: Seamlessly integrate external data with Salesforce features, such as reports, dashboards, and search functionalities.

External Data Sources

  • OData 2.0 and 4.0: Utilize OData protocols to connect with various external data sources that support this standard.
  • Custom Adapters: Develop and use custom adapters to connect to proprietary or non-standard data sources.
  • Data Integration Patterns: Apply integration patterns like read-only, read-write, and hybrid to suit different business requirements.

Real-Time Data Access

  • Live Data Viewing: View and interact with real-time data from external systems directly within Salesforce.
  • Up-to-Date Information: Ensure that users have access to the most current data without manual updates or synchronization.
  • On-Demand Access: Retrieve and display external data on demand, based on user interactions and needs.

External Objects

  • Object Definition: Define external objects that map to the data structures in external systems.
  • Field Mapping: Map fields from external data sources to Salesforce fields, ensuring proper data representation.
  • Data Interaction: Interact with external objects like standard Salesforce objects, including performing searches, filters, and lookups.

Custom Adapters

  • Custom API Development: Create custom APIs to interface with external systems that do not support standard protocols.
  • Adapter Configuration: Configure custom adapters to handle authentication, data mapping, and data retrieval processes.
  • Integration Flexibility: Tailor integrations to meet specific business needs and accommodate unique data structures.

Data Integration

  • Reports and Dashboards: Incorporate external data into Salesforce reports and dashboards for comprehensive analysis and decision-making.
  • Search Functionality: Use Salesforce’s search capabilities to include external data in global search results and queries.
  • Workflow Automation: Automate workflows that involve external data, such as approval processes and notifications.

Example Use Case: Integrating with an ERP System

  • Real-Time Order Data: Use Salesforce Connect to access real-time order data from an ERP system without duplicating the data in Salesforce.
  • Customer Information: View and interact with customer information stored in the ERP system directly from Salesforce, providing a unified view of customer data.
  • Sales and Service Integration: Integrate order and customer data with Salesforce’s sales and service processes, such as creating cases, managing opportunities, and generating reports.
  • Reporting and Analytics: Generate comprehensive reports and dashboards that include both Salesforce and ERP data, offering insights into sales performance, order fulfillment, and customer satisfaction.

Learning Resources

  • Salesforce Trailhead: Explore Trailhead modules on Salesforce Connect to learn about setting up and using external data sources.
  • Salesforce Documentation: Access detailed guides and documentation on configuring external data sources, external objects, and custom adapters.
  • Salesforce Community: Engage with other Salesforce users and experts to discuss best practices, share experiences, and seek advice on Salesforce Connect implementations.

Summary

Salesforce Connect provides a powerful mechanism for integrating and accessing external data sources in real-time within Salesforce. By using external objects, standard protocols like OData, and custom adapters, businesses can seamlessly connect Salesforce with external systems, ensuring up-to-date information and streamlined workflows. Salesforce Connect enhances data accessibility, supports integration with various data sources, and enables comprehensive reporting and analytics, all while avoiding data duplication.
