PHP Loops

Overview

Loops in PHP are used to repeatedly execute a block of code as long as a specified condition is met. Loops are useful for automating repetitive tasks, such as iterating through arrays or performing actions a fixed number of times.

Types of Loops

PHP supports four main types of loops:

  • while loop
  • do-while loop
  • for loop
  • foreach loop

The while Loop

The while loop executes a block of code as long as the specified condition is true.

    $x = 1;
    while ($x <= 5) {
        echo "The number is: $x <br>";
        $x++;
    }
    // Outputs: The number is: 1, The number is: 2, ..., The number is: 5

The do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the code block will be executed at least once before checking the condition.

    $x = 1;
    do {
        echo "The number is: $x <br>";
        $x++;
    } while ($x <= 5);
    // Outputs: The number is: 1, The number is: 2, ..., The number is: 5

The for Loop

The for loop is used when you know how many times you want to execute a statement or a block of statements. It is a more compact form of the while loop, combining initialization, condition, and increment/decrement in a single line.

    for ($x = 1; $x <= 5; $x++) {
        echo "The number is: $x <br>";
    }
    // Outputs: The number is: 1, The number is: 2, ..., The number is: 5

The foreach Loop

The foreach loop is designed specifically for iterating over arrays. It automatically loops through each value in the array without needing to know the array's size.

    $colors = array("red", "green", "blue", "yellow");
    foreach ($colors as $color) {
        echo "The color is: $color <br>";
    }
    // Outputs: The color is: red, The color is: green, ..., The color is: yellow

Breaking Out of Loops

You can terminate the loop prematurely using the break statement. This is useful when you want to stop the loop based on a condition.

    for ($x = 1; $x <= 10; $x++) {
        if ($x == 5) {
            break;
        }
        echo "The number is: $x <br>";
    }
    // Outputs: The number is: 1, The number is: 2, ..., The number is: 4

Continuing Loops

The continue statement can be used to skip the current iteration of the loop and continue with the next iteration.

    for ($x = 1; $x <= 5; $x++) {
        if ($x == 3) {
            continue;
        }
        echo "The number is: $x <br>";
    }
    // Outputs: The number is: 1, The number is: 2, The number is: 4, The number is: 5
