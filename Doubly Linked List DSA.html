Doubly Linked List

What is a Doubly Linked List ?

A doubly linked list is a type of linked list where each node contains a data value and two references: one to the next node and another to the previous node. This allows traversal of the list in both directions: forward and backward.

Doubly Linked List Explanation:

A doubly linked list is a sequence of nodes where each node points to both the next and previous nodes. In this animation:

  • Node 1 points to Node 2 and has a backward pointer to NULL
  • Node 2 points to Node 3 and has a backward pointer to Node 1
  • Node 3 points to Node 4 and has a backward pointer to Node 2
  • Node 4 points to NULL and has a backward pointer to Node 3

Basic Operations on Doubly Linked Lists

  • Insertion: Adding a new node at the beginning, end, or a specific position in the list.
  • Deletion: Removing a node from the beginning, end, or a specific position in the list.
  • Traversal: Visiting each node in both forward and backward directions to perform operations or retrieve values.
  • Searching: Finding a node with a specific value.
  • Displaying: Printing the contents of the list.

1. Implementing a Doubly Linked List

Description: Implement a doubly linked list with basic operations such as insertion, deletion, traversal, and display.

C


#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node* next;
    struct Node* prev;
};

void insertAtBeginning(struct Node** head, int value) {
    struct Node* new_node = (struct Node*) malloc(sizeof(struct Node));
    new_node->data = value;
    new_node->next = *head;
    new_node->prev = NULL;

    if (*head != NULL) {
        (*head)->prev = new_node;
    }
    *head = new_node;
}

void insertAtEnd(struct Node** head, int value) {
    struct Node* new_node = (struct Node*) malloc(sizeof(struct Node));
    struct Node* last = *head;
    new_node->data = value;
    new_node->next = NULL;

    if (*head == NULL) {
        new_node->prev = NULL;
        *head = new_node;
        return;
    }

    while (last->next != NULL) {
        last = last->next;
    }
    last->next = new_node;
    new_node->prev = last;
}

void deleteNode(struct Node** head, int value) {
    struct Node* temp = *head;

    while (temp != NULL && temp->data != value) {
        temp = temp->next;
    }

    if (temp == NULL) return;

    if (temp->prev != NULL) {
        temp->prev->next = temp->next;
    } else {
        *head = temp->next;
    }

    if (temp->next != NULL) {
        temp->next->prev = temp->prev;
    }

    free(temp);
}

void traverseForward(struct Node* head) {
    struct Node* temp = head;
    while (temp != NULL) {
        printf("%d <-> ", temp->data);
        temp = temp->next;
    }
    printf("NULL\n");
}

void traverseBackward(struct Node* head) {
    struct Node* temp = head;
    if (temp == NULL) return;

    while (temp->next != NULL) {
        temp = temp->next;
    }

    while (temp != NULL) {
        printf("%d <-> ", temp->data);
        temp = temp->prev;
    }
    printf("NULL\n");
}

int main() {
    struct Node* head = NULL;
    insertAtBeginning(&head, 10);
    insertAtBeginning(&head, 20);
    insertAtEnd(&head, 30);
    traverseForward(head);
    traverseBackward(head);
    deleteNode(&head, 20);
    traverseForward(head);
    traverseBackward(head);
    return 0;
}

class Node:

        self.data = value
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def insertAtBeginning(self, value):
        new_node = Node(value)
        new_node.next = self.head
        new_node.prev = None

        if self.head is not None:
            self.head.prev = new_node
        self.head = new_node

    def insertAtEnd(self, value):
        new_node = Node(value)
        if self.head is None:
            new_node.prev = None
            self.head = new_node
            return
        last = self.head
        while last.next is not None:
            last = last.next
        last.next = new_node
        new_node.prev = last

    def deleteNode(self, value):
        temp = self.head

        while temp is not None and temp.data != value:
            temp = temp.next

        if temp is None:
            return

        if temp.prev is not None:
            temp.prev.next = temp.next
        else:
            self.head = temp.next

        if temp.next is not None:
            temp.next.prev = temp.prev

    def traverseForward(self):
        temp = self.head
        while temp is not None:
            print(temp.data, end=" <-> ")
            temp = temp.next
        print("NULL")

        temp = self.head

            return

Real-World Applications

  • Navigation Systems: Doubly linked lists are used in navigation systems to allow forward and backward traversal through a series of locations or waypoints.
  • Browser History: Web browsers use doubly linked lists to maintain and navigate through the history of visited pages.
  • Undo Functionality: Applications that support undo and redo operations often use doubly linked lists to manage the sequence of actions.
  • Text Editors: Doubly linked lists are used to implement text buffers, enabling efficient insertion and deletion of text at various positions.

Summary

Doubly linked lists are an important data structure that extends the functionality of singly linked lists by allowing bidirectional traversal. They are useful in various real-world applications that require flexible navigation and efficient manipulation of data. Understanding and implementing doubly linked lists can enhance your programming skills and problem-solving abilities.
