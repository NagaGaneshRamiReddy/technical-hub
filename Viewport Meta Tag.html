Viewport Meta Tag

What is the Viewport Meta Tag?

The <meta name="viewport"> tag is used in HTML to control the layout and scaling of a webpage on different devices, particularly on mobile devices. It ensures that the webpage is rendered correctly and is responsive to various screen sizes.

Syntax

The viewport meta tag is placed in the <head> section of the HTML document. Here is a basic example:


<meta name="viewport" content="width=device-width, initial-scale=1.0">

Attributes

  • width: Specifies the width of the viewport. width=device-width sets the viewport width to the device's width.
  • initial-scale: Defines the initial zoom level when the page is first loaded. initial-scale=1.0 sets the zoom level to 100%.
  • maximum-scale: Limits the maximum zoom level. For example, maximum-scale=1.0 prevents zooming in beyond 100%.
  • minimum-scale: Limits the minimum zoom level. For example, minimum-scale=1.0 prevents zooming out below 100%.
  • user-scalable: Controls whether the user can zoom in or out. user-scalable=no disables zooming.

Code Example


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Viewport Meta Tag Example</title>
    <style>
        .container-15 {
            display: flex;
            flex-wrap: wrap;
            gap: 20px;
        }
        .box {
            background-color: #4CAF50;
            color: white;
            padding: 1rem;
            text-align: center;
            border: 2px solid #333;
            box-sizing: border-box;
            flex: 1 1 calc(33.333% - 20px);
        }

        @media (max-width: 600px) {
            .box {
                flex: 1 1 100%;
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="box">Box 1</div>
        <div class="box">Box 2</div>
        <div class="box">Box 3</div>
    </div>
</body>
</html>

The viewport meta tag is a crucial tool for making web pages responsive. By setting the viewport width and initial scale, you can ensure that your website looks good and functions well on a variety of devices and screen sizes.
