HTML Interview Questions

What is HTML? ›
HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages and web applications.
What are HTML forms? ›
HTML forms are used to collect user input. A form can contain elements like input fields, radio buttons, checkboxes, and submit buttons.
What are the new features of HTML5? ›
HTML5 introduced new elements like <header>, <footer>, <article>, <section>, and media elements like <audio> and <video>.
What is the difference between HTML and XHTML? ›
XHTML is a stricter version of HTML, following XML rules. In XHTML, all tags must be properly closed, and attributes must be quoted and written in lowercase.
What is the <doctype> declaration? ›
The <!DOCTYPE> declaration defines the document type and version of HTML being used. In HTML5, it is simply <!DOCTYPE html>.
What is the purpose of the <meta> tag? ›
The <meta> tag provides metadata about the HTML document. It includes information such as character set, author, description, and viewport settings for responsive design.
What is the difference between <div> and <span> tags? ›
The <div> tag is a block-level element, typically used to group larger chunks of content. The <span> tag is an inline element, often used to style small pieces of content within a larger block.
How do you create a hyperlink in HTML? ›
A hyperlink is created using the <a> tag, with the href attribute specifying the URL. For example: <a href="https://example.com">Visit Example</a>.
What is the purpose of the <head> section? ›
The <head> section contains metadata about the document, links to external resources (like stylesheets), and defines the document's title and character set.
What is semantic HTML? ›
Semantic HTML refers to using HTML5 elements like <header>, <article>, and <section>, which clearly describe their meaning in a way that helps both the developer and search engines understand the content.
What is the <nav> element used for? ›
The <nav> element is used to define a block of navigation links, such as a websiteâ€™s primary navigation menu.
What is the purpose of the <section> element in HTML5? ›
The <section> element is used to define a section of a document, typically with a heading. It groups related content into a specific area.
What is the <aside> element used for? ›
The <aside> element represents content that is tangentially related to the content around it, like a sidebar or call-out box.
How do you include comments in HTML? ›
Comments in HTML are included using <!-- comment text -->. They are not displayed by browsers and are used for documentation purposes.
What is the <em> tag used for? ›
The <em> tag is used to emphasize text, which is typically displayed in italics. It indicates stress or importance of a word or phrase.
What is the <strong> tag used for? ›
The <strong> tag is used to give importance to text, which is typically displayed in bold. It indicates that the content inside is of strong importance.
How do you add images in HTML? ›
Images are added in HTML using the <img> tag, with the src attribute specifying the image source and alt attribute providing alternative text. Example: <img src="image.jpg" alt="Description">.
What is the purpose of the <table> tag? ›
The <table> tag is used to create a table in HTML. It contains rows (<tr>) and cells (<td>), along with optional table headers (<th>).
What is the difference between <ul> and <ol>? ›
The <ul> tag creates an unordered list (bullets), while the <ol> tag creates an ordered list (numbers).
What are HTML attributes? ›
HTML attributes provide additional information about an element, such as href for links, src for images, and id for unique identification.
What is the purpose of the alt attribute in the <img> tag? ›
The alt attribute provides alternative text for an image if it cannot be displayed. It is also important for accessibility and SEO.
What is the <footer> element used for? ›
The <footer> element defines the footer section of a document or section, often containing copyright information, links, and other relevant details.
What is the <header> element used for? ›
The <header> element defines the header section of a document or section, typically containing headings, logos, or navigation.
What is the purpose of the <mark> tag? ›
The <mark> tag highlights text, usually for reference or emphasis, and it is typically displayed with a yellow background.
What is the <blockquote> element used for? ›
The <blockquote> element is used for longer quotes, typically displayed as an indented block of text.
How do you create a drop-down list in HTML? ›
A drop-down list is created using the <select> element. Individual options are defined using the <option> tags. Example: <select><option>Option 1</option></select>.
What is the <iframe> tag used for? ›
The <iframe> tag is used to embed another HTML page within the current page. Example: <iframe src="https://example.com"></iframe>.
What is the <progress> element used for? ›
The <progress> element is used to display the progress of a task, such as a file download or an upload progress bar.
What is the purpose of the <datalist> element? ›
The <datalist> element is used to provide an autocomplete feature for input elements. It defines a list of pre-defined options for a user to choose from.
How do you create a checkbox in HTML? ›
A checkbox is created using the <input> element with the type attribute set to "checkbox". Example: <input type="checkbox">.
What are inline elements in HTML? ›
Inline elements do not start on a new line and only take up as much width as necessary. Examples include <span>, <a>, and <img>.
What are block-level elements in HTML? ›
Block-level elements start on a new line and take up the full width available. Examples include <div>, <p>, and <h1>.
What is the <canvas> element used for in HTML5? ›
The <canvas> element is used to draw graphics, such as charts, games, or animations, via scripting (usually JavaScript).
What is the purpose of the <article> tag in HTML5? ›
The <article> tag is used to represent a self-contained composition in a document, such as a blog post or news article.
How do you embed audio in HTML5? ›
The <audio> element is used to embed audio. Example: <audio controls><source src="audio.mp3" type="audio/mpeg"></audio>.
What is the difference between <b> and <strong>? ›
The <b> tag displays text in bold for stylistic purposes, while the <strong> tag gives text strong importance, typically displayed in bold as well but with semantic meaning.
What is the difference between <i> and <em>? ›
The <i> tag displays text in italics for stylistic purposes, while the <em> tag emphasizes text semantically, often displayed in italics but with importance.
What is the <time> element used for? ›
The <time> element is used to represent a specific time (with optional machine-readable formatting using the datetime attribute).
What is the purpose of the <picture> element in HTML5? ›
The <picture> element allows for different image resources to be used depending on screen size or pixel density. It is often used for responsive image support.
What is a favicon? ›
A favicon is a small icon displayed in the browser tab, next to the website title. It is linked using a <link rel="icon" href="favicon.ico"> tag in the document's <head>.
What is the purpose of the <meta> viewport tag? ›
The <meta> viewport tag is used to control the layout on mobile browsers. Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">.
How do you add a video in HTML5? ›
Videos can be embedded using the <video> element. Example: <video controls><source src="video.mp4" type="video/mp4"></video>.
What is the purpose of the <noscript> tag? ›
The <noscript> tag defines alternative content to be displayed if the browser does not support or has disabled JavaScript.
What are semantic elements in HTML? ›
Semantic elements clearly describe their meaning to both the browser and the developer. Examples include <article>, <footer>, and <section>.
What is the <abbr> tag used for? ›
The <abbr> tag is used to define an abbreviation or acronym, with optional title attribute for full form display. Example: <abbr title="Hypertext Markup Language">HTML</abbr>.
How do you create a radio button in HTML? ›
Radio buttons are created using the <input> element with type="radio". Example: <input type="radio" name="choice">.
What is the <figure> and <figcaption> used for? ›
The <figure> tag is used to encapsulate media content such as images, and the <figcaption> provides a caption for that media.
What is the purpose of the <code> tag? ›
The <code> tag is used to display inline code snippets in a monospace font. It is useful for showing small pieces of computer code.
What is the <pre> element used for? ›
The <pre> element displays preformatted text, preserving both spaces and line breaks.
What is a self-closing tag? ›
A self-closing tag does not need a closing tag. It ends with a />. Examples: <br />, <img />.
What is the <nav> element used for? ›
The <nav> element defines a section of navigation links, such as menus or table of contents.
What is the <aside> element used for? ›
The <aside> element represents a sidebar or other tangentially related content.
How do you define an input field for email in HTML? ›
Use <input type="email"> to create an input field specifically for email addresses.
What is the purpose of the <label> tag? ›
The <label> tag provides a caption or label for an <input> element, improving usability and accessibility.
How do you add a tooltip to an HTML element? ›
Use the title attribute to add a tooltip. Example: <p title="This is a tooltip">Hover over me!</p>.
What is the <summary> element used for? ›
The <summary> element is used inside the <details> tag to provide a summary or label for the details that the user can toggle.
What are custom data attributes in HTML? ›
Custom data attributes allow you to store extra information on HTML elements using the data-* attribute. Example: <div data-user="123">.
What is the <wbr> tag used for? ›
The <wbr> tag is used to define a possible line break opportunity in a long word, which will only be visible when needed (e.g., if the word is too long for the current container).
What is the <base> tag used for in HTML? ›
The <base> tag defines the base URL for all relative URLs in the document. It is typically placed inside the <head> element.
What is the purpose of the <output> element? ›
The <output> element represents the result of a calculation or user action, often used in forms.
What is the difference between <section> and <div>? ›
The <section> element is used for defining sections of a document with thematic grouping, while <div> is a general-purpose container without any specific meaning.
What is the <legend> element used for in forms? ›
The <legend> element is used to provide a caption or title for the <fieldset> element, grouping related form controls together. It helps improve the accessibility of the form.
What is the <optgroup> element used for? ›
The <optgroup> element is used to group related <option> elements in a drop-down list. It helps users better navigate long lists by organizing them into categories.
What is the <fieldset> element used for in forms? ›
The <fieldset> element is used to group related elements within a form. It helps to visually and semantically group inputs and controls, often alongside a <legend> element.
What is the purpose of the <details> element? ›
The <details> element is used to create a collapsible section of content that the user can toggle open or closed. It is often used with the <summary> element.
What does the <progress> element represent? ›
The <progress> element represents the completion progress of a task. It is displayed as a progress bar, showing how much of the task has been completed.
How do you include audio in HTML? ›
Audio is included in HTML using the <audio> element, with the src attribute specifying the audio file. Example: <audio controls src="audio.mp3"></audio>.
How do you embed a video in HTML? ›
A video is embedded using the <video> element with the src attribute. You can also include controls for play, pause, and volume. Example: <video controls src="video.mp4"></video>.
What is the <output> element used for? ›
The <output> element is used to represent the result of a calculation or user action, often in combination with JavaScript for dynamic updates.
What is the <time> element used for? ›
The <time> element is used to represent a specific time or date. It can be used to mark events and improve accessibility for time-related content.
What is the <code> element used for? ›
The <code> element is used to display code snippets or programming instructions. It is typically styled with a monospace font by default.
What is the <kbd> element used for? ›
The <kbd> element represents user input, typically from a keyboard, and is often styled with a monospace font.
What is the <samp> element used for? ›
The <samp> element represents sample output from a program or computing system, often displayed in a monospace font.
What is the <var> element used for? ›
The <var> element is used to define a variable in programming or mathematical expressions, typically displayed in italics by default.
What is the <dfn> element used for? ›
The <dfn> element is used to indicate the defining instance of a term. It is often styled in italics to highlight the term.
What is the <bdi> element used for? ›
The <bdi> (bidirectional isolation) element isolates a section of text that might be formatted in a different direction, making it easier to handle mixed language content.
What is the <wbr> element used for? ›
The <wbr> (word break opportunity) element is used to suggest where a line break could occur in a long word or URL to improve readability.
How do you specify the language of a document in HTML? ›
You specify the language of an HTML document using the lang attribute in the <html> tag. Example: <html lang="en">.
What is the purpose of the <small> element? ›
The <small> element is used to display text in a smaller font size, often for legal disclaimers or secondary information.
What is the difference between <sup> and <sub>? ›
The <sup> element displays text as superscript (above the baseline), while the <sub> element displays text as subscript (below the baseline).
What is the <abbr> element used for? ›
The <abbr> element represents an abbreviation or acronym, and the title attribute can be used to provide the full meaning. Example: <abbr title="Hypertext Markup Language">HTML</abbr>.
What is the purpose of the <cite> element? ›
The <cite> element is used to reference a title of a work, such as a book, article, or other creative work, typically displayed in italics.
What is the <ins> element used for? ›
The <ins> element is used to indicate inserted content, often styled with an underline. It shows that text has been added to the document.
What is the <del> element used for? ›
The <del> element is used to represent deleted content, typically displayed with a strikethrough to show it has been removed.
What is the purpose of the <s> element? ›
The <s> element is used to indicate that the text has been struck through and is no longer accurate or relevant.
What is the <u> element used for? ›
The <u> element is used to underline text, which is often used to indicate misspelled words or for stylistic purposes.
What is the purpose of the <hr> element? ›
The <hr> element represents a thematic break or horizontal rule, often used to separate sections of content visually.
What is the <pre> element used for? ›
The <pre> element is used to display preformatted text. It preserves white spaces, line breaks, and indentation exactly as written in the HTML.
What is the difference between <strong> and <b>? ›
The <strong> element indicates text of strong importance, typically bold by default. The <b> element only makes text bold without adding semantic importance.
What is the difference between <i> and <em>? ›
The <i> element italicizes text for stylistic purposes, while the <em> element emphasizes text, indicating importance, often italicized by default.
What is the <source> element used for in media elements? ›
The <source> element is used to specify multiple media sources for the <video> or <audio> elements. It allows the browser to choose the appropriate format for playback.
What is the <track> element used for? ›
The <track> element is used in the <video> and <audio> elements to specify text tracks like subtitles, captions, or descriptions.
What is the <summary> element used for? ›
The <summary> element is used as a heading for the <details> element. When clicked, it toggles the visibility of the content inside the <details>.
What is the <canvas> element used for? ›
The <canvas> element is used to draw graphics, animations, and other visual content using JavaScript, such as game elements, charts, and interactive graphics.
What is the <noscript> element used for? ›
The <noscript> element provides alternative content to display if the user's browser does not support JavaScript or has JavaScript disabled.
What is the difference between <link> and <a>? ›
The <link> element is used to link external resources like stylesheets, while the <a> element is used to create hyperlinks to navigate between web pages or to external websites.
What is the purpose of the <base> tag? ›
The <base> tag specifies the base URL for all relative URLs in the document, allowing you to set a standard base for all links or resource references.
What is the <picture> element used for? ›
The <picture> element is used to provide multiple versions of an image for responsive design, allowing the browser to choose the appropriate image based on screen size or format.
What is the purpose of the <iframe> element? ›
The <iframe> element is used to embed another HTML document or external content like videos or websites within the current page.
What is the purpose of the <meta> tag in HTML? ›
The <meta> tag provides metadata about the HTML document, such as descriptions, keywords, and author information. It is also used to control the document's character set and viewport settings.
