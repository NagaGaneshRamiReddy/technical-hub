Pointers and arrays in C++

Introduction

Pointers and arrays are fundamental concepts in C++. They are closely related and understanding their relationship is crucial for efficient memory management and manipulation. This section explores how pointers and arrays interact in C++ and provides examples to illustrate their relationship.

Arrays as Pointers

In C++, an array name is essentially a pointer to the first element of the array. This means that array indexing and pointer arithmetic can be used interchangeably to access array elements.

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;  // Equivalent to int *ptr = &arr[0];

    std::cout << "Using array indexing:" << std::endl;
    for (int i = 0; i < 5; ++i) {
        std::cout << "Element " << i << ": " << arr[i] << std::endl;
    }

    std::cout << "Using pointer arithmetic:" << std::endl;
    for (int i = 0; i < 5; ++i) {
        std::cout << "Element " << i << ": " << *(ptr + i) << std::endl;
    }

    return 0;
}

In this example:

  • int *ptr = arr; sets ptr to point to the first element of arr.
  • arr[i] and *(ptr + i) access the same array elements.

Pointer Arithmetic with Arrays

Pointer arithmetic allows you to navigate through array elements efficiently. You can use pointer addition and subtraction to move through the array, as well as pointer comparisons to check positions.

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;

    // Accessing elements using pointer arithmetic
    std::cout << "Pointer arithmetic:" << std::endl;
    std::cout << "First element: " << *ptr << std::endl;
    std::cout << "Second element: " << *(ptr + 1) << std::endl;
    std::cout << "Third element: " << *(ptr + 2) << std::endl;

    // Moving the pointer
    ptr += 2;
    std::cout << "Element after moving pointer by 2 positions: " << *ptr << std::endl;

    return 0;
}

In this example:

  • *(ptr + 1) accesses the second element of the array.
  • ptr += 2; moves the pointer to the third element of the array.

Passing Arrays to Functions

When passing an array to a function, what is actually passed is a pointer to the first element of the array. This allows the function to access and manipulate the array elements.

#include <iostream>

void printArray(int *arr, int size) {
    for (int i = 0; i < size; ++i) {
        std::cout << "Element " << i << ": " << *(arr + i) << std::endl;
    }
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    printArray(arr, 5);

    return 0;
}

In this example:

  • The printArray function takes a pointer to an integer and the size of the array as parameters.
  • *(arr + i) is used to access array elements within the function.

Dynamic Arrays and Pointers

Dynamic arrays are created using pointers and memory allocation functions like new and delete. This allows for arrays whose size is determined at runtime.

#include <iostream>

int main() {
    int size = 5;
    int *arr = new int[size];  // Allocate memory for a dynamic array

    // Initialize and print the array
    for (int i = 0; i < size; ++i) {
        arr[i] = i * 10;
        std::cout << "Element " << i << ": " << arr[i] << std::endl;
    }

    delete[] arr;  // Free the allocated memory
    return 0;
}

In this example:

  • new int[size] dynamically allocates an array of integers.
  • delete[] arr; deallocates the memory used by the dynamic array.
