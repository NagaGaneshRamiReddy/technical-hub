File pointers and file positioning



File pointers and file positioning are essential concepts in C language when working with files. Let's delve into each of these concepts in detail:

1. File Pointers:



In C, a file pointer is a special pointer used to identify and navigate through files. It is represented by the `FILE` data type, which is defined in the `` header file. File pointers are used to perform various operations on files, such as opening, reading, writing, and closing.

Functions for File Pointers:



fopen(): Opens a file and returns a file pointer.

fclose(): Closes a file stream and releases associated resources.

fread(): Reads binary data from a file.

fwrite(): Writes binary data to a file.

fscanf():Reads formatted data from a file.

fprintf(): Writes formatted data to a file.

fgets(): Reads a line of text from a file.

fputs(): Writes a string to a file.

feof(): Checks for the end-of-file indicator.

ferror(): Checks for file error indicators.

2. File Positioning:



File positioning refers to the ability to move the file position indicator within a file. The file position indicator keeps track of the current position within the file and determines where the next read or write operation will occur.

Functions for File Positioning:



fseek(): Sets the file position indicator to a specified position within the file.

ftell() : Returns the current position of the file position indicator within the file.

rewind() : Moves the file position indicator to the beginning of the file.

Example:

#include < stdio.h >

int main() {

FILE* file_ptr = fopen("example.txt", "r");

if (file_ptr == NULL) {

printf("Error opening file\n");

return 1;

}

// Move file position indicator to the end of the file

fseek(file_ptr, 0, SEEK_END);

// Get the current position (end of file)

long file_size = ftell(file_ptr);

printf("File size: %ld bytes\n", file_size);

// Move file position indicator to the beginning of the file

rewind(file_ptr);

// Close the file

fclose(file_ptr);

return 0;

}

In this example:



We open a file "example.txt" for reading.

We use fseek() to move the file position indicator to the end of the file.

We use ftell() to get the current position of the file position indicator (which corresponds to the end of the file), giving us the file size.

We use rewind() to move the file position indicator to the beginning of the file.

Finally, we close the file.

Understanding file pointers and file positioning is essential for effectively working with files in C. These concepts allow you to navigate through files, read and write data, and perform various file operations with precision.
