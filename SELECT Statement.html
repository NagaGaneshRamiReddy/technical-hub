SELECT Statement

SQL SELECT Statement

The SELECT statement is one of the most commonly used commands in SQL. It is used to retrieve data from one or more tables in a database. The result is returned in the form of a result set, which is a table-like structure.

Basic Syntax

    SELECT column1, column2, ...
    FROM table_name
    WHERE condition;
  • column1, column2, ... - The names of the columns you want to retrieve.
  • table_name - The name of the table from which to retrieve data.
  • condition (optional) - Used to filter the results based on specific conditions.

Examples of the SELECT Statement

1. Selecting All Columns

You can use * to select all columns from a table.


    SELECT *
    FROM employees;

This will retrieve all columns and rows from the "employees" table.

2. Selecting Specific Columns

To retrieve specific columns, list the column names after the SELECT keyword.


    SELECT emp_name, department
    FROM employees;

This will retrieve only the emp_name and department columns from the "employees" table.

3. Using the WHERE Clause

The WHERE clause is used to filter records that meet specific conditions.


    SELECT emp_name, emp_id
    FROM employees
    WHERE department = 'HR';

This will retrieve the names and IDs of employees who work in the "HR" department.

4. Using the ORDER BY Clause

The ORDER BY clause is used to sort the result set in either ascending (ASC) or descending (DESC) order.


    SELECT emp_name, emp_id
    FROM employees
    ORDER BY emp_name ASC;

This will retrieve employee names and IDs sorted alphabetically by name.

5. Using the DISTINCT Keyword

The DISTINCT keyword is used to return only unique values from a column.


    SELECT DISTINCT department
    FROM employees;

This will return a list of unique department names from the "employees" table.

6. Using Aggregate Functions

You can use SQL's aggregate functions, such as COUNT, SUM, AVG, MAX, and MIN, with the SELECT statement to perform calculations on data.


    SELECT COUNT(*)
    FROM employees;

This will return the total number of rows in the "employees" table.

Combining Multiple Conditions with AND/OR

Use the AND and OR operators to combine multiple conditions in the WHERE clause.


    SELECT emp_name, emp_id
    FROM employees
    WHERE department = 'HR' AND emp_id > 100;

This will return the names and IDs of employees in the "HR" department whose employee ID is greater than 100.

Summary

The SELECT statement is a fundamental part of SQL, allowing you to retrieve and filter data from tables. With the WHERE, ORDER BY, and DISTINCT clauses, along with aggregate functions and logical operators, you can craft powerful queries to interact with your database and extract meaningful insights.
