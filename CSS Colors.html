CSS Colors

Introduction to CSS Colors

Colors in CSS can be specified using various methods, including color names, hexadecimal codes, RGB values, RGBA values, HSL values, and HSLA values. Each method offers different levels of control and flexibility when styling web pages.

Color Names

CSS supports a set of named colors that are predefined and recognized by all browsers. Here are a few examples:

Red

Green

Blue

Hexadecimal Colors

Hexadecimal color codes start with a hash (#) followed by six hexadecimal digits. The first two digits represent red, the next two represent green, and the last two represent blue.

#ff5733

#33ff57

#3357ff

RGB Colors

RGB color values are specified using the rgb() function, where the red, green, and blue values are given as integers between 0 and 255.

rgb(255, 87, 51)

rgb(51, 255, 87)

rgb(51, 87, 255)

RGBA Colors

RGBA color values are similar to RGB values but include an alpha channel for opacity. The rgba() function takes four parameters: red, green, blue, and alpha (where alpha is a value between 0 and 1).

rgba(255, 87, 51, 0.5)

rgba(51, 255, 87, 0.5)

rgba(51, 87, 255, 0.5)

HSL Colors

HSL color values are specified using the hsl() function, where the hue is given in degrees (0-360), and saturation and lightness are given as percentages.

hsl(12, 100%, 60%)

hsl(120, 100%, 60%)

hsl(240, 100%, 60%)

HSLA Colors

HSLA color values are similar to HSL values but include an alpha channel for opacity. The hsla() function takes four parameters: hue, saturation, lightness, and alpha (where alpha is a value between 0 and 1).

hsla(12, 100%, 60%, 0.5)

hsla(120, 100%, 60%, 0.5)

hsla(240, 100%, 60%, 0.5)

Summary

CSS provides a variety of ways to specify colors, including color names, hexadecimal codes, RGB, RGBA, HSL, and HSLA values. Understanding these methods allows you to create a wide range of color effects and ensure consistency across different web browsers.
