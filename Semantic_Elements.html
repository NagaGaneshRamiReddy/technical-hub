Semantic Elements

What are Semantic Elements?

Semantic elements in HTML are those that clearly describe their meaning in a human- and machine-readable way. They provide information about the structure and content of web pages, making them easier to read and understand.

Common Semantic HTML Elements

Element Description Example
<header> Defines a header for a document or a section. 
<header>
    <h1>Site Title</h1>
</header>
<nav> Defines a set of navigation links. 
<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
    </ul>
</nav>
<main> Specifies the main content of a document. 
<main>
    <h2>Main Content</h2>
    <p>This is the main area of the document.</p>
</main>
<article> Defines a self-contained piece of content. 
<article>
    <h2>Article Title</h2>
    <p>This is the content of the article.</p>
</article>
<footer> Defines a footer for a document or section. 
<footer>
    <p>Copyright Â© 2024</p>
</footer>

Benefits of Using Semantic Elements

  • Accessibility: Semantic tags improve accessibility for screen readers and assistive technologies, providing context and meaning to the content.
  • SEO Optimization: Search engines can better understand the content of web pages, potentially improving search rankings.
  • Maintainability: Semantic HTML leads to clearer code, making it easier for developers to understand the structure of a web page.
