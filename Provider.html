Using Provider for State Management in Flutter

Understanding Provider

The Provider package is a popular and powerful state management solution for Flutter applications. It simplifies sharing state across widgets and makes it easier to manage complex applications.

1. Adding Provider to Your Project

To use Provider, first add it to your pubspec.yaml file:

dependencies:
  provider: ^6.0.0

2. Creating a Model Class

Start by creating a model class that extends ChangeNotifier. This class will hold the state and notify listeners when the state changes.

class CounterModel with ChangeNotifier {
  int _count = 0;

  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

3. Setting Up the Provider

Wrap your application in a ChangeNotifierProvider to make your model available to the widget tree.

void main() {
  runApp(
    ChangeNotifierProvider(
      create: (context) => CounterModel(),
      child: MyApp(),
    ),
  );
}

4. Consuming the Provider

You can access the model in your widgets using the Provider.of method or by using the Consumer widget.

class CounterScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final counter = Provider.of(context);

    return Scaffold(
      appBar: AppBar(title: Text('Counter')),
      body: Center(child: Text('Count: ${counter.count}')),
      floatingActionButton: FloatingActionButton(
        onPressed: () => counter.increment(),
        child: Icon(Icons.add),
      ),
    );
  }
}

Expected Output:

Pressing the floating action button increments the count and updates the displayed value in real time across the app.

5. Using the Consumer Widget

The Consumer widget provides a more efficient way to rebuild only the necessary parts of your widget tree.

class CounterScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Counter')),
      body: Center(
        child: Consumer(
          builder: (context, counter, child) {
            return Text('Count: ${counter.count}');
          },
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () => Provider.of(context, listen: false).increment(),
        child: Icon(Icons.add),
      ),
    );
  }
}

Summary

Provider is a powerful and flexible state management solution in Flutter. It helps manage and share state across the widget tree efficiently, making it ideal for building scalable applications.
