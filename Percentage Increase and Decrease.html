Percentage Increase and Decrease

Percentage increase and decrease are important concepts for understanding how values change relative to their original amounts. These calculations are useful in many real-life situations such as budgeting, investing, and analyzing trends.

Percentage Increase

Percentage Increase shows how much a value has grown compared to its original amount, expressed as a percentage. It helps in understanding the extent of growth.

Formula for Percentage Increase:
Percentage Increase = [(New Value - Original Value) / Original Value] × 100%

Where:
New Value: The value after the increase.
Original Value: The value before the increase.

Example:
Suppose the price of a laptop increased from ₹40,000 to ₹50,000. To find the percentage increase:

Increase = New Value - Original Value = ₹50,000 - ₹40,000 = ₹10,000.
To calculate the percentage increase:

Percentage Increase = [(₹50,000 - ₹40,000) / ₹40,000] × 100% = [₹10,000 / ₹40,000] × 100% = 25%.
Therefore, the percentage increase in the price of the laptop is 25%.

Percentage Decrease

Percentage Decrease shows how much a value has reduced compared to its original amount, expressed as a percentage. It helps in understanding the extent of reduction.

Formula for Percentage Decrease:
Percentage Decrease = [(Original Value - New Value) / Original Value] × 100%

Where:
New Value: The value after the decrease.
Original Value: The value before the decrease.

Example:
If the price of a book decreased from ₹300 to ₹240, the amount of decrease is:

Decrease = Original Value - New Value = ₹300 - ₹240 = ₹60.
To find the percentage decrease:

Percentage Decrease = [(₹300 - ₹240) / ₹300] × 100% = [₹60 / ₹300] × 100% = 20%.
Therefore, the percentage decrease in the price of the book is 20%.

Steps to Calculate Percentage Increase and Decrease

  • Determine the Original Value: Identify the initial value before any changes.
  • Find the New Value: Determine the value after the increase or decrease.
  • Calculate the Change: Subtract the Original Value from the New Value for an increase, or vice versa for a decrease.
  • Apply the Formula: Use the formulas provided to find the percentage increase or decrease.

Real-Life Applications

Understanding percentage increase and decrease is useful in various real-life scenarios:

  • Finance: Analyzing changes in investment values or loan interest rates.
  • Shopping: Calculating discounts during sales or price changes over time.
  • Business: Evaluating sales performance or changes in market trends.
