Unions in C

What is a Union?

In C programming, a union is a special data type that allows you to store different data types in the same memory location. Unlike structures, where each member has its own memory location, a union uses a single memory location to store one of its members at a time. This can save memory when you need to store different types of data but only one type at a time.

Defining a Union

To define a union, you use the union keyword followed by the union name and its members. Here’s the basic syntax:

union UnionName {
    dataType member1;
    dataType member2;
    ...
};

Each member of the union shares the same memory location, so the size of the union is determined by the size of its largest member.

Example: Defining and Using a Union

Let’s look at an example where we define a union that can store either an integer, a float, or a character, and then use it:

#include <stdio.h>

// Define a union named Data
union Data {
    int intValue;
    float floatValue;
    char charValue;
};

int main() {
    // Create a variable of type Data
    union Data data;

    // Assign a value to intValue
    data.intValue = 10;
    printf("Integer Value: %d\n", data.intValue);

    // Assign a value to floatValue (previous value of intValue is overwritten)
    data.floatValue = 3.14;
    printf("Float Value: %.2f\n", data.floatValue);

    // Assign a value to charValue (previous value of floatValue is overwritten)
    data.charValue = 'A';
    printf("Character Value: %c\n", data.charValue);

    return 0;
}

In this example, the Data union can hold an integer, a float, or a character. However, since all members share the same memory location, setting one member will overwrite the value of any previously set member. Therefore, only the last assigned value will be correctly stored and accessible.

Accessing Union Members

To access the members of a union, you use the dot operator just like with structures. However, remember that only one member can hold a valid value at a time:

data.memberName

Where memberName is the name of the union member you want to access.

Example: Accessing Union Members

Here’s an example showing how to access the members of the union:

#include <stdio.h>

// Define a union named Value
union Value {
    int num;
    float real;
    char letter;
};

int main() {
    // Create a variable of type Value
    union Value value;

    // Set the integer value
    value.num = 5;
    printf("Integer Value: %d\n", value.num);

    // Set the float value (overwrites the integer value)
    value.real = 7.5;
    printf("Float Value: %.2f\n", value.real);

    // Set the character value (overwrites the float value)
    value.letter = 'B';
    printf("Character Value: %c\n", value.letter);

    return 0;
}

In this example, assigning values to different members of the Value union demonstrates how only one member holds a valid value at any given time.

When to Use Unions

Unions are particularly useful when you need to store different types of data but only one type at a time. They are commonly used in situations where memory efficiency is important, such as embedded systems or when dealing with various types of data in a single interface.
