nth-child Pseudo-Class

What is the nth-child Pseudo-Class?

The :nth-child() pseudo-class allows you to target elements based on their order within a parent element. You can use formulas to select elements like every odd element, every even element, or a specific one.

Basic Usage

Here are some common ways to use the nth-child pseudo-class:

1. Select Odd Elements

The rule :nth-child(odd) selects all odd-numbered elements (1st, 3rd, 5th, etc.).


ul li:nth-child(odd) {
    background-color: #b70707;
}

2. Select Even Elements

The rule :nth-child(even) selects all even-numbered elements (2nd, 4th, 6th, etc.).


ul li:nth-child(even) {
    background-color: #2066a0;
}

3. Select a Specific Child

You can also select a specific element, for example, the 3rd child:


ul li:nth-child(3) {
    background-color: #ffd700;
}

Summary

The :nth-child() pseudo-class is a powerful tool for selecting elements based on their position within a parent element. You can easily apply different styles to specific children, alternating rows, or any element in a sequence.
