INSERT INTO Statement

SQL INSERT INTO Statement

The INSERT INTO statement is used to add new rows to a table in a SQL database. This command allows you to insert data into one or more columns of a table.

Basic Syntax

    INSERT INTO table_name (column1, column2, ...)
    VALUES (value1, value2, ...);
  • table_name - The name of the table where data will be inserted.
  • column1, column2, ... - The names of the columns where data will be inserted.
  • value1, value2, ... - The values to be inserted into the corresponding columns.

Examples of Using the INSERT INTO Statement

1. Inserting a Single Row

To insert a single row into a table, specify the column names and corresponding values:

    INSERT INTO employees (emp_name, emp_id, department, salary)
    VALUES ('John Doe', 101, 'Finance', 55000);

This query inserts a new employee record into the "employees" table with specified values for name, ID, department, and salary.

2. Inserting Multiple Rows

To insert multiple rows in a single statement, separate each set of values with commas:

    INSERT INTO employees (emp_name, emp_id, department, salary)
    VALUES 
        ('Alice Smith', 102, 'Marketing', 62000),
        ('Bob Johnson', 103, 'HR', 57000),
        ('Carol Williams', 104, 'Sales', 60000);

This query inserts three new employee records into the "employees" table.

3. Inserting Data with Default Values

If certain columns have default values or allow NULLs, you can omit them in the INSERT INTO statement:

    INSERT INTO employees (emp_name, department)
    VALUES ('David Brown', 'IT');

This query inserts a new employee with the default or NULL value for columns not specified (e.g., emp_id and salary).

4. Inserting Data from Another Table

You can also insert data into a table based on the results of a query from another table:

    INSERT INTO new_employees (emp_name, department, salary)
    SELECT emp_name, department, salary
    FROM employees
    WHERE hire_date > '2023-01-01';

This query inserts employee records into the "new_employees" table, selecting only those hired after January 1, 2023.

5. Handling Duplicate Records

To handle duplicates, you might use options like ON DUPLICATE KEY UPDATE (MySQL) or similar features in other SQL databases. However, basic SQL does not include built-in mechanisms for handling duplicates during insertion.

Summary

The INSERT INTO statement is a fundamental SQL command used for adding new data to tables. Whether inserting a single row, multiple rows, or data from another table, this command allows you to efficiently populate your database with new information.
