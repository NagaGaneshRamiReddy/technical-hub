Python Tutorial

Django

What is Django ?

Django is a high-level web framework for Python that encourages rapid development and clean, pragmatic design. It was created to help developers take applications from concept to completion as swiftly as possible. Django emphasizes reusability, modularity, and the "don't repeat yourself" (DRY) principle.

Features of Django

  • Rapid Development: Django provides a robust set of tools and built-in features to help developers create applications quickly.
  • Built-in Admin Interface: Django includes a powerful admin interface for managing application data, users, and more.
  • ORM (Object-Relational Mapping): Django’s ORM allows you to interact with your database using Python code, abstracting away SQL queries.
  • Security: Django offers a range of security features to protect against common web vulnerabilities, such as SQL injection and cross-site scripting.
  • Scalability: Django is designed to handle large-scale applications and supports a variety of databases and caching systems.
  • Modular Architecture: Django’s project structure promotes modular development, making it easier to manage and scale projects.

Installing Django

To start using Django, you need to have Python installed. Django can be installed via pip, the Python package manager. Open your terminal or command prompt and run:

pip install django

Creating a Simple Django Project

Here are the basic steps to create a Django project:

  1. Create a Project: Use the Django command-line tool to create a new project.
    2. django-admin startproject myproject
  2. Start the Development Server: Navigate to the project directory and start the development server.
    3. cd myproject
python manage.py runserver
  3. Create an App: Create a new app within the project to handle specific functionality.
    4. python manage.py startapp myapp
  4. Define Models and Views: Define your data models and views in the `models.py` and `views.py` files of your app.
  5. Configure URLs: Map URLs to your views by configuring the `urls.py` file.

Example of a Simple Django Application

Here is an example of a basic Django application:

# models.py in myapp
from django.db import models

class Greeting(models.Model):
    name = models.CharField(max_length=100)

# views.py in myapp
from django.http import HttpResponse
from .models import Greeting

def index(request):
    return HttpResponse("Hello, world!")

# urls.py in myapp
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

# urls.py in myproject
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]

Django Admin Interface

Django’s admin interface is a powerful tool for managing application data. You can register models with the admin site to make them accessible through the admin interface.

admin.py in myapp
from django.contrib import admin
from .models import Greeting

admin.site.register(Greeting)

Common Django Commands

  • python manage.py makemigrations: Create new migrations based on changes to models.
  • python manage.py migrate: Apply and unapply migrations.
  • python manage.py createsuperuser: Create a superuser account for the admin interface.
  • python manage.py test: Run tests for your application.

Summary

Django is a powerful and flexible web framework that simplifies the development of web applications with Python. Its emphasis on rapid development, security, and scalability makes it an excellent choice for a wide range of projects. By leveraging Django's built-in features and extensive ecosystem, you can build robust and scalable web applications efficiently.
