Functions in SQL

Functions in SQL

SQL Functions are built-in operations that perform calculations, manipulate data, or return specific values. Functions help streamline and simplify tasks within SQL queries by providing ready-made operations.

Types of SQL Functions

  • Aggregate Functions: Perform calculations on a set of values and return a single value. Commonly used with GROUP BY clauses.
  • String Functions: Operate on string values to manipulate and analyze text.
  • Date and Time Functions: Deal with date and time values to perform calculations or extract components.
  • Mathematical Functions: Perform mathematical operations on numeric values.
  • Conversion Functions: Convert data from one type to another.

Common SQL Functions

Aggregate Functions

Function Description Example
COUNT() Counts the number of rows in a set. SELECT COUNT(*) FROM Employees;
SUM() Calculates the sum of a numeric column. SELECT SUM(Salary) FROM Employees;
AVG() Calculates the average value of a numeric column. SELECT AVG(Salary) FROM Employees;
MIN() Returns the minimum value of a column. SELECT MIN(Salary) FROM Employees;
MAX() Returns the maximum value of a column. SELECT MAX(Salary) FROM Employees;

String Functions

Function Description Example
CONCAT() Concatenates two or more strings into one string. SELECT CONCAT(FirstName, ' ', LastName) AS FullName FROM Employees;
LENGTH() Returns the length of a string. SELECT LENGTH(FirstName) FROM Employees;
SUBSTRING() Extracts a substring from a string. SELECT SUBSTRING(FirstName, 1, 3) FROM Employees;
UPPER() Converts a string to uppercase. SELECT UPPER(FirstName) FROM Employees;
LOWER() Converts a string to lowercase. SELECT LOWER(FirstName) FROM Employees;

Date and Time Functions

Function Description Example
NOW() Returns the current date and time. SELECT NOW();
DATEADD() Adds a specified time interval to a date. SELECT DATEADD(day, 5, GETDATE());
DATEDIFF() Calculates the difference between two dates. SELECT DATEDIFF(day, '2024-01-01', GETDATE());
YEAR() Extracts the year from a date. SELECT YEAR(HireDate) FROM Employees;

Mathematical Functions

Function Description Example
ROUND() Rounds a numeric value to the nearest integer or specified decimal places. SELECT ROUND(Salary, 2) FROM Employees;
CEILING() Returns the smallest integer greater than or equal to a numeric value. SELECT CEILING(Salary) FROM Employees;
FLOOR() Returns the largest integer less than or equal to a numeric value. SELECT FLOOR(Salary) FROM Employees;
ABS() Returns the absolute value of a number. SELECT ABS(Salary) FROM Employees;

Conversion Functions

Function Description Example
CAST() Converts a value from one data type to another. SELECT CAST(Salary AS DECIMAL(10, 2)) FROM Employees;
CONVERT() Converts a value from one data type to another, with formatting options. SELECT CONVERT(VARCHAR, HireDate, 103) FROM Employees;

Example SQL Query Using Functions

Here is an example SQL query that utilizes various functions:

SELECT
    FirstName,
    LastName,
    ROUND(Salary, 2) AS RoundedSalary,
    UPPER(CONCAT(FirstName, ' ', LastName)) AS FullName,
    YEAR(HireDate) AS HireYear
FROM Employees
WHERE YEAR(HireDate) > 2020;
