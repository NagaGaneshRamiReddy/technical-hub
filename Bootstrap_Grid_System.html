Bootstrap Grid System


Introduction to Bootstrap Grid System

The Bootstrap grid system allows you to create responsive layouts using a series of containers, rows, and columns. It is based on a 12-column layout that enables you to control the positioning and sizing of elements across various screen sizes.

Grid Structure

  • Container: A fixed-width or fluid-width wrapper that holds the rows.
  • Row: A horizontal group of columns that create a new line in the layout.
  • Column: Vertical blocks that contain your content. Columns can be resized based on the screen size.

Creating a Basic Grid

A basic grid in Bootstrap can be created by defining a container, adding a row, and then specifying columns:

<div class="container">
    <div class="row">
        <div class="col">Column 1</div>
        <div class="col">Column 2</div>
        <div class="col">Column 3</div>
    </div>
</div>

Example of a Responsive Grid

This example creates a responsive grid layout:

<div class="container">
    <div class="row">
        <div class="col-md-4">Column 1</div>
        <div class="col-md-4">Column 2</div>
        <div class="col-md-4">Column 3</div>
    </div>
    <div class="row">
        <div class="col-md-6">Column 1</div>
        <div class="col-md-6">Column 2</div>
    </div>
</div>

Expected Output:

This code will create two rows:

  • The first row will have three equal columns, each taking one-third of the width.
  • The second row will have two equal columns, each taking half of the width.
