Python Tutorial

Syntax Basics in Python

1. Python Syntax Basics

Python syntax is designed to be readable and straightforward. Here are some key aspects of Python syntax:

2. Indentation

Python uses indentation to define blocks of code. Unlike many languages that use curly braces, Python relies on consistent indentation levels to group statements:

if True:
    print("This is indented")
    print("This is also part of the block")
print("This is outside the block")

In this example, the two print statements are part of the if block because they are indented.

3. Variables and Data Types

Variables in Python are used to store data. You don't need to declare the type of a variable explicitly. Here are some examples:

age = 25        # Integer
height = 5.9     # Float
name = "Alice"   # String
is_student = True  # Boolean

Python automatically infers the type of the variable based on the value assigned.

4. Operators

Python supports various operators for arithmetic, comparison, and logical operations:

  • Arithmetic Operators: + (addition), - (subtraction), * (multiplication), / (division), % (modulus), ** (exponentiation), // (floor division)
  • Comparison Operators: == (equal to), != (not equal to), > (greater than), < (less than), >= (greater than or equal to), <= (less than or equal to)
  • Logical Operators: and, or, not

Examples of operators in use:

a = 10
b = 5
print(a + b)     # Output: 15
print(a > b)     # Output: True
print(a == b)    # Output: False
print(not (a > b))  # Output: False

5. Comments

Comments are used to annotate code and are ignored during execution. Single-line comments start with #, and multi-line comments are enclosed in triple quotes:

# This is a single-line comment

"""
This is a
multi-line comment
"""

6. Basic Input/Output

Python provides built-in functions for input and output. To read input from the user, use input(), and to display output, use print():

name = input("Enter your name: ")
print("Hello, " + name)

Summary

Python's syntax is designed to be clean and readable, with indentation defining code blocks. Understanding variables, data types, operators, comments, and basic input/output operations are fundamental for writing Python programs.
