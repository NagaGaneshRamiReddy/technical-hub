Rust Filesystem

Understanding the Filesystem in Rust

Rust provides a powerful and flexible way to interact with the filesystem through the std::fs module. This module allows you to perform various file operations such as reading, writing, and manipulating files and directories.

Common File Operations

Here are some common operations you can perform using the std::fs module:

1. Creating a New File

You can create a new file using the File::create function:

use std::fs::File;
        use std::io::Write;
        
        fn main() {
            let mut file = File::create("example.txt").expect("Unable to create file");
            writeln!(file, "Hello, Rust!").expect("Unable to write to file");
        }

2. Reading a File

To read a file, you can use the read_to_string method:

use std::fs;
        
        fn main() {
            let contents = fs::read_to_string("example.txt").expect("Unable to read file");
            println!("File contents: {}", contents);
        }

3. Listing Files in a Directory

You can list all files in a directory using the read_dir function:

use std::fs;
        
        fn main() {
            let paths = fs::read_dir("./").expect("Unable to read directory");
        
            for path in paths {
                let path = path.expect("Unable to read path").path();
                println!("{}", path.display());
            }
        }

4. Deleting a File

To delete a file, use the remove_file function:

use std::fs;
        
        fn main() {
            fs::remove_file("example.txt").expect("Unable to delete file");
            println!("File deleted successfully.");
        }

Handling Errors

use std::fs;
        
        fn main() {
            match fs::read_to_string("nonexistent.txt") {
                Ok(contents) => println!("File contents: {}", contents),
                Err(e) => println!("Error reading file: {}", e),
            }
        }
