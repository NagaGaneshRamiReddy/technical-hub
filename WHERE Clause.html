WHERE Clause

SQL WHERE Clause

The WHERE clause is used in SQL to filter records based on specific conditions. It is used in conjunction with the SELECT, UPDATE, DELETE, and INSERT INTO statements to specify which rows should be affected by the query.

Basic Syntax

    SELECT column1, column2, ...
    FROM table_name
    WHERE condition;
  • column1, column2, ... - The names of the columns you want to retrieve.
  • table_name - The name of the table from which to retrieve data.
  • condition - The condition that must be met for rows to be included in the result set.

Examples of Using the WHERE Clause

1. Basic Filtering

Retrieve records from a table where a specific condition is met.

    SELECT emp_name, emp_id
    FROM employees
    WHERE department = 'Sales';

This query retrieves employee names and IDs from the "employees" table where the department is "Sales".

2. Using Comparison Operators

Comparison operators can be used in the WHERE clause to filter data based on numerical values or dates.

    SELECT emp_name, salary
    FROM employees
    WHERE salary > 50000;

This query retrieves employee names and salaries where the salary is greater than 50,000.

3. Combining Conditions with AND

Use the AND operator to combine multiple conditions in the WHERE clause.

    SELECT emp_name, department
    FROM employees
    WHERE department = 'HR' AND hire_date > '2020-01-01';

This query retrieves employee names and departments where the department is "HR" and the hire date is after January 1, 2020.

4. Combining Conditions with OR

Use the OR operator to include rows that meet at least one of multiple conditions.

    SELECT emp_name, department
    FROM employees
    WHERE department = 'Sales' OR department = 'Marketing';

This query retrieves employee names and departments where the department is either "Sales" or "Marketing".

5. Using IN to Match Multiple Values

The IN operator allows you to specify multiple values in a WHERE clause.

    SELECT emp_name, department
    FROM employees
    WHERE department IN ('Sales', 'Marketing', 'HR');

This query retrieves employee names and departments where the department is either "Sales", "Marketing", or "HR".

6. Using BETWEEN for Range Checks

The BETWEEN operator is used to filter the results within a specified range.

    SELECT emp_name, salary
    FROM employees
    WHERE salary BETWEEN 30000 AND 60000;

This query retrieves employee names and salaries where the salary is between 30,000 and 60,000.

7. Using LIKE for Pattern Matching

The LIKE operator is used for pattern matching with wildcard characters.

    SELECT emp_name, email
    FROM employees
    WHERE email LIKE '%@example.com';

This query retrieves employee names and email addresses where the email ends with "@example.com".

8. Using IS NULL to Check for Null Values

The IS NULL operator is used to filter rows where a column contains NULL values.

    SELECT emp_name, department
    FROM employees
    WHERE department IS NULL;

This query retrieves employee names and departments where the department column has NULL values.

Summary

The WHERE clause is a powerful tool for filtering records based on specific conditions in SQL queries. By combining conditions with operators like AND, OR, and IN, and using pattern matching with LIKE, you can refine your data retrieval to meet your needs.
