Hash table

Introduction

A hash table is a data structure that provides efficient key-value pair storage and retrieval. It uses a hash function to compute an index into an array of buckets or slots, from which the desired value can be found. Hash tables offer average-case time complexity of O(1) for insertions, deletions, and lookups, making them highly efficient for many applications.

How Hash Tables Work

Hash tables use a hash function to map keys to specific indices in an array. When a key-value pair is added, the key is hashed to determine the index in the array where the value should be stored. To handle collisions (when multiple keys hash to the same index), various techniques such as chaining (linked lists) or open addressing (probing) are used.

  1. Hash Function: Computes an index based on the key.
  2. Collision Handling: Manages cases where different keys hash to the same index.
  3. Storage: Stores the key-value pair in the determined index or linked list.

Implementation in C++


#include <iostream>
#include <unordered_map>

int main() {
    std::unordered_map<std::string, int> hashTable;

    // Insert key-value pairs
    hashTable["apple"] = 1;
    hashTable["banana"] = 2;
    hashTable["cherry"] = 3;

    // Retrieve and display values
    std::cout << "Apple: " << hashTable["apple"] << std::endl;
    std::cout << "Banana: " << hashTable["banana"] << std::endl;
    std::cout << "Cherry: " << hashTable["cherry"] << std::endl;

    // Delete a key-value pair
    hashTable.erase("banana");

    // Check if key exists
    if (hashTable.find("banana") == hashTable.end()) {
        std::cout << "Banana is not found in the hash table" << std::endl;
    }

    return 0;
}

Implementation in Python

hash_table = {}

Implementation in Java


import java.util.HashMap;

public class HashTableExample {
    public static void main(String[] args) {
        HashMap<String, Integer> hashTable = new HashMap<>();

        // Insert key-value pairs
        hashTable.put("apple", 1);
        hashTable.put("banana", 2);
        hashTable.put("cherry", 3);

        // Retrieve and display values
        System.out.println("Apple: " + hashTable.get("apple"));
        System.out.println("Banana: " + hashTable.get("banana"));
        System.out.println("Cherry: " + hashTable.get("cherry"));

        // Delete a key-value pair
        hashTable.remove("banana");

        // Check if key exists
        if (!hashTable.containsKey("banana")) {
            System.out.println("Banana is not found in the hash table");
        }
    }
}

Summary

Hash tables are a fundamental data structure that provide fast and efficient storage and retrieval of key-value pairs. They leverage a hash function to map keys to indices in an array, and use techniques like collision handling to ensure data integrity. With average-case time complexities of O(1) for key operations, hash tables are widely used in various applications such as databases, caches, and sets. Understanding hash tables and their underlying principles can significantly enhance one's problem-solving and programming skills in computer science.
