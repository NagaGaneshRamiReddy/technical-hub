Flutter Widgets

Introduction to Flutter Widgets

In Flutter, widgets are the building blocks of a user interface. Everything in Flutter is a widget, including layout models, buttons, text, and images. Widgets describe how your application looks and behaves, and they can be composed to build complex UIs from simple, reusable components.

Types of Flutter Widgets

1. Stateless Widgets

A StatelessWidget is a widget that does not change its state once it is built. These widgets are immutable and are typically used when the UI does not depend on dynamic data. Examples of stateless widgets include Text, Icon, and Container.

class MyStatelessWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Text('Hello, Flutter!');
  }
}

2. Stateful Widgets

A StatefulWidget can change its state during the application's lifetime. Stateful widgets are useful when you need to handle dynamic data or user interactions. Examples include Checkbox, Slider, and TextField.

class MyStatefulWidget extends StatefulWidget {
  @override
  _MyStatefulWidgetState createState() => _MyStatefulWidgetState();
}

class _MyStatefulWidgetState extends State<MyStatefulWidget> {
  int _counter = 0;

  void _incrementCounter() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text('Counter: \$_counter'),
        ElevatedButton(
          onPressed: _incrementCounter,
          child: Text('Increment'),
        ),
      ],
    );
  }
}

Commonly Used Flutter Widgets

1. Text

The Text widget is used to display text in Flutter. You can customize the text style, color, size, and more using the TextStyle property.

Text(
  'Hello, Flutter!',
  style: TextStyle(fontSize: 24, color: Colors.blue),
)

2. Container

The Container widget is a versatile widget that can hold a single child and apply various visual properties like padding, margin, border, and background color.

Container(
  padding: EdgeInsets.all(16.0),
  margin: EdgeInsets.all(10.0),
  color: Colors.amber,
  child: Text('This is a Container'),
)

3. Row and Column

Row and Column are layout widgets that organize their children in a horizontal or vertical direction, respectively. They are essential for building complex layouts in Flutter.

Column(
  children: [
    Text('First item'),
    Text('Second item'),
    Text('Third item'),
  ],
)

4. ListView

The ListView widget is used to display a scrollable list of items. It's perfect for handling large sets of data, especially when the number of items is not fixed.

ListView(
  children: [
    ListTile(title: Text('Item 1')),
    ListTile(title: Text('Item 2')),
    ListTile(title: Text('Item 3')),
  ],
)

5. Stack

The Stack widget allows you to stack widgets on top of each other. This is useful when you need overlapping elements or want to position widgets absolutely.

Stack(
  children: [
    Container(color: Colors.red, width: 100, height: 100),
    Positioned(
      top: 10,
      left: 10,
      child: Text('Overlayed Text'),
    ),
  ],
)

Flutter Widget Catalog

Flutter provides an extensive library of widgets, categorized into different types such as layout, input, animation, and more. Here are some other important widget types you may want to explore:

  • Scaffold: Provides a basic layout structure for Flutter apps, including an AppBar, body, and floating action button.
  • MaterialApp: Wraps your application in Material Design visual components and provides a default app structure.
  • GestureDetector: Detects gestures like taps, swipes, and pinches and responds to user input.
  • Form: A container for grouping multiple form fields like TextFormField and handling form validation.

Explore More Widgets

Flutter's flexibility allows you to combine widgets to create complex layouts. You can explore more widgets by referring to Flutter's comprehensive widget catalog available in their official documentation.
