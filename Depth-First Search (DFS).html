Depth-First Search (DFS)

Introduction

Depth-First Search (DFS) is a fundamental graph traversal algorithm that explores as far down a branch as possible before backtracking. This method is useful for tasks such as searching for a path, solving puzzles, and navigating networks.

How DFS Works

DFS starts at a selected node (the "root" or "start" node) and explores as far as possible along each branch before backtracking. The traversal can be implemented using recursion or an explicit stack.

DFS Implementation

DFS in C++


#include <iostream>
#include <> adjList; // Adjacency list

public:
    Graph(int V) : V(V), adjList(V) {}

    void addEdge(int u, int v) {
        adjList[u].push_back(v);
        adjList[v].push_back(u); // For undirected graph
    }

    void DFS(int start) {
        std::vector visited(V, false);
        std::stack stack;
        stack.push(start);

        while (!stack.empty()) {
            int vertex = stack.top();
            stack.pop();

            if (!visited[vertex]) {
                visited[vertex] = true;
                std::cout << vertex << " ";

                for (int neighbor : adjList[vertex]) {
                    if (!visited[neighbor]) {
                        stack.push(neighbor);
                    }
                }
            }
        }
    }
};

int main() {
    Graph g(5); // Create a graph with 5 vertices
    g.addEdge(0, 1);
    g.addEdge(0, 4);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    std::cout << "DFS starting from vertex 0:" << std::endl;
    g.DFS(0);
    std::cout << std::endl;
    return 0;
}

DFS in Python


class Graph:
    def __init__(self, vertices):
        self.V = vertices
        self.adjList = [[] for _ in range(vertices)]

    def add_edge(self, u, v):
        self.adjList[u].append(v)
        self.adjList[v].append(u)  # For undirected graph

    def DFS(self, start):
        visited = [False] * self.V
        stack = [start]

        while stack:
            vertex = stack.pop()

            if not visited[vertex]:
                visited[vertex] = True
                print(vertex, end=" ")

                for neighbor in self.adjList[vertex]:
                    if not visited[neighbor]:
                        stack.append(neighbor)

# Example usage
g = Graph(5)  # Create a graph with 5 vertices
g.add_edge(0, 1)
g.add_edge(0, 4)
g.add_edge(1, 2)
g.add_edge(1, 3)
g.add_edge(2, 3)
g.add_edge(3, 4)
print("DFS starting from vertex 0:")
g.DFS(0)
print()

Applications of DFS

  • Pathfinding: DFS can be used to find paths between nodes in a maze or network.
  • Topological Sorting: In directed acyclic graphs (DAGs), DFS helps in topological sorting.
  • Cycle Detection: DFS can be used to detect cycles in a graph.
  • Connected Components: DFS helps in finding all connected components in a graph.

Summary

Depth-First Search (DFS) is a fundamental graph traversal technique that explores nodes deeply along a branch before backtracking. It can be implemented using a stack or recursion. DFS is useful for solving various problems such as pathfinding, topological sorting, cycle detection, and finding connected components. Understanding DFS and its applications is crucial for tackling complex graph-related problems.
