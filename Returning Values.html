Returning Values in PHP

Overview

In PHP, functions can return values to the part of the program that called them. This allows functions to produce and provide data for further processing. Using the return statement, a function can send back a value, which can then be stored in a variable or used immediately.

Using the return Statement

The return statement is used to send a value back to the caller. Once a return statement is executed, the function ends, and the returned value is passed back:

    function add($a, $b) {
        return $a + $b;
    }

    $sum = add(5, 7);
    echo $sum; // Outputs: 12

Returning Multiple Values

PHP does not support returning multiple values directly, but you can return them in an array or object:

Returning an Array:

    function getDetails() {
        return ["John", 25, "Developer"];
    }

    $details = getDetails();
    echo $details[0]; // Outputs: John

Returning an Object:

    function getUser() {
        return (object)["name" => "Alice", "age" => 30];
    }

    $user = getUser();
    echo $user->name; // Outputs: Alice

Returning Values Conditionally

You can use the return statement conditionally within a function to return different values based on logic:

    function checkAge($age) {
        if ($age >= 18) {
            return "You are an adult.";
        } else {
            return "You are a minor.";
        }
    }

    echo checkAge(20); // Outputs: You are an adult.

Return Type Declarations

PHP allows specifying the return type of a function using return type declarations. This ensures the function returns a value of the specified type:

    function multiply(int $a, int $b): int {
        return $a * $b;
    }

    echo multiply(3, 4); // Outputs: 12

If the return value doesn’t match the declared type, PHP will throw an error.

Returning Void

A function can be declared with a void return type, which means it doesn't return any value:

    function printMessage(): void {
        echo "This function doesn't return a value.";
    }

    printMessage(); // Outputs: This function doesn't return a value.

Summary

Returning values from a function allows you to build more dynamic and reusable code. With the ability to return single values, arrays, or objects, and use type declarations, PHP provides flexible options for handling data in functions.
