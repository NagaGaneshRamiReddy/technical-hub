Rust If-Else Statements

Understanding If-Else Statements in Rust

If-else statements are fundamental control flow constructs that allow you to execute code based on conditions. In Rust, these statements help you manage the flow of execution by making decisions based on boolean expressions.

Basic If Statement

The simplest form of an if statement evaluates a condition and executes a block of code if the condition is true:

let number = 10;

if number > 0 {
    println!("The number is positive.");
}

In this example, the message will be printed because the condition (number > 0) is true.

If-Else Statement

You can use an else statement to define what should happen if the condition is false:

if number > 0 {
    println!("The number is positive.");
} else {
    println!("The number is non-positive.");
}

Here, if the number variable is not greater than 0, the second message will be printed.

Else If Statement

Rust also supports chaining conditions using else if. This allows you to check multiple conditions in sequence:

if number > 0 {
    println!("The number is positive.");
} else if number < 0 {
    println!("The number is negative.");
} else {
    println!("The number is zero.");
}

This code will print different messages based on the value of number, handling all possible cases: positive, negative, or zero.

Multiple Conditions

You can also combine conditions using logical operators:

let age = 20;

if age >= 18 {
    println!("You are an adult.");
} else {
    println!("You are a minor.");
}

In this example, if the age is 18 or older, the program will indicate that the person is an adult.

Using If in Expressions

In Rust, if statements can also be used as expressions, allowing you to assign a value based on a condition:

let number = 5;
let result = if number > 0 { "Positive" } else { "Non-positive" };

println!("The number is: {}", result);

This assigns the string "Positive" or "Non-positive" to the result variable based on the condition.

Summary

If-else statements are essential for controlling the flow of your Rust programs. They allow you to make decisions based on conditions and can be combined to handle complex logic. Understanding how to use these statements effectively will enhance your ability to write dynamic and responsive applications in Rust.
