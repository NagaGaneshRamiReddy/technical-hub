HTML Images

Understanding Images in HTML

Images are a crucial part of web content, enhancing user experience and visual appeal. The <img> tag is used to embed images in HTML documents.

Basic Image Usage

The simplest way to use an image is to specify the source (URL) and an alternative text (alt) for accessibility:

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

Expected Output:

Image with Dimensions

You can specify the width and height attributes to control the size of the image:

<img src="https://via.placeholder.com/300" alt="Placeholder Image" width="300" height="200">

Expected Output:

Responsive Images

To make images responsive, you can set the CSS properties to allow them to scale:

<img src="https://via.placeholder.com/400" alt="Responsive Image" style="max-width: 100%; height: auto;">

Expected Output:

Using Images as Links

Images can also serve as hyperlinks by wrapping the <img> tag with an <a> tag:

<a href="https://www.example.com">
        <img src="https://via.placeholder.com/150" alt="Link Image">
    </a>
