CSS Combinators

What are CSS Combinators?

CSS combinators describe the relationship between two or more selectors. They allow you to apply styles based on the structure and relationships between HTML elements.

1. Descendant Combinator (space)

The descendant combinator (represented by a space) selects all elements that are descendants of a specified element.


/* Select all <p> elements inside <div> elements */
div p {
    color: blue;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Descendant Combinator Example</title>
    <style>
        div p {
            color: blue;
        }
    </style>
</head>
<body>
    <div>
        <p>This paragraph is inside a <div> and will be blue.</p>
    </div>
    <p>This paragraph is not inside a <div> and will not be blue.</p>
</body>
</html>

2. Child Combinator (>)

The child combinator (represented by the greater-than sign >) selects only direct children of a specified element.


/* Select <p> elements that are direct children of a <div> */
div > p {
    color: green;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Child Combinator Example</title>
    <style>
        div > p {
            color: green;
        }
    </style>
</head>
<body>
    <div>
        <p>This paragraph is a direct child of a <div> and will be green.</p>
        <div>
            <p>This paragraph is nested inside another <div> and will not be green.</p>
        </div>
    </div>
</body>
</html>

3. Adjacent Sibling Combinator (+)

The adjacent sibling combinator (represented by the plus sign +) selects the element that is immediately after the specified element.


/* Select the <p> element that immediately follows a <div> */
div + p {
    color: red;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>Adjacent Sibling Combinator Example</title>
    <style>
        div + p {
            color: red;
        }
    </style>
</head>
<body>
    <div>Div content</div>
    <p>This paragraph comes immediately after a <div> and will be red.</p>
    <p>This paragraph does not immediately follow a <div> and will not be red.</p>
</body>
</html>

4. General Sibling Combinator (~)

The general sibling combinator (represented by the tilde ~) selects all elements that are siblings of the specified element.


/* Select all <p> elements that are siblings of a <div> */
div ~ p {
    color: purple;
}

Example:


<!DOCTYPE html>
<html lang="en">
<head>
    <title>General Sibling Combinator Example</title>
    <style>
        div ~ p {
            color: purple;
        }
    </style>
</head>
<body>
    <div>Div content</div>
    <p>This paragraph is a sibling of the <div> and will be purple.</p>
    <p>This one will also be purple because it's a sibling of the <div>.</p>
</body>
</html>

Summary

CSS combinators are useful for selecting elements based on their relationship with other elements. They provide fine control over how you apply styles in complex HTML structures.
