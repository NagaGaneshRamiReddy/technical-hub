Enums in Java

Introduction to Enums

Enums in Java are a special type of class used to define a collection of constants. They provide a type-safe way to work with a fixed set of constants and improve code readability and maintainability.

Defining an Enum

Enums are defined using the enum keyword. Here’s the basic syntax for defining an enum:

public enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
}

In this example, Day is an enum representing the days of the week.

Using Enums

Enums can be used to create variables of the enum type and access their constants. Here’s how you can use enums:

public class EnumExample {
    public static void main(String[] args) {
        Day today = Day.MONDAY;
        System.out.println("Today is: " + today);
    }
}

Enum Methods

Enums in Java come with several built-in methods:

  • values(): Returns an array of all the constants in the enum.
  • valueOf(String name): Returns the enum constant with the specified name.
  • name(): Returns the name of the enum constant.
  • ordinal(): Returns the ordinal (position) of the enum constant.

Examples of these methods:

public class EnumMethods {
    public static void main(String[] args) {
        // Get all enum constants
        for (Day day : Day.values()) {
            System.out.println(day);
        }

        // Get enum constant by name
        Day friday = Day.valueOf("FRIDAY");
        System.out.println("Day: " + friday);

        // Get name and ordinal
        System.out.println("Name: " + friday.name());
        System.out.println("Ordinal: " + friday.ordinal());
    }
}

Enum with Fields and Methods

Enums can have fields, methods, and constructors. You can add additional functionality to enums by defining fields and methods within them:

public enum Day {
    SUNDAY("Weekend"), MONDAY("Weekday"), TUESDAY("Weekday"),
    WEDNESDAY("Weekday"), THURSDAY("Weekday"), FRIDAY("Weekday"),
    SATURDAY("Weekend");

    private String type;

    Day(String type) {
        this.type = type;
    }

    public String getType() {
        return type;
    }
}

Using the enum with fields and methods:

public class EnumWithFields {
    public static void main(String[] args) {
        for (Day day : Day.values()) {
            System.out.println(day + " is a " + day.getType());
        }
    }
}

Switch Statement with Enums

Enums work seamlessly with the switch statement, allowing you to perform different actions based on the enum constant:

public class EnumSwitch {
    public static void main(String[] args) {
        Day today = Day.FRIDAY;

        switch (today) {
            case MONDAY:
                System.out.println("Start of the work week.");
                break;
            case FRIDAY:
                System.out.println("End of the work week.");
                break;
            case SATURDAY:
            case SUNDAY:
                System.out.println("Weekend!");
                break;
            default:
                System.out.println("Midweek day.");
        }
    }
}

Summary

Enums in Java provide a powerful way to work with a fixed set of constants in a type-safe manner. By defining enums, you can improve code readability and maintainability, use built-in methods, add fields and methods, and leverage enums in switch statements. Understanding how to define and use enums is essential for writing clean and effective Java code.
