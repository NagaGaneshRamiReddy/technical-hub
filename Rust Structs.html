Rust Structs

Understanding Structs in Rust

Structs are a fundamental feature in Rust that allow you to create custom data types. They enable you to group related data together, making your code more organized and expressive.

Defining a Struct

To define a struct, use the struct keyword followed by the struct name and its fields:

struct Person {
    name: String,
    age: u32,
}

In this example, we defined a Person struct with two fields: name of type String and age of type u32.

Creating an Instance of a Struct

To create an instance of a struct, you can use the following syntax:

fn main() {
    let person = Person {
        name: String::from("Alice"),
        age: 30,
    };

    println!("Name: {}, Age: {}", person.name, person.age);
}

Structs with Functions

You can define methods for your structs by implementing functions within an impl block:

impl Person {
    fn greet(&self) {
        println!("Hello, my name is {} and I am {} years old.", self.name, self.age);
    }
}

fn main() {
    let person = Person {
        name: String::from("Bob"),
        age: 25,
    };

    person.greet(); // Outputs: Hello, my name is Bob and I am 25 years old.
}

Tuple Structs

Rust also supports tuple structs, which are like regular structs but without named fields:

struct Color(u32, u32, u32); // Tuple struct

fn main() {
    let black = Color(0, 0, 0);
    println!("Black color: ({}, {}, {})", black.0, black.1, black.2);
}
