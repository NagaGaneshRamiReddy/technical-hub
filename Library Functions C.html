Library functions in C

Introduction

Library functions are pre-defined functions provided by the C standard library and other libraries that simplify common tasks. These functions are categorized into various headers and can be used to perform operations such as mathematical calculations, input/output, string manipulations, and more.

1. Standard Library Headers

The C standard library includes several headers, each providing functions related to specific tasks. Here are some common headers and their associated functions:

  • <stdio.h>: Functions for input and output operations, such as printf and scanf.
  • <stdlib.h>: Functions for memory allocation, process control, conversions, and more, such as malloc and exit.
  • <string.h>: Functions for string manipulation, such as strlen and strcpy.
  • <math.h>: Functions for mathematical operations, such as sqrt and pow.
  • <ctype.h>: Functions for character classification and conversion, such as isdigit and toupper.
  • <time.h>: Functions for time and date manipulation, such as time and localtime.

2. Examples of Common Library Functions

Here are some examples of frequently used library functions:

Input and Output Functions


#include <stdio.h>

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);
    printf("You entered: %d\n", num);
    return 0;
}

This example uses printf to print a message and scanf to read user input.

String Manipulation Functions


#include <string.h>

int main() {
    char str1[50] = "Hello";
    char str2[50] = "World";

    strcat(str1, " ");
    strcat(str1, str2); // Concatenates str2 to str1

    printf("Concatenated string: %s\n", str1);
    return 0;
}

This example uses strcat to concatenate two strings.

Mathematical Functions


#include <math.h>

int main() {
    double x = 9.0;
    double result = sqrt(x); // Calculates the square root of x
    printf("Square root of %.2f is %.2f\n", x, result);
    return 0;
}

This example uses sqrt to calculate the square root of a number.

Character Classification Functions


#include <ctype.h>

int main() {
    char ch = 'A';
    if (isalpha(ch)) {
        printf("%c is an alphabetic character.\n", ch);
    }
    return 0;
}

This example uses isalpha to check if a character is an alphabetic letter.

3. How to Use Library Functions

To use a library function:

  1. Include the relevant header file at the beginning of your program using the #include directive.
  2. Call the function in your program as needed, providing the required arguments and handling the return value appropriately.
