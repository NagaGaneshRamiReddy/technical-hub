File Input/Output (I/O) in C++

Introduction

File Input/Output (I/O) in C++ allows you to read from and write to files. It is essential for applications that need to handle data storage and retrieval. The C++ Standard Library provides functionality for file operations through the `` header, which includes classes for file stream operations.

File Stream Classes

C++ provides three primary classes for file operations:

  • std::ifstream: Used for reading from files.
  • std::ofstream: Used for writing to files.
  • std::fstream: Used for both reading from and writing to files.

Basic File Operations

Here are some common operations for file I/O:

1. Writing to a File

To write data to a file, use std::ofstream. Here’s an example:

#include <iostream>
#include <fstream>

int main() {
    std::ofstream outFile("example.txt"); // Open file for writing

    if (outFile.is_open()) {
        outFile << "Hello, File World!" << std::endl;
        outFile.close(); // Close the file
    } else {
        std::cout << "Unable to open file for writing" << std::endl;
    }

    return 0;
}

2. Reading from a File

To read data from a file, use std::ifstream. Here’s an example:

#include <iostream>
#include <fstream>

int main() {
    std::ifstream inFile("example.txt"); // Open file for reading
    std::string line;

    if (inFile.is_open()) {
        while (std::getline(inFile, line)) {
            std::cout << line << std::endl;
        }
        inFile.close(); // Close the file
    } else {
        std::cout << "Unable to open file for reading" << std::endl;
    }

    return 0;
}

3. Reading and Writing with std::fstream

To perform both reading and writing operations, use std::fstream. Here’s an example:

#include <iostream>
#include <fstream>

int main() {
    std::fstream file("example.txt", std::ios::in | std::ios::out | std::ios::trunc); // Open file for both reading and writing

    if (file.is_open()) {
        file << "This is a test" << std::endl; // Write to file
        file.seekg(0); // Move to the beginning of the file
        std::string line;
        while (std::getline(file, line)) {
            std::cout << line << std::endl; // Read from file
        }
        file.close(); // Close the file
    } else {
        std::cout << "Unable to open file" << std::endl;
    }

    return 0;
}

Handling File Errors

When working with files, it’s important to handle potential errors:

  • Check if the file is open: Always check if the file stream is successfully opened before performing operations.
  • Handle exceptions: Use try-catch blocks to handle exceptions related to file operations.
#include <iostream>
#include <fstream>

int main() {
    try {
        std::ifstream inFile("nonexistent.txt"); // Attempt to open a non-existent file

        if (!inFile) {
            throw std::runtime_error("File could not be opened");
        }

        // File operations...

    } catch (const std::exception &e) {
        std::cout << "Exception: " << e.what() << std::endl;
    }

    return 0;
}

File Stream Modes

When opening a file, you can specify various modes:

  • std::ios::in: Open for reading.
  • std::ios::out: Open for writing (creates a new file or truncates an existing file).
  • std::ios::app: Append to the end of the file.
  • std::ios::binary: Open in binary mode.
